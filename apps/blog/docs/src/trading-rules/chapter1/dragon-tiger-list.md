# 龙虎榜规则

## 一、 定义与核心逻辑

### 1.1 定义

**龙虎榜**（官方称为“交易公开信息”）是交易所对满足价格、换手、振幅等特定异动标准的个股，披露其当日买卖金额最大的前 5 名席位（营业部或机构专用席位）名称及交易金额的透明化制度。

### 1.2 披露名额与优先逻辑

1. **名额限制**：沪深主板/双创板块单日异动原则上仅披露偏离值最大的 **前 3 只**；北交所通常披露 **前 5 只**。
2. **择一原则**：若个股同时触及多项披露标准（如单日偏离与3日累计），交易所通常仅选择其中一项进行披露（通常优先披露 3 日累计数据）。

## 二、 官方触发规则（精准量化）

### 2.1 沪深主板

| 触发类型 | 单日触发标准 | 3日累计触发标准 |
| :--- | :--- | :--- |
| **偏离值** | 日涨跌幅偏离值达 **±7%** (ST为±5%) | 连续3个交易日偏离值累计达 **±20%** (ST为±12%) |
| **换手率** | 日换手率达 **20%** | - |
| **振幅** | 日振幅达 **15%** | - |

### 2.2 创业板 / 科创板 (注册制)

| 触发类型 | 单日触发标准 | 3日累计触发标准 |
| :--- | :--- | :--- |
| **偏离值** | **日涨跌幅(或偏离值)**达 **±15%** | 连续3个交易日偏离值累计达 **±30%** |
| **换手率** | 日换手率达 **30%** | - |
| **振幅** | 日振幅达 **30%** | - |

### 2.3 北交所

| 触发类型 | 单日触发标准 (披露前5名) | 3日累计触发标准 |
| :--- | :--- | :--- |
| **涨跌幅** | 日涨跌幅达 **±30%** | 连续3个交易日偏离值累计达 **±40%** |

## 三、 席位属性识别（实战标签）

| 席位名称关键词 | 系统标签 | 资金属性及特征 |
| :--- | :--- | :--- |
| **机构专用** | **机构席位** | 公募、私募、社保、险资。反映主流大资金对基本面的定价权。 |
| **香港中央结算** | **北向资金** | 沪/深股通。代表外资趋势，重点观察其在波段拐点处的动向。 |
| **XX分公司/总部** | **顶级游资** | 如中信证券西安朱雀大街等。代表活跃的大额短线游资。 |
| **东财拉萨/财富中心** | **散户合力** | 东方财富旗下席位。反映市场人气博弈及散户资金的流向。 |

## 四、 重要说明与计算公式

### 4.1 偏离值精确计算
>
> `单日偏离值 = 个股当日涨跌幅 - 对应基准指数当日涨跌幅`

**对应基准指数表：**

* **上交所（主板/科创）**：上证指数 (000001) / 科创综指 (000046)
* **深交所（主板）**：**深证成指** (399001)
* **深交所（创业板）**：**创业板综指** (399102)

### 4.2 披露时间

* 正常交易日 **16:30 - 19:00** 陆续发布，20:00 前基本完成全量更新。
