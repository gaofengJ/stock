# 股票交易异常波动

## 一、定义与目的

### 定义

**股票交易异常波动**是指股票在一定交易周期内，**价格、成交量或换手率等交易指标出现明显偏离正常水平**，达到交易所设定的监测或监管关注条件。

### 目的

设立交易异常波动规则，主要用于：

1. **提示交易风险**，防止投资者在非理性波动中盲目参与；
2. **督促信息披露核查**，保障信息披露真实、公平、及时；
3. **识别异常交易行为**，维护市场交易秩序；
4. **实施分级监管管理**，对高风险证券进行重点监控。

> **总结**：  
> 异动不是对涨跌方向的判断，而是对**异常交易状态**的风险提示机制。

## 二、监管强度等级说明（系统通用）

| 等级 | 监管强度 | 含义说明 |
| :---: | :---: | :--- |
| L1 | 提示级 | 常规异常，风险提示为主 |
| L2 | 关注级 | 异动幅度较大，需重点观察 |
| L3 | 重点核查 | 交易所重点关注，信息披露核查 |
| L4 | 持续监管 | 持续异常状态，可能伴随监管措施 |

## 七、名词解释

### 什么是“偏离值”？

**偏离值**用于衡量个股走势相对市场的独立性。

**计算公式：**

`单日偏离值 = 个股当日涨跌幅 - 对应板块指数当日涨跌幅`

**指数对应关系：**

- 上交所主板：上证指数（000001）
- 深交所主板：深证综指（399106）
- 创业板：创业板指（399006）
- 科创板：科创50（000688）

> 注：异动计算通常基于收盘价，不包含无涨跌幅限制情形。
