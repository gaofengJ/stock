# 股票交易异常波动

## 一、定义与目的

### 定义

**股票交易异常波动**是指股票在一定交易周期内，**价格、成交量或换手率等交易指标出现明显偏离正常水平**，达到交易所设定的监测阈值。

### 目的

设立异动规则的目的在于：

1. **及时提示交易风险**：防止投资者在非理性波动中盲目交易。
2. **督促上市公司披露信息**：保障信息披露的真实、公平。
3. **识别异常交易行为**：维护市场交易秩序。
4. **为监管分级管理提供依据**：对高风险股票实施重点监控。

> **总结**：异动不是对涨跌的判断，而是对“异常交易状态”的风险提示机制。
