# 股票交易异常波动

## 一、 定义与判定逻辑

**股票交易异常波动**是指股票在特定周期内，收盘价格涨跌幅相对于基准指数出现严重偏离，触及交易所设定的监管阈值。

### 核心公式：偏离值计算
>
> `单日涨跌幅偏离值 = 个股当日涨跌幅 - 对应基准指数当日涨跌幅`

| 板块 | 对应基准指数 | 指数代码 |
| :--- | :--- | :--- |
| **上交所主板** | 上证指数 | SH000001 |
| **深交所主板** | **深证成指** | SZ399001 |
| **创业板** | 创业板综指 | SZ399102 |
| **科创板** | **科创综指** | SH000046 |

## 二、 官方强制监控标准（量化数值）

### 2.1 基础异常波动（3日规则）

触及以下标准需发布《异动公告》并披露龙虎榜（交易公开信息）：

| 板块 | 统计窗口 | 触发条件 (收盘价累计偏离值) | ST/风险警示个股 |
| :--- | :---: | :--- | :--- |
| **沪深主板** | 3个交易日 | 累计达 **±20%** | ST为 **±12%**(沪) / **±15%**(深) |
| **创业/科创** | 3个交易日 | 累计达 **±30%** | ST/退市整理期通常为 ±15% |
| **北交所** | 3个交易日 | 累计达 **±40%** | - |

### 2.2 严重异常波动（停牌核查红线）

| 监控维度 | 累计偏离阈值 (上涨/下跌) | 适用范围 | 核心备注 |
| :---: | :--- | :--- | :--- |
| **10个交易日** | 累计偏离达到 **+100% / -50%** | 沪深全板块 | 市场公认的“翻倍监管线” |
| **10个交易日** | **触及 3 次同向异常波动** | **深交所 (主、创)** | 以交易所认定的异动公告次数为准 |
| **20个交易日** | 累计偏离达到 **+200% / -70%** | 沪深全板块 | 针对长周期“慢妖”品种 |
| **30个交易日** | 累计偏离达到 **±200%** | **北交所** | 触发后需立即公告并核查 |

## 三、 复盘系统行为类指标（自定义风控）

| 指标名称 | 建议阈值 | 风险标签 | 含义说明 |
| :--- | :--- | :---: | :--- |
| **高换手率** | 主板 >20% / 双创 >30% | 筹码派发/接力 | 衡量存量筹码交换烈度，连板末端高换手通常为风险 |
| **宽幅震荡** | 日内振幅 >15% | 盘中洗盘/出货 | 游资博弈特征，常伴随“天地/地天板”及龙虎榜 |
| **爆量异动** | 量比 >2.5 | 资金异动 | 情绪资金突发性涌入，多见于题材启动或利好刺激 |

## 四、 监管响应分级

1. **L1 (披露级)**：强制发布《异动公告》。公司需核实是否存在应披露而未披露的重大信息。
2. **L2 (关注级)**：下发《关注函》。要求核实基本面变化、股东减持计划及是否存在内幕交易。
3. **L3 (限制级)**：**停牌核查**。触及严重异常波动标准后，由交易所判定是否需进入停牌程序。

## 五、 重要说明与特殊豁免

1. **同向性原则**：深交所“10天3次”异动需为同向。若10天内 2次大涨异动 + 1次大跌异动，不计入严重异常。
2. **计算基准**：偏离值仅以**收盘价**为准，盘中毛刺（如盘中瞬时涨幅）不计入异动逻辑。
3. **豁免日期**：新股上市首日、无涨跌幅限制的特殊交易日（如改革首日）不计入异常波动统计。
