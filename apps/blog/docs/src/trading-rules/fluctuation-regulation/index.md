# 异动监管

本章主要涵盖了市场交易中关于“异常波动”的监管定义以及“龙虎榜”等公开信息的披露规则。理解这些规则有助于投资者识别监管边界，规避潜在的停牌核查风险，并利用公开信息辅助交易决策。

- [龙虎榜规则](./dragon-tiger-list.md)
  > 系统介绍了交易公开信息的披露逻辑、触发条件（如日涨跌幅偏离值、换手率等），以及如何识别机构、游资等不同席位的资金属性。
