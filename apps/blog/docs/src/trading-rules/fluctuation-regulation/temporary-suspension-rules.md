# A股临时停牌（临停）规则

## 一、 核心逻辑与触发条件

临时停牌（临停）主要发生在**无涨跌幅限制**的证券交易时段，旨在缓解极端情绪，提供“冷静期”。

### 1. 触发阈值（价格波动临停）

当盘中成交价格较**当日开盘价**首次上涨或下跌达到以下幅度时，实施停牌：

| 触发阈值 | 停牌时长 | 备注 |
| :--- | :--- | :--- |
| **首次达到 ±30%** | **10 分钟** | 每个方向全天仅限一次 |
| **首次达到 ±60%** | **10 分钟** | 每个方向全天仅限一次 |

### 2. 不同板块的适用时段

由于各板块“无涨跌幅限制”的天数不同，临停的适用期也不同：

* **沪深主板 / 创业板 / 科创板：** 上市前 **5** 个交易日均适用。
* **北交所：** 仅上市 **首日** 适用（第二天起实行 ±30% 涨跌幅，不再临停）。

## 二、 交易操作指南（临停期间）

临停期间并非完全停止业务，系统依然接受特定的指令处理。

* **订单处理：** 临停前已申报且未成交的订单**继续有效**，按原时间顺序排队。
* **允许操作：** 投资者可以继续进行**申报（挂单）**和**撤单**。
* **复牌方式：** 临停结束时，系统会对期间积压的所有订单进行一次**集合竞价撮合**，随后恢复连续竞价。
* **时间冲突处理：** 若临停时间跨越 **14:57**，则无论是否满10分钟，均在 **14:57** 强行复牌进入收盘集合竞价。

## 三、 监管与异常干预

除系统自动触发的价格临停外，交易所拥有“即时监管措施”权限：

1. **异常交易临停：** 涉及对倒、操纵股价或大规模异常程序化交易时，交易所可手动干预。
2. **停牌核查（特停）：** 若证券近期累计涨跌幅严重偏离基本面，可能由临时停牌升级为**停牌核查**（通常需停牌数日并发布公告）。
3. **技术故障：** 交易系统或通信异常导致的非自愿性停牌。

## 四、 交易风险提示

1. **北交所特殊性：** 务必记得北交所**第二天起无临停**，股价波动可能极大。
2. **价格笼子（重点）：** 在临停期间挂单，申报价格需在有效申报价格范围内（通常为基准价的 **±10%**）。若挂单价格过高或过低，复牌时可能被视为**废单**。
3. **撤单权：** 如果临停期间市场情绪发生重大逆转，应及时利用临停期撤回不符合预期的挂单。
