# 龙虎榜规则

## 一、 定义与核心计算逻辑

### 1.1 定义

**龙虎榜**（官方称为“交易公开信息”）是交易所对满足价格、换手、振幅等特定异动标准的个股，披露其当日买卖金额最大的前 5 名席位（证券营业部或机构专用席位）名称及交易金额的透明化披露制度。

### 1.2 核心计算公式：偏离值

**偏离值**是判定龙虎榜上榜最核心的量化指标，用于衡量个股脱离大盘的独立走势。

> **单日偏离值** = 个股当日涨跌幅 - 对应基准指数当日涨跌幅
>
> **3日累计偏离值** = 个股连续3个交易日涨跌幅累计 - 对应基准指数连续3个交易日涨跌幅累计

### 1.3 基准指数对应表（实操唯一指定）

计算偏离值时，必须根据个股所属板块对号入座，否则计算结果将与官方榜单产生误差：

| 交易板块 | 对应基准指数名称 | 指数代码 |
| :--- | :--- | :--- |
| **上交所主板** | 上证指数 | SH000001 |
| **深交所主板** | **深证成指** | SZ399001 |
| **创业板** | **创业板综指** | SZ399102 |
| **科创板** | **科创综指** | SH000046 |

### 1.4 披露名额与优先逻辑

1. **名额限制**：沪深主板/双创板块单日标准原则上仅披露偏离值最大的 **前 3 只**；北交所通常披露 **前 5 只**。
2. **择一原则**：若个股同时触及多项标准（如单日偏离与3日累计），交易所通常仅选择其中一项进行披露（通常优先披露信息量更大的 **3日累计数据**）。

## 二、 官方触发规则（精准量化数值）

### 2.1 沪深主板

| 触发类型 | 单日触发标准 | 3日累计触发标准 |
| :--- | :--- | :--- |
| **价格偏离** | 日涨跌幅偏离值达 **±7%** (ST为±5%) | 连续3个交易日偏离值累计达 **±20%** (ST为±12%) |
| **换手率** | 日换手率达 **20%** | - |
| **振幅** | 日振幅达 **15%** | - |

### 2.2 创业板 / 科创板 (注册制)

| 触发类型 | 单日触发标准 | 3日累计触发标准 |
| :--- | :--- | :--- |
| **价格异动** | **日涨跌幅(或偏离值)**达 **±15%** | 连续3个交易日偏离值累计达 **±30%** |
| **换手率** | 日换手率达 **30%** | - |
| **振幅** | 日振幅达 **30%** | - |

### 2.3 北交所

| 触发类型 | 单日触发标准 (披露前5名) | 3日累计触发标准 |
| :--- | :--- | :--- |
| **价格异动** | 日涨跌幅达 **±30%** | 连续3个交易日偏离值累计达 **±40%** |

## 三、 席位属性识别（实战标签）

| 席位名称关键词 | 系统标签 | 资金属性及特征 |
| :--- | :--- | :--- |
| **机构专用** | **机构席位** | 公募、私募、社保、险资。反映主流大资金对个股基本面的定价权。 |
| **香港中央结算** | **北向资金** | 沪/深股通。代表外资动向，重点观察其在波段拐点处的连续性。 |
| **XX分公司/总部** | **顶级游资** | 知名席位（如中信证券西安朱雀大街等）。代表活跃的大额短线游资。 |
| **东财拉萨/财富中心** | **散户合力** | 东方财富旗下席位。反映散户活跃度及市场人气博弈的极致程度。 |

## 四、 披露时间与特殊说明

* **披露时间**：正常交易日 **16:30 - 19:00** 陆续发布，20:00 前基本完成全量更新。
* **数据剔除**：新股上市首日、无涨跌幅限制的特殊交易日不纳入统计周期。
* **计算基准**：所有偏离值判定均以**收盘价**为准，盘中突破但收盘拉回不触发规则。
