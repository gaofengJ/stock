# 2018-2-27复盘

作者：**爱在冰川**

## 原文内容

闲 聊

先讲重点错了，先看公告

19:00【韩建河山四连板 股票停牌核查】
19:24【奇精机械拟10转4派2 上交所下发问询函】
20:31【贵州燃气三连板 股票停牌核查】
21:32【梅雁吉祥：股价涨幅异常，停牌核查】

这里看管的第一感觉一定是，连板股不能炒了，高送转也监管了。
在这里我们要重点看一下他们的代码都是60开头的，（包括之前特停，明日复牌的天华院）
60开头的都是上海交易所的，00和30开头的都属于深圳交易所这个基本常识大家都知道吧。
上海交易所的监管，一直都很严。
来个图给大家感受下。

易见股份，之前停牌之后，复盘继续一字板，当然立马给关进去。
上交所的股，，出了特停你，还会给你打电话，窗口指导你，叫你撤单。。
今天这密集的公告之后，明天会有两种情况

1：全军卒。这两天的盘面，全靠黄线撑的，而撑住黄线的，就是市场人气，而特停打击的就是人气。
2：上证卒，继续转战深交所的个股，逮着深交所的连板股，继续狂怼。

近端时间唯一的没被特停的的妖股，是智慧农业，仅仅是发个公告了事，
但环境是当时行情正处于暴跌的时候，估计深交所也考虑了下情绪，
所以，在现在这个位置疯狂特停，可能上交所觉得，长得太多了，需要回调下了。
无明确尺度定义的特停，真是我大A的特色呀~~~~炒股好难。

赚钱效应

得益于中小创的继续强势，所以昨天的涨停股，今天都得到了较好的溢价，还算不错，
但是今晚的海量特停，明天是情绪是巨大打击，打击之后怎么走，只能明天看啦~~~

大 盘

大盘目前比较难分析，个人认为，可以拆分成两块。
上证50本来昨天就要杀的，但是被创业板救了一条命，所以形成了个下影线，今天继续补跌，反包了昨天的下影线。
当然，这里面主跌的主力，还是受到老板电器影响，另外一个就是黄线。
就是领先指数，全天震荡，横盘。
创业板继续放量，相对昨天来说，只能算是一个小涨了。
原来我的预计是要震荡两天的，今天上证50先一步开跌，中小创震荡上走。
所以明天的行情，也不会太好了。
明天还是谨慎为主，6连阳之后，有获利回吐的需要了，
创业板的两个权重，同花顺的和东方财富，今天业绩都小幅度的不及预期，明天创业板指数还会承受压力。
今天盘中的置顶留言已经建议各位减仓了，明天的策略，如果你重仓，那么也可以减仓了，后天就周四了。

热 点

及同年的热点，无比散乱。。。轮动了很多板块，，很多板块都有龙头，明天还会继续抢资金。
热点太多，不想一一点评了。
另外今天有条暗线是大国重器，但是我不建议给位明天再去了，
今天盘中的时候，基本上已经被挖成明线了，都有不同程度的上涨。
明天再去，很有可能就是接盘了。

个 股

个股复盘：讲下我的操作。

1：今天尾盘把智慧农业给卖了。
在尾盘的翻红以后，回落再翻红，再回落，再连翻红都没有力气的时候，基本可以判断日内不会再有拉升的机会了，
所以就卖掉了，一个短线股，拿了5个交易，靠做T做了23个点利润，我已经心满意足了。
虽然仓位不多，但是对于交易的信心和手感回复起了非常大的作用，最积极的影响是，让我拿住了美联新才。

2：建科院尾盘拉起来的时候卖了，亏了一个多点，这笔尾盘潜伏以失败收场。

3：另外盘中看见三丰智能上板的时候，名字里面带智能的所有股都在拉，
就追了个中端超跌次新，海川智能，尾盘也回落了。
明天准备看一下三丰智能的走势，来判断是出是留，毕竟后面《大国重器》的第七集讲的就是智能制造。

4：美联新才，没什么好说的，高送转龙头，
从龙虎榜来看，出了之前锁仓和离场的，今天新进两个席位，也就各买了1000多万，还是以市场合力为主。
市场合力还是很不错的。
如果没这些乱七八糟的特停，明天4板是妥妥的。
现在增加的分歧，现在就是看，明天被情绪打击挂掉，还是资金转战深圳了。
因为我利润比较多，所以我会锁仓观察~~~看情况而定，说实话，不是很修想卖。

持仓如下：

个股展望：

1：连板股
目前市场的情绪就在连板股上面。
我把60的都划掉了，另外华森主要是跟风贵燃的，所以明天可能会比较差。
银之杰，征信龙头，美联新才，高送龙头，南京聚隆，次新龙头
三维丝，环保龙头，上海新阳，芯片龙头，除了银之杰，都各自有自己的小弟，明天又将是一场混战。

2：次新+高送
这个周五肯定会有IPO的，所有周五次新肯定要杀的，那么肯定会有一小撮人，周四就要卖了。
所以，明天是次新可参与的最后一天了。
另外说一个题外话，再我给各位讲解南京聚隆这个股的第一天的时候，
现在大家知道，为啥我那天提一嘴时，把这是个深圳股放在了最前面了吧。。因为，没上交所会特停了啊。。。。
另外明天可以做接力的，分别有，排名有先后。
另外炒新的时候，炒高送往往是一起的。
目前整个高送板块都在走趋势，所以你上车的人，建议是持股，没上车的人，这个位置，也很难找到好上车的位置了。

3：2会概念
有两个明标的可以考虑做一下低吸，
新华网，现阶段被拉出来的明牌，明天要低开的，会有低吸的机会。
中科信息，以前的老龙头，今天长上影线，明天也要低开的，会有低吸的机会。
但是，如果行情走差的话，低吸吸挂的案例，也是比比皆是。

目前的行情，基本就是重复之前那段做接力股日子的行情，但是越到后面风险越大。
