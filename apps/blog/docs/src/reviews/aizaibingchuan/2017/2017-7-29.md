# 2017-7-29复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/wR51YVjCIf1tTvlA_aTDpA>

## 原文内容

1：新股发行的关系

这个其实我讲过好多次了

之前我打了很多字，有人还是不懂，所以我做了个图片，不晓得能不能看的懂。

A: 外库

目前外库排队八百多家，这个是没有参考意义的，

为什么这么说，因为现在发行速度这么快，排队数量没有丝毫减少，反而越来越多。

举个例子 6月份预批就有116家。。。

B：内库

这个非常具有参考意义

如我图一所示，只有过会通过，才有资格进入内库，

进入内库后，你才有上市的机会。

这个我下文会详细介绍这里面的关系。

C: 最终发行

每周证监会发行的个股，就是从内裤里掏出来的玩意。

所以内裤里有多少货直接，直接关系到，能出来多少货。

当然这个一定成度是由证监会主观调控的，但是其客观增长的存量，也会最终影响出来的量。

2：内库与每周发行量以及板块的关系

之前我讲过，每周上会多少，过会多少，多少会对每周IPO公布的数量有关系。

这篇文章我想写很久了，昨天终于抽空，把所有的数据都统计完了。

上图看上去比较杂乱，肯定你也没有耐心去细看。

我们可以直接看下面你这张图

在过年期间，上会过会的暂停，一度导致了次新神奇逆转，过年前后的两个多月，次新一度张飞了。

我们把股价叠进去，如下图，你再看看。

有没有发现趋势的相反性？

基本是对的上的。

只有3月份次新板块的高点看似对不准。

其实只要看对下上会数量，就能看出来了。

一次上会15家横了一星期，后面1次上会15家，直接就挂了。

3：次新板块走熊的根本原因

可以看下3个箭头，

紫色箭头，年初一段次新小牛市，量价起飞，（符合你的认知）

红色箭头，走熊的一段，可以看到初期一小段的缩量之后，成交量持续放大，股价却慢慢下跌，（这还符合你的认知吗？）

绿色箭头，这是今年整年的箭头，次新板块整年是下跌的，成交量越来越大。

究其原因，很简单，因为板块内的成分股，越来越多，但增量资金并没有跟上。

这就是我以前写过，抽血效应。

炒次新的资金是有限的，股票池却越来越大。

资金匀不过来，就只能一直跌了。

4：操作思路及未来展望

A:操作思路

可以看到，随着内库的越来越大，次新指数也是越来越小。

你不得不否认两者之间的关系。

想想也简单，你膀胱里的水越来越多，你还不得赶紧找厕所？

憋的多了，你还不得尿的多一点？

波段操作建议参考思路：

目前持续4周9家，发布，那么结合85%的通过率。

每周上会小于等于10，你可以视为缓解压力，为利好，数量越少，越利好。

每周上会大于等于10，你可以视为增加压力，为利空，数量越大，越利空。

每周，周中的时候，结合当年的过会情况，若，出现大量枪毙的情况，也是利好。

可以结合此来进行操作。

另外周四晚上一定要看一看，本周已上会的情况，若毫无枪毙，

你估计，周五结束内库要增加9家以上的，周五早上要坚决把次新回避掉，就跟我本周五早上，给你们的建议一样。

B:未来展望

目前深跌的次新，已经给人一种感觉，就是跌不下去了。

不得不说，次新里面存在了很大的机会。

从上周的江丰电子，杭州园林等多个牛股你可以看出来了。

未来板块对应内库大小，

可能会持续走熊。

但是会有多趋势的大牛股走出来。不要秉持一味对次新的偏见，而放弃次新。

行情总是在犹豫中走出来的。
