# 2017-7-3复盘

## 原文内容

这。。本事昨晚的推送，

但是发送失败了，

早上才发现。。。。。。

-----------------------------------------------------------------------------

上周二提到了7月的几个投资机会

上周二本想写，

但是时间较晚，

没写成

周五已经补上了关于次新板块的一些看法，

尤其是新次新时代748666的投资机会。

本来打算周末写的，

但是，光补军师传奇了，

就没时间写了。。。

----------------------------------------------------------------------------------------

简单写一写，中报的几个机会

1：中报的公布速度，

第一家公布的一定会炒，

每年都炒，

中报年报，

谁第一家公布，

一个板是跑不掉的，

但是没想到周五就出了，

我还没来得及写。。。

2：中报的业绩预增，

拼增长复读，

当然也要靠这个业绩增长是靠什么完成的？

有的是因为收购资产，

有的是政府补贴，

有的是卖了房子，

这里面是优缺点的。

我记得以前有一家公司，

业绩增长了几百倍，

原因是卖了地，

卖地这种事，

有可持续性吗？

没有，

所以出中报的那一天，

就是他的最高点了，

后面一路下跌。

3：中报持仓，

这里面有很多道道，

比如看谁进了什么股。

今年的中报可以尤其注意两个，

一：成泉资本，

号称最大的内幕基金，

它入了哪个，

市场一定会炒。

二：养老金席位，

养老金应该是从二季度开始入市的，

养老金概念，

这是一个新概念，

很有可能会被炒作。

而且，我炒作养老金，

利国利民，

你还能特停我？

4：中报高送这个不需多言，

客观自然知道。

想想去年的煌上煌。

-------------------------------------------------------------------------------------------------

写到这，

窗外又开始下暴雨了。

我上周二的文章里，

曾近介绍过这个题材，

可以潜伏

周末看了看新闻和朋友圈，

铺天盖地的洪水和暴雨的新闻。

真给我说中了。

有人说这种对业绩没有影响不值得炒，

我只是想说，

炒股向来炒预期，

水灾的投资标的有两个当年

1，重建，

就是利好各种灾后重建的公司，

记得有一次深圳超级台风，

东南网架应声而起。

2，预防，

就是兴修水利 ，

古有大禹治水，

今有各种大坝河道，

都是一个理，

堵疏结合。

虽然天威浩荡，

有时非人力所能挡，

但是你该干的还是要干，

总有公司要出来干，

你出来接活自然有业绩预期，

这就是为何去年水灾，

安徽水利能两连板。

至于我之前为何说这是个潜伏题材，

很简单，

目前超短生态 环境不好，

提前潜伏方能进退可据。

若你等消息发酵，

明日再进，

极有可能惨遭接盘。

那么这个题材就不能参与了吗？

我觉得应该还有机会，

1:明日早盘集合竞价的时候，

最强股，成了，

周二给高溢价。

不成，集合进也亏不了太多

2:切忌追龙N的追高，

会死的很惨

3:最有确定性的套路，

板块起，

龙一在天，

其他冲高回落，

尾盘买入长上影的龙二或龙三，

次日龙一冲高不成时，

了解你的票。

这三点，其实可以应该用在任何一个板块上。

当然这个题材也很有可能起不来，

谁知道呢。

中证投资参考推的是新界泵业和苏交科。

-----------------------------------------------------------------------------

次新我乐家居最大的炒作点，

其实就是中报业绩大增+次新

所以得到爆炒

次新板块还有没有持续性，

周一全看我乐家居了。
