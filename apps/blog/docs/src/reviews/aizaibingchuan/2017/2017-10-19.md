# 2017-10-19复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/p_1C6VCDq-wR1UK7XYUo9w>

## 原文内容

从大盘来看，之前的缺口成为了强支撑。

今天基本没看盘，但是有一点应证了我昨天说的，板少的日子，每一个股都值得你关注，都有连板的机会。

昨天的最好板的乐金健康直接突破他股性不好的历史，干脆上板，但是他在板上的烂板又体现出了他的股性不好。

农商环保，昨天的环保龙头，尾盘上班，这就是我之前讲过的，龙头战法里的信仰持股，不到最后一科，你根本不知道会发生什么。

以上两股，个人不建议再参与了。

下面要讲讲机会。

海联讯，这股成妖了，弱势中的独立属性的个股，最容易成妖，因为他没有板块，就像黑夜的独行侠一样，独自前行，没有板块的猪队友拖累。

这里有两种参与方式。

1，无脑上。从以往的妖股历史中，你回首去看，无论你是开盘进，还是打板进，都是正确的，妖股的唯一敌人，就是特停和天空。

2，如果妖股参与不进去，那么可以关注板块个股，那么这里可能会说，不是说独立个股吗？资金不会放任一个妖股独立前行的，肯定会深挖的。板块哪里，其实是有一个的，复牌股概念，可以关注下南兴装备。

另外还要讲个就是新华都，这个股我讲过，有两命，因为有三江购物的预期。

这里我还是讲下我对题材高度的预感还是比较准的，我在37.2的时候说川金诺能到55，他就真的到了，后面的黑天鹅公告也不是我能控制的。

新华都我预估八板高度，就真的在第八个板龙低头了，但是我说这这种股有两条命，今天他就迎来了反包板，明天基本能解套前天所有的套牢盘了。

在这里，我昨天列给你们的，零售+次新板块可以继续关注，双十一在前，这个板块还会有表现的机会的。

另外昨天有人问我有个股我没说名字，因为我觉得他现在不是进场时间，虽然逻辑很好，但它现在空头趋势，怕你们进了又套，套了又问，问了又割。

手机打字，没办法打太多，请见谅。

祝各位明天安好。

三个包子在小号。
