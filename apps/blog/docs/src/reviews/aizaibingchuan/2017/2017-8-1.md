# 2017-8-1复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/RtknE0IbDWJarVLJx7mtDQ>

## 原文内容

大盘今天保险的带领下。再次上攻。
但是盘面却比较平均，上涨家数与下跌家数几乎持平。
大盘今天居然突破了上升通道。
这个并不健康，管理层需要的蛮牛，这样的快涨，很容易引发打压。
这不今天又特停了一家。
大盘来到了去年年底和今天4月构成的双顶区域。
这个地方接下来横盘，盘整，不可避免的。
大盘，若要横盘，盘面还要不死。那就只能靠创业板和中小创了。

创业板昨天预判今天是大盘横盘，创业板起飞的。
事实上，今天创业板资金选择了昆仑万维来作为进攻点，发动了4次点火，居然全部嗝屁。。。
我不得不说，这个资金选择的标的也太搓了。从我的股票池里挑一个多好。。
直到资金挑选了这个股的逻辑也很简单，华大基因超预期的连板，带动同概念的次新股封板。
虽然公司实力差了十万外八千里，但是不妨碍炒概念。

本周只有上会了3家，还否了一家。
什么概念。居然只有上会3家。
上一次这个数据是什么时候？而那之后次新是怎么走的？
监管又一次可耻的动用了两大杀器
1：特停资源股龙头
2：减少上会
次新风格即将要转换了。

昨日个股回顾
1：万达信息，全天水下，但是相对于我前晚推荐，昨天的开盘价，整天都是有盈利的，可以不着急出，观察创业板的整体情况。
2：任子行：同上。
3：光线传媒和科大国创，我都是今天进的。博的就是创业板的起飞。今天没飞，等明天了。其中光线传媒的博弈概念比较多，赚钱的几率也比较大。
4：北京文化，今天你要是进了，跟捡钱一样的。后天冲高出。

明日关注标的：创业板，次新股。
理由我已经讲了两天了。

1：索通发展
这是我之前一直关注的，新股中市盈率最低的股，今天放量，按理明天要开板了。但是，我发现，今天好多大V都在推这个股。。我也是懵比，估计明天难开了。

2：基蛋生物
今天的次新龙头股，明天要是低点，我会选择上，毕竟市场在炒他对标华大基因的不开板，华大基因一日不放量，就可以继续干。

再说下雄安。
我今天看很多人说雄安，是超跌反抽。
我认为不然，我认为这是一波趋势的加速。
现在再发现，其实已经晚了，他马上要进入筹码套牢区了。
下面都是震荡博弈了，不是高手很难从里面再去赚钱了。
所以我今天没有再推。
但是雄安里面几个次新股，明天可以关注下他们的低吸机会。毕竟叠加概念。
