# 2017-8-28复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/ILha5NJc2GtE5lw-qlISSA>

## 原文内容

首先，祝大家今天七夕快乐。
虽然这是一个比较乌龙的中国情人节。
中国传统真正意义上情人节，实际是元宵节，
看灯，猜谜，把妹。
你在看灯，我在看你。
但是呢，不要和市场较劲，
市场都说七夕是情人节，那么你就只能顺应这个市场。
下班路上买束花，花不了多少钱。
却能琴瑟和玹然后。

今天大盘又爆了而且还爆量了。
今天这个爆量是意料之中的，
周末各路大神都会复盘周五的这个大阳线。
在这个时候，一致看多，是可以想象的。
所以今天这个阳线，意料之中。

**大盘**

大盘今天会继续冲，意料之中。
但是呢，冲的太猛了。
昨天有说过，大盘目前下有支撑，上无顶。
所以未来的空间充满的了无限的可能。
那么大盘会天天一根根大阳往上走吗？
这当然不可能了。
你唯一能拿来作参照物的，就是这个上升通道了。
今天大阳太猛，步子夸的太大，会怎么样来着？
会扯到啥来着的？
今天大盘，触及上轨后，回落。
后市继续看多，但是对板块的选择，需要谨慎。

**热点**

次新板块：
今天最强的依然是次新股。
昨天我也说了，次新股是目前主流板块，
这个板块里依然还有大量的机会，不要恐高。
恐高，你就不要追高位沽。
高位沽打出空间后。
低位股会普遍补涨20~50%左右。
所以不用担心。

证券板块：
证券是今天大阳的功勋板块。
但是呢，有个问题，今天600亿的成交量。
创下了近一年新高。
明天该板块大概率一根上影线。
继续不建议追高。
另外，我对昨天建议高抛的浙商证券的同学表示歉意，
让你们卖飞了。
但是呢，不用担心。就看这个成交量。明天最多一个冲高。
虽然3000点以后，存量市场会向增量市场转变。
但人家的钱也不是大风刮来的。
明天的承接会有难度。
当然主要的胜负手，还是在于浙商证券。。。龙头嘛。。。。
想想我仅两个礼拜一直推荐你们的证券股，浙商证券。。
一眨眼就成为了大盘连续大阳的龙头。
我对于我卖飞在了前夜，真是心里。。。。

次新板块：
不用担心。受益于上会数量的持续下降，
内库水平持续下降，目前已经下降到阙线。
从股票的术语来说，就是，4跟阴线吞掉了3个月阳线的涨幅。
目前的上会情况来看，本周极有可能IPO数量会下降，小于7家。
SO，本周次新行情的延续，是大概率的。
同样次新板块里，也有很多分支。
后续，次新会在各个分支中轮转。
昨天重点讲的杭州园林（当然我打了错别字，打成了杭州电子。）
今天也是板了。而且成功上位成了龙头。
后续次新大概率会在低位次新中，轮转。
这个很难提前预估标的，只能跟随市场。

雄安：
详见我盘中发淘股吧的贴。看好第一批目录
