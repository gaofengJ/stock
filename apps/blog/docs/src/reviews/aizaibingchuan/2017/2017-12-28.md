# 2017-12-28复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/cllIHeJ68Qe8N0S_ooNbog>

## 原文内容

闲聊

闲聊一：
熟悉我的老铁都知道，
每天我都会把我盘中看到一些机会，及逻辑，
分享在最新文章的置顶评论里。
不定期更新，
有时盘中，有时盘前，有时尾盘。
一般来说，
有我认为比较好的逻辑，我就会讲一下。
比如今天，
在开盘前，和开盘没多久，
分别写了借楼1和2讲了两条逻辑。
借楼1：我是在开盘前写的。
借楼2：我是开盘10分钟看见凯普生物涨到3%时写的。
一共6个票，
50%的涨停几率。
实体涨幅全红，
只要你不追高，日内全是浮盈。
这钱捡了吗？？

闲聊二：
今天晚上出去吃饭了，
年底了饭局比较多，
所以今天只能简单写写了，抱歉。

赚钱效应

赚钱效应和市场热度都开始退潮了。

大盘

大盘在这个区间已经震荡了近一个月了。
A股有句老话，
就做横有多长，竖有多长，
如果在此直接长阳或者长阴变盘的话，
那么行情将会延续的很长。
今天的大盘阳线，和以往的都不一样，
今天是放量阳线。
放量阳线，
即意味这增量做多资金入场，
这是个情绪转好的标记。
但是明天巨量解禁，
下周二巨量解禁，
继续谨慎，是不会错的。
如果真有长阳而至，不用担心，
一根大阳顶三天，
你随便买个热门股，都能随意捡钱的。

热点

主要讲三个热点。

1：有色。
其实有色行情我已经讲了很久了，
基本这波铜带起来的行情，
我可以说我是在相当早的时候就挖给各位了，
只是可惜我自己也没拿住票。
很多时候，
知是一会事，行就非常的难了。
昨天开始由LME铜带起来的行情，
扩展到了LME铝，
所以今天我起床看见了LME铝的大阳，
立刻置顶了留言，
今天铝板块更是全面爆发，
当之无愧的最强板块。
在铝的带动下，
又正反馈给了铜，
然后又扩展到黄金。
当然，
紫金矿业本身的业绩组成很大一部分也是铜。
这基本就是今天这波有色行情的始末了，
不知道您看懂了没有。
关于看期货看股的干货，
我之前有在一篇复盘中写过，
大家可以回去看看，
我也不记得哪篇了。

2：次新。
之前我跟大家讲过一个次新炒作估值的逻辑，
不知道大家还记不记得，24亿。
昨天贵州燃气特停，
我就想到，
这个妖股特停了，
药石科技会不会出来当补位龙头。
但是我没有写，
为啥呢，
我当就看了眼药石科技的市值，
我就放弃了。
你回头找找我那篇复盘，
再看看，说的有没有道理，
是不是又应验了。
目前次新开始极度分化了，
从炒低价，
向炒小流通盘扩散了。
但更多的是低位补涨了。

3：高送转。
逻辑就是我今天白天讲的，
次新和高送转，
是题材炒作指望，炒概念双子星。
次新如此火爆，
当次新开始退潮的时候，
资金会选择下一个目标进行过渡炒作。
高送转无疑是最好的承接标的。
今年一共就是两个10送10的个股，
梅泰诺停牌了，
只有凯普生物了，
而且还是次新，
而且还是新高。
所以今天板了，毫不意外。
今年的强监管之下，
高送转已经成为了稀缺标的了。
往年这个时候，
高送转已经一捆公布了，
今年只有7家，
还有一家停牌，
流通市值小于30亿的只有3家。
稀缺标的。
一旦炒起来的话，
还能扩散到中报填权，
又是一个板块的崛起。
正是基于这个逻辑，
今天开盘不久，
就向大家分享这条逻辑。

最后讲个题外话，
关于涨价概念，
为啥股票不涨。
我3周前写过猪肉涨价，
但是猪肉股也一直不涨啊。
但是突然有一天，
风来了，他们就涨了。
讲真，
我只分享逻辑，
股票的核心还是筹码博弈。
不要来天天问我为啥股票不涨了，
我真的不知道啊。

个股

个股复盘。
我今天买了点鹏翎股份。
不要问我为啥不买凯普生物，
因为按照我的常规战法，
我一定是去潜凯普生物的。
毫无疑问的。
但是我昨天不是写了嘛，
目前的市场倾向于作连板接力。
然后我想了下，
鹏翎股份，
即使没有接力板，
再不济也是个上海物贸、银信科技的图形，
所以开盘就上了点鹏翎股份。
今天收盘一看，
还真是又被我乌鸦嘴言中了。

但这也是经验教训：
不要轻易的改变你的模式。
不要因为他的若干天不赚钱就抛弃了他。
时间轮转，
终会回到应有的轨道上的。
如果我今天不背弃我只低吸潜伏的模式，
那么今天可以收获一个8+%的板。

明天继续持仓观望：
1：观望凯普生物的高送龙头走势。
2：看看有没有反包出现。

末了，
不要问我为啥每天只买一个股，
因为我讲了好久了：
行情不好建议空仓，
管不住手，就小仓试试。
我反正订的计划就是，
管不住手，就一天只买一个股。

个股展望

今天复盘复的晚，
看了一圈，
发现毫无方向。
只看到一个比较好的公告。
这个公告搁在平时一点用没有，
但是现在有个高送转接力次新继续炒作的预期啊，
那么他明天就有点看头了。
其他的机会我晚点再研究下吧，
各位个早点休息吧。
明天早上或者盘中要是有啥机会，
我会再更新在评论里面的。
祝好。
