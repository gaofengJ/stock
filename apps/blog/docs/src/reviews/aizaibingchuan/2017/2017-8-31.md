# 2017-8-31复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/t_yNXumyeHYdT5vF-Gm_IQ>

## 原文内容

**闲聊**

今天惯例，周四，晚上我要踢球，回家就很晚了。
肯定没时间写公号了。
只能简单写写了。

**大盘**

基本和昨天的预判一样。
盘中一度回调的还可以，很多看空的都直喊要回补缺口。
这个中口号其实就是为了爽爽。
无数人盯着呢，轻指数，重个股，精挑细选，照样赚钱。
明天预测大盘低开，收个缩量小阳。
依旧是精挑细选慢慢赚钱。

**热点**

简单点评下:

1: 次新
今天最热的板块，没有之二。
前天次新尾盘杀跌开始，很多人都开始看空次新了。
巴不得一路跌下来，这种人其实是非常二的。
千万不要踏空了就唱空，不要让你的屁股决定你的脑袋。
昨天我就说了，低位股会各种补涨，不用担心的。
前天我发布的数据也说了，本周ipo发布数量有可能低于7家。
但明天是周五，周五资金有抛售次新的惯例。
虽然上周打破了这个惯例，但还是要小心点的。
低位股会持续补涨，可以持续关注小市值的次新。
标的太多，不一一点评了。

2：高送转
我周日晚的一周展望中明确写到。
次新暴涨之后的延续分支就是次新+高送。
杭州园林我也是一直持股未动，这个利润大家都能看得到。
今天更新板块迎来了高潮，明天肯定要分化了。

3：涨价
收到三孚股份股份的一字板，今天涨价题材也涨的不错。
确实考研联想力的。

4：炒名字
今天涨停的股里面，各种名字带科技。

5：钴
钴板块今天也高潮了，得益于龙头的4板。

**个股**

个人操作：

可能关注我的人看到我今天白天的直播了。
因为公司宣布收购的事，一天都在开会。
所以手机看盘，很不适合操作。
早上按照计划，进了皮阿诺。
因为手机操作不熟悉的问题，真是日了狗了。
有一笔追高了，手机操作看不到当时的百分比，倒腾来倒腾去的。

下午原本计划，出一半景家微，进泰山石油的。
因为是计划个股，今天首阴，符合回马枪战法。
但是突然临时看到一个好股，先河环保，就买了先河环保了。
揉搓线+晚上时报会推。

持仓就是这样了。
全是红了，真是悦目，这就是我说的，轻大盘，重个股。

然后说一个题外话。
然后说个题外话，我在这几天连续推荐的杭萧钢构，今天终于预期迎来了大阳。
虽然涨的慢，但也是10个点。
当然我相信，可能拿的住的人不多，因为很多人心态会被影响，目前市场赚钱效应爆棚，都想赚快钱，这种考验技术的股，往往都会拿不住的。
我很久前在其他地方讲过一个短线技术，叫做下影线插5日线上叉10日线交点的战法。
在杭萧钢构上完美体现。

三孚股份，今天下午给了上车机会，不知道有没有人上，这个股是百分百赚钱的股。
非常的好。
理由昨天写了。

立讯精密停牌了，就不多说了。

看好的雄安的几个个股，真是走了相当不错的趋势。
做做高抛低吸也是稳稳的幸福。

明日看好个股：
抱歉，今天没有特别看好的个股。
晚上我看看新闻，看看会不会有看到。
有的话我会写到评论里。
毕竟明天周五了，不开新仓总是对的。
比如上个周五，即使看错踏空了又如何，好好复盘，收拾心态，顺应趋势，看见我这个星期赚爆了没，已经完全忘却了上周5的踏空了。
