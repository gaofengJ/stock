# 2017-8-22复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/7SNsCBhGHW7j9LEWrw_Txg>

## 原文内容

因为出差在外，无法看盘，所以也不知道复盘怎么写。

首先先说了国药股份。
晚上出了中报，没有养老金。
so，明天开盘就走，不要有任何幻想。
潜伏就是这样，成功了，吃块大肉，失败了割块小肉。

简单看了下大盘。
指数涨的挺好。
但是盘面不怎么样。
夜盘期货大跌。
很多原本看来是做短线调整的期货，一下子就变成了M顶。
所以明天资源股（尤其是钢铁）会对指数承压。

另外从技术面来看。
macd看上去金叉了，但是没有，没有叉起来，虽然说明天平开他就会金叉了。
但要是跌了，金叉变死叉，那这个跌起来就要厉害了。

所以，我明天开盘准备清仓了。
原因1是上面写了，原因2是出差期间，没发看盘，减仓也是理所当然的。

只留少量仓位的雄安的钢构不动观察（因为还有不少利润）
其他的股准备都清仓了。
比如中线股，浙商证券，拿了很久了之前高抛过一次，这几天涨的还可以，可惜下午回落了。
比如套了我好久的鸡蛋生物，纯粹是看中他对标华大基因的逻辑买的，其实想想也很弱鸡，看好华大基因就应该直接买华大基因，买个跟风小弟股，整了很久还是原地踏步。
比如国药股份，这个就不再赘述了。

另外讲一个中短线板块。旅游板块。
这个板块每年十一之前都会涨。
虽然涨的不多，但是稳，一般持股一个月也有个十几个点。
每年都是从8.20左右，开始涨到9.20左右，然后这时候新闻会报道：预计十一出行有XXX人，旅游的火爆预期，旅游股大涨。
然后新闻报道后，第二天还有冲高，然后就开始一路跌了。去年这个节点是9.23左右。

虽然我有看好的个股。但我还是建议你们去研究研究。
