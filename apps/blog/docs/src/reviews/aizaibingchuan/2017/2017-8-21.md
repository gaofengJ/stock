# 2017-8-21复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/N2YK8JXWoW7DoIm0AMkHSw>

## 原文内容

开篇闲聊。

我知道，今天你们肯定有人肉飞了。

连推两天低吸杭萧钢构，今天开拉，虽然没有板，但利润也很丰厚了。

而且还要涨。

--------------------------------

说下我昨晚的新朋友

以往我推雄安钢构的时候，从没提过这个，这是我周末在家重点复盘挖掘的个股。

今天推的最大原因除了，我白天淘股吧发贴里写的那些原因。还有一个很重要的点。

就是今晚要出中报。这就是所谓的博中报。

今晚东南网架的中报，简直就是太棒了。完美的内增长。还出了三季度的业绩预增。

除了数据意外，最诱人的就是

变动原因：

1：积极开拓市场，

2：业务增长，

3：盈利能力不断增强，

对于投资客来说，没有比这三句话更诱人的了。

因为这是价投人最看中的一点----------内增长。

三季度才过去一半多，就敢发三季报预增，这是怎样的信心？

这说明，从7.1~今天8.21，就已经完成了8215.82万元的净利润，预计在8.21~9.30会达到。1.03亿元。

说明，即使现在开始，都是躺着的业绩倍增。

又有价投，又有概念，又双低（低价，低流通）。今天上午买的，这把断线，肯定又是赚！到！了！

--------------------------------------

再回到杭萧钢构上，看图，你懂我意思了吗？

---------------------------------------

好了，言归正传，开始复盘。

**大盘**

今天三大股指，全部缩量涨。而且上证严重偏离5日线。

后面两天继续看调整。

但是赚钱效应爆棚。

这是就是传说中的轻大盘，重个股的节奏。

每当这种节奏到来的时候。

不要大盘不出现风险。赚钱效应依旧会持续。

但，大盘风险依旧存在。需要小心，随时离场。

别赚钱的时候没赶上，亏钱的时候每次都不落。

**热点**

我昨天写到次新，我写在了第一个。

今天。。。又爆了。。

最强板块。

热点们太多。不一一点评了。直接说个股。

**个股**

先说说我自己的操作。

今天直接开盘获利了解了周五尾盘买的深深宝A，赚了2个多点。（因为是盘中股，所以买与卖都发在了淘股吧）

本来还感觉良好，特么的他晚上停盘了。

我心里真是，特么的几万头的草泥马奔过去啊啊啊啊啊 啊啊啊啊啊啊。

这要是混改相关的停牌，出来凭借他妖股的属性，要多少个板啊啊啊啊啊啊啊 啊啊。

昨日个股复盘。

今天按计划，买了九阳，明天持股看情况，有盈利就先卖掉。

中粮高开太多，直接放弃了。

开盘低吸了东南网架，杭萧钢构继续持股未动。

今天主要的盈利都来自于这两个股。

明日个股展望。

次新今天高潮啦~明天不摸啦~高风险低回报啦·~

养老金明天预计会收阴线。尾盘买，正海磁材

人工智能本来今天十分看好了一个股，中科曙光，看着太棒了。

结果晚上董秘回答，把所以的热炒点，全部否认。

太神了，搞得我原本要推这个股的，现在也不敢推了。。。

**雄安**

优先看雄安装配式建筑。

个股依然是昨天两个。

但是这里有个区别，东南网架是利好公告，要么下面是加速，要么就是兑现。需要看盘面拿捏。

杭萧钢构，26号中报，依然可以低吸，加仓，博中报！

其次，水处理。

明天博天环境还要回调。回调深的话，可以继续低吸。

水处理和装配式建筑将会是雄安未来的两大核心。

低吸！高抛！踩住节点。

盈利不是问题。

-----------------------------------

话说为啥昨天没有详细写东南网架。

因为，时间不够了啊。我10点开始写，一写就到了11点，还没写完的，不发也得发了啊。。。。。。

还有，我明天开始要出差3天啦~

另外，今天的封面是东南网架参与的杭州奥体中心，漂亮吗？

杭萧钢构，东南网架，嘿嘿
