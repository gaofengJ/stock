# 2017-11-23复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/5OvPzHoZr-D0xccOMf3YZg>

## 原文内容

闲聊

今天下午一直在开会，
一直开到了5.30。
中间间或瞄了两眼大盘，
发现务必惨烈。
立刻尿遁出来用小号给各位发了文（如下）断头铡刀。
很多会觉得我大惊小怪。

这里我要讲一下。
很多人问我逻辑是怎么培养出来的。
我回答是亏出来的。
然后人家说，你说笑了。
可我并没有说笑。

15年大盘见顶到现在的所有千股跌停，
我一次都不落的满仓全吃了。
这表里没统计全，
16年后面还有几次。
你跌停吃多了，
稍微能动动的脑子话。
你会发现，
任何事情之间，都是有联系的。
就是所谓的逻辑。
最让我印象深刻了的就是断头铡刀了。
所以我赶紧码文发了出来。

然后回去开会，
然后去踢球，
然后回来就好晚了，
开始写复盘。

大盘

看看这张图，真是呵呵了。
大部分人今天的自选几乎全绿。
这种时候段子手就又出码了。
说：我有个持仓股没绿恩，非常的欣慰。

说说K线。
大盘今天一阴穿N线，
这种形态叫断头铡刀。
我还是那句话，
每天会知道明天会发生什么？
每次我都去从历史里寻找参考。
温故而知新。
读史可以明鉴。
你可以自己去翻翻历史上那些端头铡刀后的走势，
我就不一一截图了。

11月31号，
是公募私募的绩效考核日，
基金经理们今年能拿多少奖金，
都是以这一天做结算的。

12月31号，
是公募私募的年度排名日，
12月怎么涨，怎么跌都跟奖金没关系了，
只关系到排名，
排名关系到明年的销量。

抢排名，
除了做大自己的市值以外，
还有个办法，
就是，砸对手基金的重仓股。

我想你最近一定听过一个段子：
2800建的仓，套在3400点。
这个这个段子，
去年也是一样的。
去年11月，
大盘冲击3000点，
那会的时候，
这个段子是这样说的，
2800建的仓，套在3300点。
今年又回到了去年相同的时间线，
发生了相同的情况。

各位还记得，
我之前讲过的，
一个朋友，
以前听我介绍买票，亏成狗的那位。
我给你看看他去年11月之后的账户曲线。

有人说现在gjd控盘不会让你跌的。
我觉得好奇怪。
去年11月的时候，
641也在任啊，
gjd也在市场里啊。
但是根本挡不住的。

可能看到这里，
你会觉得悲观。
no no no。
我的意思，是，
你任何时候要对市场行情有清新的认识，
不要YY有人会来救你，
gjd来救你，
某路游资来救你。
第一，你不是城堡里的公主。
第二，虽然有时候，
会有人来救城堡里的睡美人的时候（江南嘉捷），
顺带把你给救了，
但是我觉得，
你有这逆天运气的话，
买彩票可能是发家致富更快的路子。

当发生不好的情况时候，
清仓，一定不会错的。

举例：
上周末看到利空新闻，
就空仓了，
当发生你看得懂的情况的时候，
再去操作。

举例：
周一清仓后，
大盘暴跌，
但是看到抄底资金进场很坚决，
遂买入巨人网络，
隔日超短赚4%，
然后大盘开始稳定，
买啥赚啥。

我昨天写的很明确。
1：大盘调整。
2：什么样的情况能操作，
什么样的情况需要离场。
今天有人给我留言，
不懂啥叫黄白线。。。

热点

没有东西东西可以点评。。。。

个股

个股复盘。
上午中广防雷开板，
凯普生物冲高回落。
黄线白线都在水下，
黄线更下。
我就一点点不停的出了，
最就出光了。

但我也是空仓就难受的人，
但是我知道大盘不好，
就买了地产股，作防守。
我也不是神仙，
我也想不到大盘会崩了。
不过阳光城还是相对比较抗跌的。
如果不是大盘崩了的话，
作防守一点问题没有。

个股展望。
今天这种日子，
还展望个啥。。

明天计划，
阳光城先拿着看看，
如果大盘无量反抽，
我就走了。
如果大盘放量杀跌，
那我也走了。

其实今天对大盘暴击最大是隆基股份跌停了，
这是今年最牛逼的白马趋势股，
3倍涨幅。
所以恐慌情绪扩散到了整个白马趋势板块。

如果你明天想买股，
也请等到大盘稳定的时候。
虽然我也有看好的板块，
但是我不准备说了。
如果大盘不好，
那么基本买啥挂啥。

而且明天我们公司要搬家。
从我上班10分钟路的地方，
搬到25分钟路的地方。
明天要收拾东西，
基本也没什么时候看盘了。
而且，明天周五。。。

尾盘可以分仓买点次新，
有高送转预期的那种，
多买几个，
说不定晚上就给你公布高送了呢？

而且，
这周新股大概发行5家，
甚至更少。
你说万一发个三四家，
那么下周一岂不是要整个高开？

各位，明天祝好。
