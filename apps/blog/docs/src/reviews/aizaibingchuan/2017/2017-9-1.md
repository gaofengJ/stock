# 2017-9-1复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/IbKae_6wd1RsugAx0Ek3Og>

## 原文内容

今天周五，也没啥复盘好写的，惯例会在周日去做详细复盘。

今天盘中真是跌宕起伏。

我说的不是盘面，是我的心态。

因为昨天是下影线。

今天低开是比较合理的。

但是呢，今天神华复牌了大盘放量上涨。

我一下子，就懵逼了。

跟我昨天的预判不服，那这就是说，我预判出现了问题。

那么，你昨天基于你的预判做的计划，都有可能是错的。

那怎么调整呢？

如我盘中发的贴。

我迅速的清空了我所有能清的股票。

然后开始重新整理思路。

因为你持股的话，很容易被你的屁股决定你的脑袋。

因为我还达不到不以物喜不以己悲的圣人境界。

清完后，我迅速冷静的分析了下盘面。

大盘带量上攻，资源打头。

大盘即使下午回落，也不会出现系统风险。

不应该空仓，应该博一下。

然后我就开始精选标的了。

前几天有个股，景家微，你们有看到过的。

今天出了研报力推，“买入评级”，高开3%

早上下跌的时候，把我洗掉了。

但是后来拉起来的，均线很稳。

我判断很有可能今天会有一个加速板。

临近中午启动，跟随启动，剩余仓位满仓杀入。

可以看看上午那个下杀的尖点，把我洗出去了。。。

还好加速板如期而至。。

顺便可以看看，今天基于错误的预判，做出的操作，最终今天证明是错误的交易。

中午复盘的时候看到煤炭板块整个都飞了。

决定下午买入惠达卫浴，理由见下帖https://www.taoguba.com.cn/Article/1772599/1

但是我已经满仓了景家微了只能再转钱进来了。

这是我从股灾以后，第一次往股市里面转钱。

我上一次往股市里面转钱是2015年6月10号！

希望这次不会一转钱，就开始坑爹之路。

SO，这个跌宕起伏的心态能看明白吗？

看上看多，买入三浮

然后看空，先河环保二话不说就出了，皮阿诺平盘就走了。

然后极致看空，非卖品杭州园林都给我卖了

然后有看多，满仓追进景家微

然后极致看多，还加钱进来了。

最后，下午大盘入我上午的与一样，果然回落了。

但是我的股都没有下来。

非常好。

今天虽然三浮一晚大面，

虽然杭州园林卖飞

虽然先河环保大卖飞

虽然皮阿诺超级大卖飞

但是也没人能保证总卖在最高点，对吧，有赚钱就好了。

毕竟还有一个重仓板在手上嘛。

所以，有的时候，看不清状况，你就空仓好了。

比如我上周末聊过《关于踏空》

没关系，先卖光了，再整理自己。

看看我上周五的超级大踏空。

不要悲伤。

一点不妨碍我这周连赚5天的钱！

最后祝各位周末愉快。
