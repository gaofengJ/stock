# 2017-11-30复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/RuvA6529yK6WQ4L4loPXmw>

## 原文内容

闲 聊

闲聊一：

今天看到一句话，
Non loqui，fortunam facere. Est optimum.
其实翻译成中文，
就是简单的闷声发大财。

今天被朋友教育，
大致如下：

就是说为啥太阳电缆今天走的这么差，
说我太高调了，
好股票都是在沉默中走出来的。

比如张家港行，
昨天拉板烂板，
今天平开，连板。

昨天中农，
今天走的犹豫无比，
也连板了。

（插一句：我昨天提醒该朋友尾盘抢次新，
他二话不说就抢了中农。。）

插完接着说，
意思是说昨天太阳电缆没有出龙虎，
你就不该说是你点的火，
而且大家都看得出来，
昨天太阳电缆点火段几乎拿不到筹码，
板上拿货的那两个大单是不是你的？

我说不是，
他说那么好类，
最大的筹码又不在你手里，
你一个劲的喊啥，
板上拿货的那两个大单，
今天肯定要开盘跑的。

只有低调，
昨晚你提都不要提太阳电缆，
让他整个平开，
让那个大单闷了，
今天才有接力的可能。

其实他咋呼了我半天，
说的我一愣一愣的，
我想说，
这个分析的好像特么的有那么点意思，
当初中科信息的前4个板，
每个都走的无比揪心。

然后我被他讲的一点心情都没有了，
握牛战法都握不起来了。
下午太阳电缆在0轴的时候，
我就一点一点出光了。

闲聊二：

上个周六的时候本来已经把这个月的月结，
提前写掉了。
11月提前月结（2017-11-25 爱在冰川复盘）
本来是准备持续空仓的，
但是想想天天空仓复盘还怎么写。
所以我又火中取栗了一把。
虎口夺了点食下来，
纯属运气好。

今天要是开盘开盘的时候砸了太阳电缆，
那么今天就不是个小回撤了，
而是60+了。

大 盘

赚钱效应

其实，
你如果每天看这个部分的话，
你会发现，
最近一段日子，
赚钱效应基本都是绿的，
如果这段时间，
你还在天天强撸，
那么大多数人，
基本都是在亏钱。

今天炸板率，
再次超过50%
这种一般都是危险信号。

大盘基本按照我周一的预演，
连续走了三天了。
昨天和今天，
都是上冲过5日线后被打下来的，
这也跟我周一复盘说的一致。

这里你可以看好，
5日线，
和3300线已经越来越近了。

下面大盘会继续盘整，
直到出一个阳线，
上探10日线，
这才是试探压力的关键点，
在确定方向之前，
抓紧机会再火中取栗一次了。

但是这里有个风险点：
今天是11/30，
基金经理们的绩效考核日，
所以说，
他们今年发多少奖金，
都已经订下来了，
下面一个月，
是他们冲排名的月了。

我之前讲过，
冲排名有两个办法。

1：提升自己的收益。
2：砸竞争对手的重仓股。

如果大家都用第一个办法，
那场面祥和。
而从历年看，
基金经历最爱用第2个办法。
所以这是个风险点，
随时要小心。

所以，
想开仓，
尽量在下午，
盘面比较稳定后，
再开。

（这也是为啥今天的两个连板股，
都是下午才有资金敢去点火封板做2板的。）

热 点

今天自然板又是少的可怜。

但是，
我之前讲过。
板少的日子，
里面的股大部分都是精品，
后面一定会有连续效应的。

今天的涨停里面，
超跌反弹依旧是主基调。
这个我讲了好几个星期了。

然后一个新的热点，
就是，
次新股出连板了。
这是一个重大信号。

想想之前的次新，
今天板，明天地板。
但是现在出现了，
今天板，
第二天继续连板了。

这是一个比较重要的信号。
要知道，
整个次新板块，
都已经处于超跌的状态了。

个 股

个股复盘

今天一整个上午，
除了看盘以外，
基本处于握牛中。
当然到了下午，
经过交流之后，
我就握不住了。。。

当时结合盘面，
我想了三个潜伏逻辑。

第三条就是，
我最近一直讲的资源股涨价，
我看到铁矿石最近创新高了，
铁矿期货涨了20%了，
而铁矿石的相关股票，
还特么跟月初一样毫无涨幅。
所以我就买了点金ling矿业。

你看下最近铁矿石的报价，
天天都在涨。。。
然鹅，
我也追高了。。

有的时候就是这样，
三个逻辑，
你三选一，
就选到最差的那个。。
很多时候你缺的不是水平，
而是运气。

持仓如下：

对了，
我还盘买了点昨晚说的金牛化工。
不成想，
盘中甲醇期货大跌。。
明天早上起来，
再看下夜盘的甲醇走势了。

金岭矿业的明天操作，
同样看铁矿石价格的走势作决定。

个股展望

1：开板新股

开板新股出了连板股。
这个信号很不一般。
下图是我4个月前画的。
但是这个圈不是绝对的，
有时候，
会从黄色直接到草绿色。
但情绪的循环是差不多的。

既然出了两连板的个股，
就需要重点关注了。

股票池，
今天涨停的三个次新，
都很好，
最看好两个。

张家港行：
这个板了，
能带银行，
带了银行能护盘。
游资可以放心大胆的接力，
又赚，又护盘。
典型的，
面子里子一起搞。
最经典的是，
他不同于其他银行次新，
他的解禁要到2020年。。。
醉了。。

佛燃股份：
这几天有个概念炒的很火，
天然气涨价，
那么叠加次新的佛燃和贵州燃气，
都有机会的，
佛燃的板少，
所以机会更大，
但是风险点是，
盘子有点大了。

2：涨价

还是涨价概念，
详细理由翻看我这几天的复盘。

在这里结合1和2，
你也可以去找下有涨价概念的次新。
比如我上面提到的天然气，
你也可以再看看其他的。
比如我看的铁矿石目前涨的挺好，
但是还没有新闻开始报道。

3：高送转

其实逻辑我上面讲过了，
梅泰诺今天巨量套牢，
明天想跑，
只有一条撸，
就是狂拉凯普生物，
围愧救赵，
不然今天的接力资金全埋。

当然也有可能，
两边资金一起狂奔，
那么高送转明天全军卒。。。。

今天晚上有点事，
所以回来晚了，
写晚了，
抱歉。

以上，
祝各位明天安好。
