# 2017-8-27复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/7-5QC_YwXOPpjgrAl3TjSw>

## 原文内容

又到这么晚了。。真是抱歉了。
这会都22.24了，才开始写。
就不闲聊了，直接开始吧

先说大盘

大盘突破了了9个月形成的顶。
并且是有效突破，并且突破3300后留了30个点的空间。
就是说，大盘后续回踩，也风险不大，下有支撑。
向上则要追述到股灾1.0~2.0之间的套牢盘了。
非常久远，压制效果弱。
SO，周五这个意料之外的大阳，是真正的鱼跃龙门的般的阳线。
之前，可能是存量博弈市场。
之后，可能会事增量市场了。
虽然未来务必美好。
但是突破必有回踩。
只是来的早与晚的区别。
一根大阳顶三天，下周前三天，可以放心撸。

再说个股

大阳之后怎么做比较好？
就是跟随市场热点。
但你又担心市场热点接了会不会高位接盘。
这个不要担心。
危险与不危险是相对的。
认识群居动物。
人多的地方就会给你安全感。

个股列表：

次新+5g：
纵横通信：龙头，没啥说的，看勇气值了。
华脉科技：踩在风口上的中标联通，非常看好。
世嘉科技：通讯基站+次新。
淘股吧吹票。
但是我分析了下，这个股超跌。
向上可以有2.5个板的空间，而且解禁要到明年。
高公积金，有高转预期，以前高送预期的龙头股。
周五放量栏板。
注定周一无法高开。
开的不高我不会直接上。

次新+低价：
目前龙头打出空间后，低价次新一定会来一波的。
理由我之前的帖子就分析过了。
之前写过了两篇帖子。
从周五来看，资金确实在发动低价次新，连迪生力都能封板了。
这个股票池我就不列举了。

次新+高送：
次新一般炒起来后，会衍生炒高送。
但是目前没啥单纯的高送次新在炒。
都是叠加叠加再叠加的。
比如那些通信老次新，都叠加了填权。

主要关注两个：
1：杭州电子：中报高送的龙头。
本波次新行情涨幅十分有限，上周买江丰电子的原因，也是因为他身为龙头却没涨幅，一旦点火，很容易引起联想，引发跟风。
2：中装建设：之前的填权次新龙头。
本波也是涨幅十分小。
而且他还有1块钱公积金，还有10送10的潜力。
目前股价才10快，符合上面说的低价次新。
新洪泽，龙神，之类的也要加在股票池里观察。
如果资金发动次新高送板块的话，龙头可追。

证券：
我之前一直大力推荐的浙商证券周五板，有很人都吃上肉了。。。
但我不建议追短线，明天建议高抛，后面继续延5,10日线低吸。

雄安：
周一雄安可能要爆，这个板块要重点关注。
尤其是嘉鱼股份，周五唯一雄安板。
周一要是开的我不高，我也会小仓位追一点。

其他雄安个股关注：
1：杭萧钢构
2：惠达卫浴，通和科技，科林电气（叠加了次新）（重点观察）
3：看异动，真的雄安板块太大，很能提前选标的。还是要看异动。
