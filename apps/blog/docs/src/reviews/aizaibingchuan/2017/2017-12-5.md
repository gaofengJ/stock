# 2017-12-5复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/qiD5yyUf9x9bN5PtjH1K7w>

## 原文内容

### 闲聊

闲聊1：
今天市场热传，一代神话佛山路，被罚了8000万。
经过多方确认，被罚的确实是佛山本人。。

佛山的模式，一直都是市场模仿的典范，
但从未有人超越过，
最重要的是，他从不吃罚单。
直到今天。。
神话打破了。
也吃罚单了。

关于这件事的点评，市场上很多，
我就不来做重复的点评了。

我只是想说，证监会的出发公告经常出，
但基本没人，偶尔被人看到，就是一顿炒作。
这是4天前的公告了，
今天在互联网的帮助下，迅速传播了。
这次主要是一代大佬被罚了。
所以特别热闹。

但是呢，其实大佬们大多都被罚过。
常在河边走，哪有不湿鞋。
相信我，过个两天，你就忘了这个事了。

7月初的时候，荣超中心被罚款1.1亿！
当时同样是群情激奋，感觉世界观都崩塌了。

但是，过了两天之后，
该低吸的还在低吸，
该点火的继续点火，
打板客们继续孜孜不倦的打板。

世界没有任何变化，
因为，决定这一切的，制度，没有变。
所以你也不用操啥心，该干啥，干啥。

你们可以看到，
今天新劲刚，一共5388万的成交量，
佛山一个人就吃了4300万。
人家刚吃完罚单，都在该干嘛，
我们还为别人操心个啥哟。

闲聊2：
刚才在看CBA，山东打新疆。
最后几分钟比较精彩。

突然想到，
由篮球比赛中呢，
就能想到一些股票中的操作细节。

比如，你领先两分，
这次进攻你打进，就是领先4分，
这个心里优势就出来了。
但是，你打的没在意，没进，
给对面一个快攻，就变成了打平。
一来一去就相差了4分。
而不是你想想中，只是丢了分。

引申股市想和大家浅谈一下，
交易中的一些细节。

我有个朋友，看见我昨天买了太阳电缆，问我能不能买，
我说这个下午承接很强，很诡异，可以尾盘买。
然后他尾盘竞价进了。

今天早上我出掉的时候，
顺便跟他讲了下，那会还有+1.4%左右。
但是他没出，觉得赚的太少，才1个多点。

最后结果呢，他在水下负一个多点割掉了。
我问，为啥不早点出，
他反正只亏一点，无所谓了。

我说这个想法是很错误的，
目前的行情很差，能赚一个点，也很不容易了，
而且从赚1个点，到亏1个点，
你实际上是少了2个点。。。

如果你每天能多挣2个点。
一个月22个交易日，你可以多获得+54%的收益。
古语有云：不积跬步，无以至千里。

### 大盘

赚钱效应今天场面相当灿烈，
但是你有没有发现，
数据居然不是极差，而是差。

数据很多时候，总是给你反应，真实客观的事情，没有主观情绪。

因为，今天尾盘的时候，
指数继续横盘而资金开始抄底小票了，
十几个股跌停打开。

因为指数严重失真，说实话，已经失去了分析的价值了。

今天大盘在极窄的空间内，竟然放量了。

下面这张图，是我在上周一的复盘做了几百字的分析后，画的。
而，时间已经过去了6个交易日，
大盘的走势如下，你比照下上面的图。
竟然还在按着我的图走。

我每天都认为，明天要变盘了，变盘了！变盘了。
不是上拉到10日线再次反压，
就是下破3300走市场底。
但。。他就是不变。。。
我也感觉超级无奈。

我想说的是：
明天，真的没有空间了。。
大A啊，您今天放量了，
明天不可能窝在更小的地方了。
真的，该变了。

### 热点

看到今天的热点。
我突然想到一个事，
那些每天复盘，就是点评涨停板的复盘。
最近是不是好轻松啊，
因为基本没有啥涨停板。。

今天唯一一个城板块的热点，就是军民融合。
林州重机命好，刚复牌，
就碰上军民融合出新闻，板的无悬念。

新金刚的是以前的老龙头了，
今天凭借记忆，被小散户就把价格抢上去了，
然后被佛山一把梭了。
其他跟风股全挂。。

其他的，就没什么好点评的了。

### 个股

个股复盘今天原计划。
开盘观望下军工的走势然后作决定（昨晚有写）。
结果早上军工开盘走的强的超出了我的预期。

看见新劲刚秒板大单堵死的那一刻，
我直接追了新雷能，
然后静静的看着新雷能板，
那一刻，我差点以为我悟道了。

然后新雷能炸了。
我就把手上的太阳电缆也出掉了。
后面新雷能回落，
浮盈 7++硬生生浮亏，
心里也是奔过了一万头***。

持仓如下：
目前我能做的只有，烧香拜佛，
跪求明天林州重机能冲板。
带带我这可怜的龙四吧。。。

个股展望。
在展望这个股之前，我要讲个事。

明天是个又赚钱的效应的日子，
欢迎市场来打我脸。

我敢说句话的原因是。
看下面这个指标今天指数破了黄线。
可以看下历史上破了黄线的日子。
对应黄线指数K线图的日子如下，
第二天基本全是涨的。

所以，明天我认为，可以抄底。

那么抄底抄什么？

1：涨价题材

A: 昨天复盘里，给大家推的涨价题材中，
新提到了一个板块，猪肉。
其实猪肉最近一直在涨。
可惜今天没有先买点。。
晚上三大报头版头条在推。。
这个板块，最好的个股是华统股份，
可惜被坐庄了，
虽然有机会，
但看看十大股东里面的资管和信托，
还是不建议碰了。

股票池：
除了3打包推的3个外，
比较好的还有唐人神（超跌叠加填权）、
傲农生物（超跌腰斩叠加次新）。

B：水泥
这是今晚水泥网的新闻。
反正水泥又涨价了。
标的还是那些，不重复说了。

C: 现货。
钢价格也在涨，包括铁矿石也在涨。
但是呢，今天盘中期货都杀了，
所以钢铁板块深度回调。
后面这个板块还会有机会的。

D: 讲个冷门涨价的。。。
木头在在涨价，
当然这个太冷门了，
你听完笑笑就好了。

2：新股

新股最近杀的太惨了。
有反弹的需求，详细理由，昨天写过了。

今天盘中，宏达电子更是摸板了，
但是不出意外的炸了，
原因两个，
1：自身原因，开的太高，成交量不够；
2：行情不好。

但是这些都说明天有资金想参与新股抄底了。
当然这个板块，我眼里，只有一个个股，
那就是封面里的绕地球两圈。。。

时间又好晚了，
其他板块，那些我讲过好久的板块，
我就不赘述啦。

各位祝明天能赚钱。
