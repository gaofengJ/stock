# 2017-11-17复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/jXLcycKAAOf4W7jDeuJvqw>

## 原文内容

三个字！亏成狗。
昨天发起了一个投票。
投票非常惊人居然是一比一。
上证50今天又新高了，但是，市场哀嚎遍野。
次新股领跌全场。
第一根阴线，还可以理解为回调。
第二跟阴线，是因为特停杀跌。
第三更阴线，就很不一般了，因为这个是喊话杀跌。
所以我昨晚写到。
今天如期一根大阴线。

早盘的时候，很多次新，都给了你跑路的几乎，因为连续两天杀跌，大部分的次新，今天基本都是平开的。
这个时候果断跑。
那你今天基本还亏不了。
这是我昨晚给各位的策略我今天开盘清光了股票，包括阿石创小仓买了3个芯片，统统吃翔这里要说个事情。
目前的市场，情绪悲观，你是在想操作，就是小仓，试一下。
输与赢，都是学习的机会，经验的积累。

媒体喊话永远是最强1击，不要怀疑昨晚新华网喊话贵州茅台。
茅台今日资金巨量出逃。
当然始作俑者，新华网也没好到哪去。

我没办法预测走势，更没办法操控走势。
我有的只是我对市场的一些理解，一些炒作的逻辑。
我只会给大家分享我的逻辑，但是什么样的逻辑会兑现，什么样的逻辑会兑现到什么程度，其实我也不知道，这些事情只有市场说了算。

不要让杀跌影响了你的心情。
如果你还郁闷，那建议你看看那些天天直播的公众号。
点开满屏幕的（上图为其他公众号）说实话。。
很多时候，我亏钱的时候，我只能靠逛论坛来找找心里慰藉了。
上个川金诺黑天鹅的时候，那几天亏的我家都不认识了。
也不知道怎么熬过来的。
熬过去之后，市场轮动，账户又是新高。
现在，要开始新一轮的熬一熬了。

看到这，我知道你肯定要问，明天该怎么办。
我只能跟你说：明天不开盘，好好过周末，除了股票，还有生活。

对了，有人跟我说，亏了很多钱，都不敢跟家里人说。、、怎么办。
其实我有个很好的办法，你可以跟你家人说。
我买了个乐视，停牌了，然后找几篇乐视的帖子给你家人看。
然后表情悲观。
但凡你家人还正常的话，那一定是反过来安慰你的。
而且停牌了嘛，绝对不会跟你说，出金吧。
你还能继续在里面玩，万一买到一个江南嘉捷了呢。
