# 2017-6-19复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/lz1S4vqc0WxV9e8LmCyg4w>

## 原文内容

〇：引语

彼得·林奇凭自己几十年的从业经历告诉你："开始寻找涨十倍的最佳地方就是在自己住处的附近：不是在院子里就是在大型购物中心，特别是你曾工作过的每一个地方。普通人一年中可能会碰到两三次有希望的赚钱机会，有时可能还会更多。"

一：不经意间的发现

昨晚和小伙伴打王者打的有点晚，陷入了高位盘整区，高位横盘不涨就要跌。

早上起来出去吃早饭，走着走着，看见如下场景。

看见这一幕，充满了社会责任感的川哥，分分钟就冲上去扶起他。老人不敢扶，这个还是没有问题的。

二：敏锐的嗅觉

扶正车之后，满意的拍了拍手，转身准备离去吃早饭之际。我脑海里瞬间划过一道闪电。

周五，摩拜单车又双叒叕获得融资了，本轮融资的金额高达6亿美元，据说破了单笔融资的行业记录。

腾讯领投，工银国际、交银国际、Farallon Capital等重磅机构入场，TPG、红杉中国、高瓴资本等老股东增持。

按照腾讯科技旗下媒体“腾讯创业”的说法，腾讯已经是摩拜单车除创始团队外的最大股东。

据摩拜介绍，这笔融资将帮助摩拜单车在三个方面提速：

加速国际化进程，计划到 2017 年底服务全球 200 个城市，将出行智能解决方案带给更多用户;

携手全球领先的物联网合作伙伴，加速推动移动物联网技术的进步和实景应用;

在人工智能、智能硬件等领域加速战略布局和技术创新，提高技术壁垒、扩大领先优势，给用户带来完美的体验。

是的摩拜要全球化，要在全球都推广共享单车。川哥认为这简直是个氢级别的题材，

A: 产能输出：将国内过剩的产能直接输出国外。

B: 赚取外汇：国家外汇储备逐年降低，像共享单车这种，国内人民币生产，投出国外，直接开始稳定持续长期的赚取外汇的，将会大量受到政策扶植。

C: 想象空间 :  摩拜单车是目前行业唯一拥有人工智能大数据平台的企业，通过全球首创的智能共享单车模式，为每一辆单车独家配备了“北斗+GPS+格洛纳斯”多模卫星定位芯片和物联网芯片，实现对车辆位置和状态的实时精准掌控，每天处理超过5TB的出行大数据。这不仅在整个共享单车行业中独树一帜，也为摩拜单车的未来前景开辟了巨大的想象空间。

而且一直以来，川哥都认为，共享单车这个行业，早晚就会只剩两家摩拜和ofo。而且从两家公司的定位来看，摩拜未来会把OFO甩在身后。例如下面这些新闻。

而且共享单车的盈利模式非常清晰，光是1亿注册用户的299亿押金，一年所产生的利息，都已经远超了很多老牌互联网公司了。

三：动手及分析

既然看好摩拜单车，那怎么从中去寻找投资标的呢，川哥连早饭都不想吃了，说动手就动手，从摩拜单车的每一个细节去寻找。

气动升降座椅，航天级钪合金铝合金车架，空气防爆轮胎，富士康代工。找不到投资机会，so pass

智能车锁，甭看了，摩拜的核心技术，不可能直接采购的。找不到投资机会，so pass

太阳能电池板！！！！这种东西，摩拜绝对不可能自己做，一定直接采购，这里面肯定有机会。

迫不及待的回家查资料，早饭都不想吃了。

四：太阳能电池板

百度竟然搜不到摩拜的太阳能电池是谁家的做，搜出来的全是广告，各种各种的小公司宣称这板子是自己做的。竟然找不到投资标的。

但是这个难不倒川哥，川哥就是应用物理系毕业的，同学在光伏行业打拼的不计其数。立马抄起手机，问了一位曾经在无锡尚德工作，后来自己出来作光伏贸易的，对国内各种太阳能供应商都门清的大咖同学。

汉能不用看了，港股珈伟？这不是300317么，我前两天找石墨烯标的的时候还看见过他，原来人家主营就是光伏产品。

而且大咖告诉我我上网一查还真是3月13日，上海市质监局同上海市自行车行业协会率先编制出了共享单车系列团体标准，其中：在服务软件要求里，我们可以注意到，标准（讨论稿）要求共享单车企业的软件系统应具备定位共享单车位置功能，这样看，太阳能电池板几乎会成为以后每一辆共享单车的标配。

紧接着大咖又给讲了个题外话，一瞬间我都懵比了。二级市场炒过共享单车，炒过共享汽车。这个共享太阳能电动自行车绝对是正在研发中的新概念啊。但是这个概念还没出来，那我们先不研究，但是有一点肯定的是，这个概念就算出来，一定会利好太阳能光伏行业。

那么让我们来细细研究下，跟大咖聊天中获取到的信息。

五：伽伟股份

A：共享单车契合度

出于公司的保密协议，并不能直接公布，我做了谁谁谁的太阳能电板。但是字里行间，总是生怕你不知道，我做了龙头公司的太阳能电板。共享单车的龙头公司是谁？不用言语了。

另外在6-5号的机构调研结束后的记录表中，更是明确的写到关键词提取：行业龙头公司----摩拜，的最大供应商。可以想象，在随着摩拜进军全球的大背景下，珈伟的太阳能电池板也会跟随进军全球。

B：处于业绩暴涨的前期

公司投资的锂电池将于9月量产，同时一直没有业绩回报的石墨烯业务也将随着销售的展开与下半年产生收益。

C: 高送转（填权）

公司自6-5除权以后涨幅仅6.5%，对比目前市场热炒的锂电+高送，石墨烯+高送，属于低位滞涨。

而且，之前公司原本是10送28，被喊话后修改成了10转8，股价一路下跌。但是这其实是一个短利长好。10转8以后，公司依然还有4.2块的公积金，下一次的高送转预期不减。

D: 技术形态

公司自监管打压高送转分配方案后，一路下跌，在目前高送转行情反转到来之后，股价已经企稳形成了非标准的头肩底更接近3重底的走势。同时右肩高于左肩，成交额开始逐步放大，说明已有资金开始介入。

从筹码峰来看，股价向上有两个半板的空间，才会遇到前期的套牢盘。随着横盘，均线走平，短期已经呈现多头排列走势。

E: 承诺不减持

恩，不减持也勉强能算是一个利好。

F: 一张诡异的图

在伽伟股份的官网上面产品介绍有一栏写的是，特殊定制，配的图片是电动车。这个难道是在暗示我们他在参与共享太阳能电动车的特殊定制太阳能电池板？额，这个脑洞有点大了，请自行脑补。

六：结语

回到开篇处彼得林奇大师说的话：十倍牛股就在身边。没想到路遇一辆躺地板的摩拜单车，竟然让我发现了一个如此之好的投资标的。简直是价值投资的典范，又遇上了短期的热点。

什么？共享单车进军全球化这个消息没有发酵？没关系，王者荣耀也是上周一出的新闻，上周四才启动的。共享单车全球化这个概念，上周五出，本周二启动，恩，差不多。
