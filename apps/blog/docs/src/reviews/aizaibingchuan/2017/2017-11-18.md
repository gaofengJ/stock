# 2017-11-18复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/tO6hPvPizkhnFdjnwYH_lw>

## 原文内容

今儿我写了个高送转的战法。
里面有很多套路可能路人皆知。
还有些套路知道的人比较少。
但是我以前也写过。
还有些套路今儿我是第一次写。

基本上，这些套路，我每年都在用。
也是大A股最具有确定性投机机会了。

下图是我某模拟盘的曲线。
2个月多最高+160%。
其中这一段的曲线贡献，来自于杭州园林。

当然我自己本人账户的这一段时间的主要利润也来自于杭州园林。
关注我早的老铁可能还记得这一段历史。
我把他们简单汇总了成了一篇战法。

大家都知道，我写公众号至今，从不互推，从不打广告。
就连腾讯的自带广告，都被封了。。
各种专题战法也给各位写了不下十篇了。
每天复盘里夹杂的干货更不是不计其数。

周末友人跟我说，不接广告你是SB吧。
我说我怕我的粉丝被坑了。
友人说，那你把你的战法卖钱吧。
我突然觉得这个提议还不错。

今儿这篇战法，我给他标了个价。
目前市面上，凡是各大V卖的战法，从168~888不等，都有。
但是水平却良莠不齐。
放心，我不会订这么高。
打骨折价，88。

这个战法基本每年可以用2次。
一次可以跨度两三个月。
而且，我这是我第一次卖战法。
所以，我准备订一个更2B的价格，28。
限时。

等到下周一开盘了，我就会把他回复到88块。
