# 2017-11-24复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/c4GwtUL1tORUqwnJZkP86Q>

## 原文内容

今天公司搬家，不远，搬了16公里。

公司有叫了搬家队，我只需要把东西打包好，搬家队会搬过去，下周一直接去新厂拆包就好了。

但是呢，为了下周一可以迅速进入战斗状态。

我就自己开车吧东西都搬过去，先整理好。

所以今天一天基本都在整理东西，搬东西，开车中渡过了。

基本没怎么看盘。

就看了到了10点多。

因为一日之计在于晨每天开盘的半个小时是最重要的。

大盘拉了下，但是没有了量，然后剧情开始了紫色箭头，预想中的杀跌开始了，

但是呢杀跌很不流畅，尤其是箭头处，给人的感觉就像，

一个人在跳水，跳到一半，被人踹了一脚一样

这种不流畅的感觉让人难以言语。

所以我就出干，清仓了。

昨天我说了我有个看好的板块，但是我没说。。

其实这个板块就是次新和填权。

因为这个板块1：超跌

2：周末发行有减少的预期，（虽然今天预期没有成功，依然是5家，）

但是我没有说的原因很简单，端头铡刀当晚，任何推荐，都有可能坑人。

所以只建议了大家尾盘的时候，抄点高送转预期的标的来进行押宝。

尾盘买嘛，买入价等于收盘价，当天无浮亏。

临打包关机前，看见有个后台留言说我一看，这把我昨晚原本想说的给说了，我就把他留言放到墙上了。

其实我当时也有点想买的，但是想想，还是尾盘买比较稳妥，不知道下午会发生什么。

然后我关了电脑了。

中午吃饭的时候，掏出手机看了一眼，发现皮阿诺涨停了。

本来正在吃的火锅的兴致也顿时被打了个骨折。。

原计划，尾盘的时候抄几个预告送转标的。

爱乐达，天目湖（中信上次推的列表里的第一个和倒数第二个，倒数第一是凯普）

但是那会在开车，车上还有同事，也不能路边停下来操作。

所以索性想想算了，钱留着晚上去排江南嘉捷去了。

目前持仓如下

空仓了，也没东西好贴了。

复盘照例会在周日晚，敬请期待。

好了，祝各位周末愉快了
