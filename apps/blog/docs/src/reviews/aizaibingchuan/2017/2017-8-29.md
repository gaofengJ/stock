# 2017-8-29复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/I5GCnco-SkbC4ORFjsVdMA>

## 原文内容

今天发生了很多惊心动魄的事。

昨天尾盘科大跳水的时候，他一边跳，方大一边涨。

然后我和朋友聊天说，今天猜猜会有谁跳水。

没有想到的是，最最热的次新板块。

我相信，今天会有一堆股神们跳出来，各种说：我上午次新卖掉了，然后我提醒你们了有木有！跳了有木有！风险规避了木有！XXXX了木有！有木有！！！

既然大家都在表达存在感那我也要刷一下:昨天是这么说的。

为什么我昨天只推了两个次新股？

因为他们就是这一波次新的滞涨股。

今天尾盘跳水，这两个魏然不动！

杭州园林连推两天！

至强音！

注意：是没涨前推的噢大家都可以跟我一起从容上车。

好了，牛逼吹完了，开始复盘。

**大盘**

昨天有说吧，今天果然继续受到上升通道的压制，没有继续狂飙了。

当然没有继续狂飙的原因昨天也预见了。

昨天拉指数的最大功臣是证券，但是放了巨量，短期来说，再难有作为了，能横住就不错了。

so，今天大盘能横住，也不错了。

赚钱效果依然不错（除了尾盘的次新）

大盘后续会大概率沿着大盘的上半通道来运行

**热点**

时间就有限，就点评两个热点：

1：次新股。

今天虽然杀的很暴力。

但是很多股都是恐慌杀跌。

很大程度上仅仅是补了量而已，比如建科院看上去纵横交错但实际上呢，人家还是缩量的呢而且收盘是红的（量能在5日10日线之下）

当然也有些股，基本就是废了可以说，基本是废了。

道理很简单。

如果明天有资金要作转势，次新还要作，那就会有个新龙头出来。

假设你是大资金，你会做哪个？

换我，我肯定作建科院，没放量，拉板了还是新高板。

成天伟业，不说今天的套牢盘，就说这拉板不创新高，就不会有什么人愿意去作的。

这两个只是举例，不构成投资建议。

而且明天次新，会有个最大的雷，就是。

明天会有大量的开板新股。

新的机会或许会在他们中产生

混改：

新闻发酵不断，这个板块还会继续有机会的。

机会还会在，龙头股，人气股之中。

联通什么的，就不要去碰了。

小资金参与联通的博弈是没有什么意思的。

**个股**

说到这，我相信你们也看到了，周日晚的复盘，推荐的杭州园林，目前成为了市场中的最强音，最最强的一个。

最晚推荐英力特，再次成为今天混改的最强音，这个时候，我就要说说我今天的标题了。

有的时候真的需要一点勇气。

有的时候你的知行不合一，可能就是你的执行力还不够，你的感性影响了你的操作。

个股复盘直接上图

杭州园林没啥好说的，继续持股。

英力特开盘等了1分钟，观察了下走势，及盘面，然后果断就上了。

虽然比开盘价少了2%的利润，但是胜在了确定性。

当然打板的确定性更高，但是我敢肯定，今天这个板，你肯定打不上。

英力特买完就板了，我当时也打板加仓了，但是一看排单根本没有机会。

立刻按照机会，低吸了当时还是绿的洛阳玻璃。

同样是我昨晚计划的个股。

嘉鱼股份，昨晚写的持续看好，今天发生了一笔乌龙操作，导致赚钱的变亏2个点，不说啥了。

万里石，今天预判到厦门国贸要板，当时看到万里石刚异动，立刻买入。

结果吃翔1.6个点。

俗话说的好，计划外的股，不要去做，一做就出问题，还不如直接上厦门国贸。

今天金砖的逻辑很简单，因为昨天印度撤兵了，那么印度来参加金砖那就稳了，利好金砖概念，今天市场动一动，很正常。

果然今天盘后就出新闻，莫迪宣布来参加金砖。。。

今天买金砖的基本明天有肉吃了。

东湖高新，下午看到的好股，妥妥吃肉的。

当时候发在了淘股吧，然后也发在了昨晚的复盘下面的评论里，不知道有没有人看到。

为了买这个股，我把惠达卫浴都出了，就为了买它，明天开盘就要走。
