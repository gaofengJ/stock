# 2017-6-14复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/5DdwNh6woHtnqVALi5dngA>

## 原文内容

时常关注我的朋友都知道，

我时常会说的两句话是：

温故而知新，华尔街无新鲜事。

今日涨停票近50只，看似花团锦簇，但是仍有隐忧。

那就是，缩量反包前根阴险实体。缩了13%的量。正如王起来在大多头里面说的那句话，今日大盘上涨无量，只卖不买。

明天大盘会怎么走？

我不知道。我们可以一起看看过去的一年里，大盘是怎么走的？

2017-1-9 大盘缩量7%，K线组合与目前类似，触底后反弹收阴后的缩量反包，具有参考性

2016-12-1 缩量0.5%，K线组合与目前类似不同的是这个时间点在相对高位

2016-9-19，缩量10%，K线组合相差太远，不做参考

2016-8-9 缩量17.5%，K线组合形态相差较远，不做参考

2016-6-22，缩量21%，K线组合很相近，且时间节点也与目前的时间节点相近，具有较大参考性，

2016-6-27，缩量11%，K线组合相差较远，不具备参考性。

综上，看完后客观自己心里有数啊。

前晚别明天一跌就赖招商证券策略会，跟人家没关系，真的。

就跟你小时候每次春游都觉得在下雨一样，其实压根就没几次下雨。

末了说点正能量，今天几大股指，只有创业板是放量反包的。这是资金要转变风格了吗？
