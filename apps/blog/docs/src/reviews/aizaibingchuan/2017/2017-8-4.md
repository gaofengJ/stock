# 2017-8-4复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/4v8J0GISGIEsIe7SjA1E9g>

## 原文内容

本来想对周五的盘面好好写一写复盘的。
因为周三的那次发送不成功所以周三的那篇文章用掉了周四的发送名额周四的文章又用掉了周五的发送名额所以的周五的复盘就只能用周六的名额了。
但是呢，昨晚我去看战狼2了。
所以，昨晚也没时间写复盘了。
到了现在这个点，各位大神的复盘已经铺天盖地了，我突然也就提不起兴致来写。
今天主要想写一写关于低吸。

周四晚上，我通过短短的复盘，迅速的锁定了一个股那就是北京文化，逻辑非常简单
1：三生三世口碑超级烂，那么就没有人能撼动战狼了
2：排片率，决定票房的一个很大因素就是，拍片率。
周3的时候，我看见周末三生三世的拍片率高达70%！！！！
但是周四晚的时候，我看见他的拍片率已经降到了45%，战狼的拍片率从二十几上升到47%，这就意味着，票房会继续燃爆。

事实上，我的逻辑也得到了验证，周五看好战狼周末票房的资金，果然把北京文化干涨停了。
这又是一次逻辑的胜利。
逻辑可以给10分。
但是，操作只有3分。
为什么?
因为我说了低吸，但是我除了在开盘买了一点以外，后面就一直在苦苦等待绿盘，这其实陷入了一个误区。

何为低吸。
低吸是相对与收盘价的买入，当天具有浮盈，那么都可以称为低吸。
但是你盘中买入的那一刻，你并不知道今天的收盘价是多少。
所以低吸是一门学问。
你的资金量太小了，所以你决定不了底，所以你不知道怎么才算低。
所以反省了下。

除了绿盘买：这个就是传统意义上的，水下低吸。
另外低于均价线的买入，也可以称为低吸。
这个可能表难以理解，我来举个例子。
股价在图中的这个阶段，就可以称为低吸。
比如，我周二买入北京文化的时候，入下图我接近6.7%的时候买的。

如果一个你不相知的人，说他今天6.5%买了一个股票，你下意识的会认为他是低吸了还是追高了？
你再看看我这个图，你觉得我是低吸了，还是追高了？
这就是均价线的意义。

所以，突然又想明白了一个事，什么是追高。
当股价急速拉升，严重偏离均价线的时候，你买入了。
你就追高了。

那为什么又说打板比追高安全呢，因为涨停板的，那一刻，成交量急速方大，成交均价会呈现直线状的拉升，这个时候你与市场中最看多，最激进的人站在了一起，所以是安全的。
（当然撤单的就不说，就是相当于你的队伍里出现了叛徒，这比敌人厉害更要命。）
