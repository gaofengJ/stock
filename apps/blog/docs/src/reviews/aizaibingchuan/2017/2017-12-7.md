# 2017-12-7复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/6f8Y8_Eu_qZVJBpvzrv8FA>

## 原文内容

### 闲聊

今天晚上出去喝酒了。  
初中好友生日，几个哥们一起吃饭。  
十几年的兄弟了。  
那种兄弟情，真的是，即使你半年不见面，都觉得近在眼前。  

今天一个初中哥们生日。  
所以大伙找了个理由聚了聚。  
喝了2斤。  
我平时不喝酒的，难得高兴嘛。  
喝了超限度的酒，肯定是不醒人事了。  

所以我现在是强行睁着一只眼睛在写复盘。  
另外一只眼睛怎么瞪也瞪不开了。  
其实我也不是很清楚，我是怎么回家的。  
只是依稀记得，还没写复盘。  

### 大盘

赚钱效应诶。  
你是不是觉得奇怪，今天跌的很多啊，为啥今天的赚钱效应居然是橙色的。  
很简单，因为昨天买股的人，今天其实大部分都是赚钱的。  
有持续性的赚钱效应，市场才能继续下去。  
市场热度与昨天相近。  
我的观点依旧同昨天。  
市场将会继续在这个区间内运行。  

目前市场有几个非常不安定的因素：  
1：要不要打仗。  
2：641要不要走人。  
3：国家要不要护盘。  
4：我妈刚才跑过来喷我，说：你喝多了吧，围巾都扔抽水马桶里面了还不去睡觉。  
我说，我还能撑一下。  

自11月中旬的那个利空以来，我一直都是号召的大家空仓。  
即使你要火中取栗，也请控制好度。  
不要栗没取到，反而烫了手。  

大盘依旧空头趋势中。  
不过我个人认为，这个区间，还能撑个几天。  

还有很多人问我，这是什么指标，昨天打到上延后，果然回落了。  
我想说的是，不要迷信任何指标。  
所有的指标都是滞后的。  
而且这个指标大部分的时候是没么用的。  
只有市场出现极端情况的时候，才可能有那么点用。  
请凭你的看到的，感觉得的，去体会市场，而不是通过指标。  
指标往往都是逐末舍本。  

### 热点

目前眼神不好。  
我只是很勉强的把热点复制过来了。  
里面的字，我竟然一个都看不清。  
不点评了。。。  

### 个股

个股复盘超级想吐啊，现在。  
持仓如下。  
兄弟转债竟然不是今天交易。。。  
傻了。。  

关于香飘飘，我上了4成仓。  
明天准备继续低吸加仓，如果有机会就作T。  

很多人问我看好的逻辑。  
其实我一个月前就说过了。  
一个月前我坐了个预言，大概有三个内容：  
1：一个月后有极端恶劣的新股行情（已验证）。  
2：因为极端恶劣的新股行情导致香飘飘提前开板（昨天有应验，可惜被大单堵了）。  
3：绝望的行情中，这个股会走出不一样的走势（待验证）。  

抱歉，写到一半，刚才去吐了会。  
反正我今天就是按计划买了香飘飘。  
另外看见深大通板，小仓买了英维克。  

个股展望抱歉，刚才又睡着了。。  
这里我就不多说了。  
明天准备ijixu加仓香飘飘，尽量低吸，然后有机会就做T。  

今天晚上我这状态肯定是没办法复盘了，各位抱歉了。  

很多人问我为啥看好香飘飘。  
qisqishi 我也没那么多理由，就是对眼。  
昨天有个人留言说，他觉得香飘一点也不好，但是想到之前的阿石创，他又犹豫了。  
我想说的是，不要犹豫，没有必要。  
我谨代表我个人的观点，谨代表我个人看好。  
我个人的想法并不代表市场。  

我只是准备拿我上个月的利润去赌一下。  
一个一个月前就想好的逻辑。  
赚了，皆大欢喜。  
亏了，利润回撤。  
仅此而已。  

所以，不要再来问我为啥看好了，问我是啥逻辑了。。  
我仅代表我自己，看好。  
明天要是给我闷个跌停，那我也心！甘！情！愿！  

撑不住了，刚又去吐了。  
明天是周五，正常来说，是不建议参与次新的。  
因为本周过会了好多家，万一周末就多发了呢。  
所以，明天买次新，本身就是有风险的。  

打不下去了，又要去吐了。  
祝各位晚安。  
今天的复盘很烂，没办法，毕竟喝了两斤酒。。  
甚至已不清，各位见谅，别留言了，我去睡觉了。
