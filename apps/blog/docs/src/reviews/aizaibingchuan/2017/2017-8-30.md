# 2017-8-30复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/t-dU1M2BWRgGtWhZI6zNWQ>

## 原文内容

**闲聊**

1：我每天写完复盘还会继续看看新闻，然后我会把看到的一些票，写到评论里。盘中的时候也是同样。

比如说昨天中午看到的东湖高新,昨天我下午买了浮盈5%，今天开盘就走了，赚了3.5%，非常开心的隔日超短。

比如昨天晚上的评论

A：水晶光电

这是一个严严实实的实体板，今天开盘买，加明天的溢价10个点妥妥到手了。当然不要考虑连板了，应该是没可能的

B：上海国改首板。

这个你没有准备好的话，补一定能打上。

2：今晚有个重大新闻，立讯精密收购了我工作的公司。

我们公司还是相当牛逼的，全球有十几个分公司，整个事业部打包一起卖

最重要的是立讯进军汽车行业了，对于苹果即将要上苹果汽车来说，绝对是重大利好。

算是内幕消息吧。

**大盘**

大盘基本符合昨天预判。但是今天稍微有点问题

今天相对昨天来说，是放量的，但是放量不涨，这个里面会有风险产生啦~~~

明天有可能会进行回调。但是不用担心，之前说过了。下有支撑，无数人盯着呢，他敢调整下去，买盘必然蜂拥而知，不用太担心。

操作上，依旧是轻指数，重个股。

不过我之前说过，一根大阳顶三天，今天就是第三天了。

原则上，明天是不建议开新仓了。

但还是那句话，思考不能停。如果能精选个股的话，赚钱效应是依然存在的。

**热点**

简单点评下：

次新：依旧是涨停最多的。高位股的涨幅在那里，各种低位股会进行补涨，延续板块的生命，不会直接挂，后市高位股不建议再参与了，可以关注刚启动的低位股。

上海国改：延续性有限，明天不建议进了，参考昨天的国电混改，英力特这么好的板，今天都泛绿了。资金明天承接会有顾虑。

AR:昨天为啥看好水晶光电，因为有新闻发酵，仅此而已，今天板块小高潮，明天也同样不建议再参与了。

周期资源股：之前大V们各种看空周期资源股，但是呢很多股都默默的走出新高了，今天好多股都是一个新高板，就问跟不跟，非常纠结的话题。

我之前也是在复盘中连续看好煤炭，主推山西焦化和平煤股份，我自己买入山西焦化后，两天割了8个点，十分惨淡。没想过没过几天，连拉两个板。心里是十分伤心的。

后市资源股怎么看？很简单，之前讲过，热点会不停的在，金融，资源，次新，雄安，智能，等板块中轮动，最好的就是潜伏，跌了买，涨了买。追涨杀跌死的快。

但是说易行难，简单点，不用3000多个股票每个都看，看你看得懂的就好了。

**个股**

说说自己的操作。

今天开盘把昨天买的所有的股票全部清了，虽然都是低开，但是耐不住昨天浮盈多。依然赚了不少。

杭州园林一天持股未动，

英力特受到上海国改影响，因为资金都去了上海国改，英力特开盘毫无承接，只能在杀跌中卖出，盈利4%

之前我股票池有列过一个股，中装建设。因为我之前说过，低位的次新，都有补涨需要，这个股今天水下我低吸下，没想到下午就拉起来了

还有你可能会问为啥我昨天补推的水晶光电和上海首板我都没参与，因为我都上了另一个票，不要问逻辑是啥，我也不知道。

朋友之间推票就是这样的，我每次给朋友推票，朋友啥都不问，就是买。朋友给我推票我也一样，啥都不问，就是买，自己作了计划的都不买了。

后市看好的个股。

1：次新+高送：皮阿诺

明天是股权登记日，后天就要除权，今年中报除权第一股。炒不炒，我不知道，就知道往年都会炒。

看好个股，杭州园林，皮阿诺，

2：混改：这个持续看好3个股：铁龙物流，洛阳玻璃，泰山石油。

天天低开高走。尤其是泰山石油，昨晚新研究的，非常好，小盘这个换手极佳。随时都有加速的可能。

关于混改，所有靴子已经落地的，不建议再参与了，比如联通，英力特，铁笼，泰山，都是鞋子未落地，存在预期的。有预期才能参与。

3：次新其他：三孚股份，我一个月前推过他，当时市场没反应。昨天终于开始涨价了，今天一个秒板。

中签中报业绩大增是基于上半年平均价格4800来算的，现在提价到7000，你算算这个股价应该涨多少？明天开盘不高，我真想直接上，当然开的高，这个风险就比较大了，很有可能走成今天杭州园林这样。

4：立讯精密我开盘有写了。我也不知道会不会涨，这个东西我不敢打包票。

好了今天就先写这么多了。祝各位明天安好。
