# 2017-7-31复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/EOPD9ktu3lu7qzrKJrIP_g>

## 原文内容

首先说声对不起，各位。

北京文化一字板，潜伏三生三世十里桃花绝对没有问题。

但是，对不起各位。我挖错标的了。

我挖了两个。

1：天神娱乐，参股儒意影业，但是我今天白天详细查资料的时候，发现，他特么已经把儒意影业的股份卖了。但是同花顺的资料还没有更新，导致我被误导了，也没有再去求证，这个标的我挖错了。

2：台基股份，参股润金文化，被证监会屡次问询，最后，他特么把收购方案给撤回了。，，，，，，，我详细查完资料，我也是惊呆了。

挖了两个标的居然全错。

我真是毕了个叉的。

对不起各位，很抱歉，但是今天低吸的朋友，也请放心，明天赚钱离场不是问题。

因为我看见有人抄了我的文章，在四处推这两个股。

就是上面这个，我相信，经过别人的努力，明天赚钱开盘赚钱跑路，不成问题。

真正三生三世十里桃花的潜伏题材，只有2个，

一：光线传媒

占战狼26%，三生三世的发行方，给三生三世保底8亿，

和战狼的保底价一样，只要票房大于8亿，光线传媒的利润就会成倍增长。

但是光线传媒已经大涨12%，振幅炒过16%，

已经不是一个合格的潜伏标的了。

只能算是一个博弈标的了，但是可以，水下低吸，毕竟8.3号才上影。

没上影之前都算潜伏。

二：东方明珠

这是正儿八经的收益标的。

可以潜伏，预计一旦起飞，预计可以有5%+的肉。

确定性较高。目前挖这个股人还比较少。

大盘

周末我对大盘是看调整的。

但是早上大盘就放量了。

放量就是体现了缩量阳的一个抛压的承接，

放完量就开始上攻。

大盘目前处于上升通道的顶部。

明天大涨的难度比较大

明天大盘继续看调整。

但是，有句谚语，不得不注意

五绝六穷七翻身，但是后面的话你还记得吗？

八休息，金九银十。

放量大涨之后，不会立刻杀跌，这点，请放心。

但是逐步降低仓位，也是需要考虑的。

目前目前又处于通道上轨了。

昨日个股

上周4的复盘里，我讲了周五开盘要低吸资源股。周五低吸的今天都爽飞了。

昨天推了6个股。

万达信息，任子行，今天若开盘低吸，明天赚钱问题不大

康泰生物更是大涨，一度摸涨停

北京文化，没能给机会，可以。

有个朋友昨晚就排队了，今天居然上车了，也是厉害。

天神娱乐，台基股份，上文已经说过了，开盘买的，明天赚钱跑路问题不大。

明日建议关注创业板

我昨天说了，今天创业板要走一个水下十字星。

十字星今天是走了，差一点点就是水下了。

明天创业板很有可能起飞，接棒资源股。

原因我周末已经讲过了。

创业板的机会，及反弹高度。

创业板也有可能会复制这轮走势。

明天谁拿创业板的龙头，至关重要。

我列选了几个创业板标的：

1：万达信息，本轮的龙头，今天尾盘抢筹明显，今天低吸的，则建议你明天先持股看一看情况。

2：科大国创，今天唯一涨停的创业板股票，超跌+填权+股权股权激励，虽然净利润同比下降，但是这是因为股权激励的费用导致的，抛去这笔费用，实际是预增的。

3：光线传媒，原因上面说过了

4：任子行，只要创业板大涨，老龙头永远都能赚钱。

5：另外北京文化要是有机会，还是可以上的。要是开的高，可以小仓位，不封板，尾盘还可以加仓。
