# 2017-12-26复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/grKJVikQXOyRfxti1Ktn3g>

## 原文内容

闲聊

闲聊一：昨天晚上的神预判逻辑，大家以为以为如何？

昨晚有人留言觉得很扯淡，
但是，我并不是在跟你信口开河，
因为在过去10年的A股，
12.26这一天，
除了13年是跌的，
其他，要么恰逢周六周日不开盘，
其他一律全涨。

每年年底，都是行情不好的时节，
原因就那么多条，
诸如资金面紧张之类的，
什么大家要从股市提钱出来发年终奖之类的，
而且年底黑天鹅也多，
所以年底大多行情不好，
这是有多方面因素的。
所以往往绝望中，
容易爆发出大行情来。

闲聊二：刚刚讲到黑天鹅，
比如今天又整一个新花样的黑天鹅。
7成利润来源的子公司说是失去控制了。。。
想想人生也是醉了，
这股今天刚刚一个反包大阳，
肯定不少人追进去了，
收盘就是黑天鹅，
而且这还是一个知名庄股，
这个里面的信托资金合规不合规，
属不属于强退的，我不知道，
但是我看了下，
竟然和天际股份裸奔的信托抬头是一样的。。。。
买了这个股的，这会想想就好害怕。

所以我还是昨天那句话，
前十股东有大量信托或者资管计划的，
请小心谨慎一点，尤其是年底了。
当年每年六月份年中也是超级缺钱，
回顾今年6月份，有多少个闪崩股，你还记得不。。
再看看今天的两个一字跌停票，
吓得我赶紧看了一眼我的持仓票前十股东有没有云南国际信托。。

大盘

今天的各项数据，全部炸裂。
我昨天预期只是个小阳，
今天一个中阳，我也涨懵比了，
有的时候情绪逆转起来，就是这么快，
讲下大盘：

今天从图形上来说缩量反包，
成交相对昨天是缩量的，
当然你也可以理解为平量。

横盘阶段，缩量反包并不是什么好事，
尤其是今天这种，
市场呼啦一下，这么多涨停板，
明天这个涨停板怎么承接呢？
有增量资金来承接吗？
如果有，ok，行情继续走好，
如果没有，那么你也懂的。

我之前有讲过，
情绪反转行情，最适合的是当日下午开仓。

----------------------干货插入--------------------------

情绪逆转后，
场外的踏空资金，一定会发动新的热点，
所以你不要看上午的股涨的很欢，
然后就去追跟风股，
追跟风股，一旦没有板，
下午大概率要回落的，
第二天大概率吃面的。

所以宁可等资金在下午发动新热点的时候，
去跟随新热点。
比如今天下午芯片次新（板两个，
我当时看盘的是有点击芯片板块，
涨幅最大的是国科微，当时是5%，
如果追的话，隔日超短，赚8%问题不大）。

国企改革如果半路上龙头两面针，是肯定正确的，
如果你没上，
那么相同图形，相同概念的洛阳玻璃同样是备选。

这些在下午都是有迹可循的，
在情绪逆转日，
你不要觉得踏空了很难受，随便就追了一个，
注意，一定要冷静，仔细观察，
一旦出先了新热点的机会，请及时跟随。

----------------------干活结束---------------

热点

我昨晚的复盘只写了一个看好板块，
那就是次新板块。

次新股是今天最强的板块，
卫信康和奥农生物的上板，
以及5板的贵州燃气，
发散出来的就是低价次新，
这个时候你手速要快，
立马打开次新板块，
挑价格低的，股性好的上，
前9低价次新股，今天板了6个。。。
66%的上板率，可怕吗？

今天的行情太爆炸了，
没啥太多的好分析了，
直接看上面的表格来的清晰一点。
各位复盘的只需要区分下，
那些是上午就爆发的，真正领头的，
哪些是下午踏空自己发动的。
一般来说，
下午踏空资金发动的，都是隔天就要跑路的。
相对来说，
上午的那些板，代表的板块，更有前途一点。

个股

个股复盘今天大盘好的出乎我的意料，
但我今天只操作了一个股，就是广哈通讯，
因为我看好今天5g+次新肯定要板一个，
早上开盘价来看，剑桥科技是高开的，
按照选来说，肯定要买剑桥科技的。
但是广哈通讯的成交量更好，
最后我还是选择了广哈通讯。

这股棒的就是成交量很稳定的，
很多突然爆发一下的次新，
资金第二天都会选择跑路的，
因为第二天不跑，后面可能就没有机会跑了，
中新赛客我等第一波翻红就走了，，
没想到昨天炸天今天还能冲这么高，
主要也是得益于大盘的环境好。

对了昨天金牛化工的纪律止损单如下，
因为小仓，亏的钱也不多。
所以看盘也不是很紧，
待看到时已经-11%了，
所以赶紧清仓。

个股展望明天其他概念死不死，我不知道，
但是次新肯定不了，分化是一定的，
今天板了34个次新，
这几天的行情，是属于打板客的，
对我这种模式的人来说，比较难做。

主要是我卖飞了贵州燃气，
不卖飞我也不用复盘，
天天只要吹水就好了。。
所以我明天也不写看好了，
反正明天的机会依然在次新及次新叠加概念，
所以我呢，明天准备持仓不动了，看情况了。

毕竟繁华的背后还有很多隐忧，
比如今天创业板盘中又新低了，
而且离年底的大解禁越来越近了，
还有一点，
虽然每年过年前都会有一段过年前红包行情，
但是今年过年过的晚，比往年晚，
所以红包行情，我觉得还没那么快来。

我的建议依旧没有变，
谨慎点，
等解禁雷释放了，
我们再看情况下手，不是更安全，
来日方长的事。

最后，祝各位格格阿哥们吉祥。
