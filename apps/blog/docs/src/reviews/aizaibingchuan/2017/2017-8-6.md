# 2017-8-6复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/cC9nbro3RiUl2DCobyivPw>

## 原文内容

眼睛一眨，两天过去了，本来想像上周一样，周末写点东西。但是这周末家事较多，目标均为完成。连好好复盘都未能完成。

突然想到前段时间频繁出现的，大神们的征婚启事。均在感叹炒股耽误了婚姻大事。但是反向思之，又何尝不是摒弃了如多杂事，让他们能全身心的投入到这项事业中，才铸就了他们的成功呢。

这世界上最可怕的事，就是，比你聪明，比你基础好的人，比你更勤奋。你只会被甩开。

推荐看一篇，文章（如下），讲述了财富的随即分配过程。最经典的是，文末加入了一个参数，就是比别人努力1%，会有什么样的结果。如果你到35岁还是个屌丝，这辈子可能就这样了如果把他代入到股市里，大部分的地方，都是相同。

好了以上，都是股市课外功夫，下面切入正题

大盘。

周五的大盘惨不忍睹，与周四截然不同。大部分的股，完成了周四的补跌。大盘已经三连阴了。而且是连续的三根长上引线的K线组合，在历史上，我翻了近几年，我都没有找到。

上影线，意味着做多的失败，连续三天做多失败。这个事情值得警惕。我翻看了下，两根上影线是由于资源股的冲高回落带下来的。中间那根是给上证50带下来的。

周五的这个更补用我说了，方大的16个点巨大振幅，一把套了100亿资金，不敢想象，解放军从哪里来。但是也不用太悲观，目前大盘的趋势还在。刚刚上来的大盘不会直接就挂，下面是个反复磨顶的过程。

我个人认为，大盘会持续横盘，直到均线走平，2~根以上的均线相互纠缠，然后变盘指明方向。

周五热点

1：北京文化，毫无疑问的最强音。

2：军工，刚起飞就高潮。纯事件驱动。所以持续性有待商榷。这种突发热点你想参与，只有龙头股可以参与，其他股，追高都是9死1生。以后我会写一篇，关于为何，板块一高潮，第二天就嗝屁了。

3：钢铁。资源周期股中，补涨+涨价+缺却库存，导致了加速。周末在家看的时候发现，很多钢铁股，都在解放了股灾1.0时期的套牢盘了。

周五个股复盘，北京文化，最强音，啥也不说了，周五要是没上车，就不在建议上车了，风险大于机会了，而且网络也快开始播战狼了。

明日个股。

1：战狼引爆的电影题材。一个存在补涨需要的题材---院线股

战狼的火爆，收益的绝不止北京文化，当然我是不会给你推什么捷成什么的。其实上周资金已经给我们指出方向了

票房收入的50%是给电影院的，剩下的才是给制片和发行的。电影院的收入，除了分成，还有爆米花。。。。和其他。。。

所以票房的火爆，对电影院的利好远远炒过北京文化，这个发行商。。。。比如上周涨停的幸福蓝海。我可以有50%的票房收入，乘以我的市场份额，接近7%的收入。而北京文化也就12%啊，北京文化涨了多少，幸福蓝海涨了多少？这个里面补涨思路各位可懂？当然，上面还有众多院线大佬，各位都可以考虑下，我只是提供了一个思路。。

2：次新股

周末ipo如期减少。标的我就不说了，好标的太多，但是讲了又不一定涨，就怕坑了大家。周五我买了个东尼电子，坐了个过山车。不晓得明天有没有机会。

3：创业板

周三周四，资金连续攻击创业板失利，可以关注周一是否会趁资源股休息的时候，再次把握机会。
