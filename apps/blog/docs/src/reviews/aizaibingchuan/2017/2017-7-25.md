# 2017-7-25复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/xmtrtg3A9Nn_6h1Swxkubw>

## 原文内容

公号写了两个月，还没收到原创认证邀请。不能开评论。

请教了高手，说我很多东西写了，又发淘股吧，又发雪球的，影响了原创。

微信在自动检索的时候。网上有重复的，就不会认为你是原创的了。。

这个好尴尬。。。

所以我，准备以后写在公号的，就不发了淘股吧了，

写在淘股吧的，也不转微信了。

所以我给标题，也写了个很俗套的名字，非常的俗不可耐，我自己的都嫌弃。

但，我相信，这俗不可耐的名字，肯定是全网唯一的，原创。

大盘

今天的大盘，如昨天所说。

低开，调整，虽然上影线短暂的击破了昨天的高点。

但是然并卵，大部分的时间，不出意料的，都在调整中。

后面几天都不打算对大盘作评论了，因为我昨天说了。

大盘下面要横盘调整很长一段时间。

热点

今天延续了昨天的情况，

热点轮动太快。

热点轮动太快，就是造成，拉啥死啥。

杭州大湾区，早盘看似最强，继而卒。

银行，护盘，继而卒

次新早盘低开，调整，昨天强势股继续强势，跟风股纷纷扑街。

上午卒的涨价和低PE，下午满血复活。

大盘迷茫之时，妖股总是最能无厘头钢在那的男人，而且第二天都给溢价。

还有高送填权板块

一天轮动这么多热点。明天又要承压。所以大盘继续看调整。

今天调整的板块，明天很有可能就要爬起来。

昨天的个股

说实话，昨天推的几个股，真不错。

尤其是最后一条，内蒙票的低吸，今天要是低吸了华资实业，那你简直要上天了。

明天的个股

今天拉起的西水，华资实业。

让我想起了07年的那波牛市。

证券大涨，末期的时候，参股券商直接飞上天了。

保险从去年到今年整个走了一波牛市。

参与保险的西水已经翻倍。

参股保险。再复制当年。

如果你西水不敢追，可以考虑下 中原高速和福建高速 。

另外，内蒙大庆的具体时间一天不公布，

鄂尔多斯和华资实业，你水下低吸永远都不会错。

今天低吸华资实业的飞天了，

明天说不定就轮到 鄂尔多斯 了。

因为热点轮动快。

最后来个金句

大盘强，尾盘板不要碰，尤其跟风板

大盘若，尾盘板可以打，第二天有大肉。

明天我们可以验证下，今天的几个尾盘龙头板，明天的大肉有多少。
