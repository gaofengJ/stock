# 2017-7-11复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/h1TUFwBWkn_qzkMs-imThg>

## 原文内容

今天涨停板少的可怜。

但是跌停板很多。

这些股都有同样的特性。

真是应了那句话，涨多了就要跌。

当然还有些还没跌的。

符合K线学上说的，双针探顶，短期需要小心。

当然还有其他的个股也要小心，不一一列举了。

周日晚的时候，我曾经点评过，远望谷。

其实我当时是极度不看好的，所以我只能说，这是勇敢者的游戏。

这种凭借勇气值来赚钱的事，一般我是不会参与的。

目前市场没有涨停板，但是有一种股，叫做涨不停，而且他们盘子还特别大，比如科大讯飞，方大碳素。

一反之前市场的风格。

开启了一种新的玩法。

叫做每天涨5%，天天涨。

这样的股太多了，方大，科大，资源股，等比比皆是。

以及今天刚开始崛起的水泥股。

正是基于如此，今天买入了西藏天路。

高送转年年炒，不会不炒，只有早炒晚炒的区别。、

平智信息，我昨晚有写到，典型的错杀，

后面一定会有修复行情，只是早晚的问题。

明天若再杀，那就等于是公布了高送转，没有涨，

这，历史上，我也没有见过。可能是我太孤陋寡闻了。
