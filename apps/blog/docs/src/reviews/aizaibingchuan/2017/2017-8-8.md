# 2017-8-8复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/9__EcKZ0iJdczWYSYGYx4g>

## 原文内容

**大盘**

今天大盘挺弱的。
但是有一点好的是，就是放了点量
给昨天的缩量阳线补了点量
但是赚钱效应还不错，
除了莫名其妙闷掉的卫信康，
和从来不给肉的酒鬼酒。
其他基本都给了溢价。

大盘在这个位置需要异常小心。
因为macd红柱已经几乎没有了，
随时有可能变绿柱。
macd已经粘合3天了，
涨起来就是空中加油，
跌下去就是死叉。
so，明天很重要。

当然这是句废话，因为每一个明天都很重要。
当然这个位置，我的建议是，你可以降低仓位，去观察，
死叉了继续减，空中加油了，直接加仓。

目前没有大的风险，
明天大盘继续看横盘，关注5日线，10日线的支撑力度。

**热点**

**关于猪**
今天的最大题材就是猪，
这个没啥好说的，涨价题材涨到我们的生活中了，
鸡蛋涨完涨猪肉。
但是，连温氏股份都涨停了。。。
明天会极度分化，参与度较难。
只有龙头正邦科技值得关注。

**关于雄安**
昨天雄安投资公司成立的新闻，炒的沸沸扬扬。
今天相关个股开盘直接秒板。
而且，昨天晚上养家的言论，也是到处传的沸沸扬扬。
今天人家不光说，今天也行动了。

但是雄安我之前分析过，要连续大涨，还不具备基础。
今天跳空高开的结果，就是直接被第一个密集套牢峰，给打下来了。
雄安后续会有反复上攻的动作，适合低吸，
但是不具备持续大涨的条件。
短期内看调整缓缓向上，待均线走平时会有启动。

**关于创业板和区块连**
今天的龙头毫无疑问又是赢时胜，
那么这个时候争论又来了。
目前舆论普遍倾向于是在炒区块连，而不是创业板，
但我不这么认为。我认为是叠加的。

目前创业板，只要是中盘的，叠加潜在概念的，都在持续走牛
比如，赢时胜
比如，华策影视，影视版块二季度还是转好，国家队将都来了
比如，今天的猪肉温氏股份

so，创业板也不具备持续大涨的可能，
但是创业板中那些叠加概念的中盘股，要优先关注，选择，并进入。

**昨日个股**

常山药业，开盘进，有浮盈，建议明天获利了解。

建科院，有人留言，说开盘就进了。
我只能说真佩服，看好，要买，也要看位置，
竞价杀尾盘，还是高开的，建议就是等一等再买。
这是我的买点，虽然也挺烂的，但是好歹今天也有浮盈了。
明天可以拿一拿，因为明天同济要出来了，看看同济的走势决定建科院出不出。

**明日个股**

1：今晚铝期货又走逼空走势了。太生猛了。
但是相关个股的走势，今天走的是要多烂有多烂。
按理说这种巨量栏板，明天都是低开调整，但是铝指还在涨啊。
这个就很尴尬了。
所以，一旦铝股有水下的机会，而期货不跳水，那可以低吸。
当然你明天早上一起来一看，铝期货挂了，那就当我这一百来个字白打了吧。

2：雄安
雄安我前两天，一直在研究
看好了一个股，但是前天出了以后，一直没有接回来，
今天反包大阳了，感觉好可以
这个股就是杭萧钢构。
理由就两条
1：雄安未来大部分都是装配式建筑
2：未来会有走大牛的机会，等待回调的机会。

3：证券板块。
和朋友讲了好多天，特别看好浙商证券。
未来涨个50%不是梦。

4：明天没有特别看好的短线股。
华大基因随时要开板了，纯粹拼勇气值的时候到了。

另外我今天跌停板买了北京文化，
被一个1400手的减持利空打压下来的跌停板，
强势股的鱼尾博弈，保佑我明天能吃上肉。
上一个强势股跌停后的第二天涨停了，第二次跌停后走了个阶段反弹

也祝各位都能吃上大肉。

安
