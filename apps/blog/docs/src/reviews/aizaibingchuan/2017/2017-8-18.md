# 2017-8-18复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/fYQFqW5D-ZV1n3CC_6CoNA>

## 原文内容

首先，一句成真今天真的板了，又是大肉了。

---------

今天周五，一周结束。简单聊几句。

回顾一周本周推过的股，可以股下，大部分都是肉，也有一点小面但，推荐标的中，涌现了本周三块大肉。
1：嘉泽新能，推荐后两板。
2：同济科技，推荐周三低吸，尾盘即浮盈+2.6%，周四板。
3：博天环境，推荐周四低吸，今日板。

**关于低吸潜伏**

低吸潜伏的最大好处是什么？提前买，进可攻退可守。

举个例子，我周三买了同济，冲高了，打到的我预期了，7%出掉了。两天10%。这是个非常夸张的收益了。
如果你是昨天打板同济的，今天开盘没有割，那就是死成狗了。

低吸潜伏的最大好处就是，不成功，也不会亏，成功了，就是一块肉。
当然你潜伏之前，要对这个潜伏行动，订好计划，止盈止损价。

**关于运气与必然性还有模式之间的关系**

有人留言说我运气真好，潜啥啥板。但是我想说，炒股本来不就是三分靠逻辑，七分靠运气吗？牌桌上4个人，水平都一样，为什么还有人赢有人输？不就是运气有差么。但，运气也是他的必然性。

比如同济科技。有8.20这个投标节点在，就一定有潜伏资金进来。我的目标就是狙击这波潜伏资金。他们来，我就抛给他们。运气好的地方在于，这波潜伏资金，周4就来了，还形成了抢筹，我毫不犹豫的就抛给了他们。

比如博天环境推荐周四低吸，今天板之前我谈到雄安的时候有讲到过，雄安最大的优点就是，利好新闻不断。看新闻炒股，永远是A股不变的真理。所以，低吸潜伏，等新闻来。运气好的地方在于，刚潜伏完，新闻就来了。

比如其他。低吸潜伏，也不永远都是赚钱的，股市里没有永远大赚的模式，任何稳定的模式，都是，小赚，小亏，大赚，小亏，小赚，小亏，小赚，小亏，小赚，小亏，大赚，小亏，小赚，小亏，大赚，小亏，小赚，小亏，小赚与小亏持平，财富的增长来自于那些穿插其中的大赚。

比如昨天潜伏的装配式建筑今天的小亏。比如我今天尾盘又买了背景文化说不定周末又有啥大吹特吹的新闻，就算没有，就这个坚挺的平台，顶多亏个一两个点割肉。

**关于大盘**

昨天和朋友聊天中谈到，昨天的那个跳空缺口，是不健康的，短期内必然回补。没想到，今天开盘就回补了，我也是目瞪口呆了。毕竟外围大跌，+，今天股指交割日

**关于复盘**

等周末再来复盘了，周五了，就不要再想股票了。大家好好过周末把。周末惯例，周六写点技术类内容。周日写复盘和展望。祝大家周末愉快，好好消化下这周吃的肉。
