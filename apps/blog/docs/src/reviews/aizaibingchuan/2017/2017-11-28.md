# 2017-11-28复盘

作者：**爱在冰川**

## 原文内容

### 闲聊

今天要聊一个话题，
是钱与良心，
你选哪样。

我们先来看看起因。

上周三色幼儿园的事大家都知道了。
除了痛心，
你还可以惩戒他们。

上文是我之前发过的文章，
从心底里，
我们大部分人都希望，
他的股价能一跌到底，
让管理层，
直接血本无归。

然后，
现实很残酷。

幼儿圆宣布这是个别教师的行为，
是个例。

资本家们更冷酷，
他们的眼中，
只有冰冷的数字：
P/B,
P/B，
每股收益，
稀释每股收益，
归母净利润，
营业收入，
同比增长30%，
每股净资产，
每股现金流，
ROE,
ROA.

股价的下跌，
大众们的心里愉悦了，
资本家们更是开心了，
对他们来说，
捡钱的机会到了。

昨天晚上。
盘中大涨13%，
收盘接近一个涨停板。

上周五开盘最恐慌的时刻抄底，
今天开盘卖出，
隔天+20%的利润。

资本市场就是这么的冰冷。

那么，，，
当你碰到这么一个，
令人愤怒，
但是却有赚钱的机会。
你会选择？

### 大盘

久违的赚钱效应市场给了人一种否极泰来的感觉。

股民很多时候，
都是金鱼，
7秒钟记忆，
前面还亏的死去活来，
市场一有起色立马嗷嗷叫的冲进去了。

这里我们来看看大盘。
今天大盘最低点3300点。

我们再来看看我昨晚的复盘。
请问，
你昨晚看了那么多家公号的复盘中，
有哪家是跟你有理有据，
细细分析，
马前炮的跟你精准预言了今天大盘的最低点的？

昨晚我写到：
杀跌出来在场外的资金，
正虎式耽耽的看着这根线呢。

诚然，
今天大盘一杀到，
3300点，
一刹那之间，
整个市场就跟活过来了一样。

可以说，
在这个时间点，
买入99%的股票，
今天都是赚钱，
只是赚多赚少的区别而已。

比如：
我随手从我自选里面拉了几个股出来，
你可以看下，
下午的启动点全都是从13：14开始启动的。

我随手拉的4个股，
分别分属4个板块。

而3300点就是一个信号，
真正意义上的摔杯为号，
场外的资金，
分属不同阵营的资金，
想猎豹一样，
扑向了自己计划中的猎物。

所以才有了上面一幕，
同一时刻，
不同的板块，
同时启动了。

从而造成了今天下午热闹的赚钱效应。

如果你能看懂我昨晚文章中的精华，
那么你今天就跟捡钱一样的轻松。

再来说说今天的盘面。
大盘今天的反弹是缩量的，
而且macd继续向南。

正常来说，
这样的市场依旧处于空头趋势，
今天反抽之后怎么走？

昨天我已写到如下今天反抽，
符合昨晚预判，
目前走势任然维持预判3300点为强支撑，
静待5日线10日线跟随下来后，
再选择方向。
短期内还比较安全。

正是基于自己对大盘走势预判的信心，
我今天又重仓进去了。

### 热点

今天满眼的涨停板，
很多人表示踏空了。
心里如同又吃了一个跌停一样。

我想说，
这样的心态，
是标准的散户心态。

你只看到了那些涨停板，
你有可曾想到，
今天那些跟风栏板，
跟风冲高回落的股，
若你高位追入，
明天又要吃几个点的面？

我敢说，
今天这些板，
很多虽然板了，
但是明天照样可能没有什么溢价。

简单说下今天的热点。
今天热点，
你不要去看5g，
芯片啥的。

总体来说两种
1：N字板（又称涨停双响炮），
核心来讲，
就是没有新热点，
把前面几天的热点翻出来再炒一边。
例：中腹痛的图形非常标准，
如下。

2：还是超跌。
包括今天集体高潮的芯片，
原则上，
还是因为这个板块，
这两天杀的太惨了。。

短期超跌了，
就会有反抽，
上面两条是技术面的主线，
然后叠加题材方面的主线，
产生了共振，
比如叠加芯片，
叠加5g，
叠加涨价，
等。

今天涨停板虽然多，
但是没什么太让我惋惜的，
如果要说遗憾，
那还是有的，
那就是中光防雷。

因为我心里一直记挂着，
只要他跌下来，
我就要买，
我觉得等他收个十字星，
我就上，
还没买，
他就板了。。

### 个股

个股复盘：
今天唯一的失败点是没有作T。

早盘皮阿诺和凯普生物都有冲高。
皮阿诺被达安股份夺了日内龙头，
但是他又没封住，
就把填权板块带崩了。

凯普生物的是因为，
梅泰诺带起了5g，
那么资金自然就不关注他了。

但是我昨天买的少，
所以就静静的看着他们，
冲高回落，
然后我在水下低吸加仓了。

每天的操作都在实时更新在文章下面的置顶评论里。

这里要说到。
为什么我选择的是水下加仓皮阿诺，
而不是去追芯片5g呢？

因为
1：板块龙1龙2都封的很稳，
你也上不去了，
追跟风龙3456风险大于收益，
不划算，
当然，
我也没能预料到，
下午踩了3300后的反攻这么超预期，
居然能板块高潮。

2：今天市场的暗线是炒N字板，
那么皮阿诺是前天板的，
昨天今天调整两天，
明天若涨停，
那也是N字板，
如果我今天追了芯片，
明天卖了芯片追皮阿诺，
要是明天N字板没来，
我又卖不掉了，
有吃大面风险，
所以我还是选择了，
不追高，
尽量低吸。

3：皮阿诺下跌的时候，
跌破我昨天建仓的成本价，
那么有更便宜的筹码，
我就选择再加仓了。

个股展望我前天晚上的复盘，
就是我周复盘，
其实标的都列的比较清晰了。
今天其中，
也是板了两家。
你可以翻看下今天涨的板块，
基本都都在我周日的复盘里面了。

这里就要拓展出来第一个看好标的：

1：涨价股，
其实这是周而复始过程。
年中的时候，
先是炒资源股，
供给侧改革，
然后一顿涨，
涨完之后，
资金出来，
进了上证50，
开始干大消费的那几个牛股。

现在格力，
美的，
茅台，
都趴了，
那么从这里面出来的资金，
很可能继续回去干周期股，
因为驱动在那里，
我周复盘里讲到过的：
涨价。

所以这里，
未来继续看好，
涨价概念股，
这里我就不列股票池了，
因为我之前的复盘也陆续一直再写，
比如我之前讲水泥趋势龙头，
万年青，
今天也板了。

大家也可以关注新的涨价标的，
比如今天的三个包子里就有，
我给你截图过来，
省得你过去看了。

2：高送转。
我依然选择持股皮阿诺和凯普生物，
静待市场发生变化，
则止盈止损。
逻辑讲过很多遍了，
不再赘述了。

3：幼儿园
回到开篇讲的哪个幼儿园的问题。
如果今晚三色幼儿园继续无视利空大涨，
那么，
说实话，
我建议你也关注下A股的幼儿园，
电光科技，
前面有过两板，
股性有激活。
威创股份，
正宗A股幼儿园龙头，
超跌。

4:5g+芯片。
明天不建议进了，
今天板块高潮。
明天风险大于机会，
分化是大概率的，
只有部分个股会有机会，
可我能力有限，
把握不住。
所以我不参与了。
想想我之前挖的第二天一刀割了，
然后两个都是超级大牛股。
想想，
这难道又是八字不合？

5：厕所革命
今晚有出个新闻，
什么厕所革命，
我第一反应就想到我当年的老铁，
惠达卫浴。
他今天的诡异高开，
我就没看懂。。
明天会不会有机会？
新闻里我就看到一张图。
这厕所我一看就觉得好尴尬。
两个人同时蹲坑，
这得碰撞出什么养的火花。
不过，
这个新闻我也没仔细研究，
也不知道有没有机会。。

好了以上。
记住大盘依然处于空头趋势之中，
这种行情下，
空仓是没有任何问题的，
不要产生踏空这种想法。

空头趋势中的博反抽，
都是火中取栗，
不要只看成功的那一小撮人，
大部分的人都尸横遍野了。

如果你忍不住，
请控制自己的仓位。

今天有个朋友说的很好，
说他右手的鼠标已经换成了茶杯了。

祝各位明天安好。
