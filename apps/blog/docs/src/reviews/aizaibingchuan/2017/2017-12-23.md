# 2017-12-23复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/Wg48Tcztk7Q0KRlT2dgenA>

## 原文内容

最近一直在追剧，
看司马懿期中提到一点很重要，
就是揣摩圣意，
一言一行之间自由一饮一啄。
这几天一直在看本周中央经济会议的各方面的解读。

下文为转载：

从“中国制造”到“中国创造”——中国经济进入新时代  
从“有没有”到“好不好”——以创新寻求高质量发展

工人在北京建工集团城市副中心A2工程工地施工。

创新驱动发展战略大力实施，
创新型国家建设成果丰硕，
天宫、蛟龙、天眼、悟空、墨子、大飞机等重大科技成果相继问世。

我国刚刚研制成功的超级显微镜，
仅有手掌大小，造型如粉笔一样，
用它来观察头发丝，
扫出来的面相当于一个足球场那么大。

不仅如此，
它还是世界上首个用在超强磁场中的显微镜。

目前我国已经在核物理、未来网络、空间和天文等多个领域建成了50个重大科学装置，
瞄准世界科技前沿，
孵化高质量新兴产业。

就在刚刚闭幕的第四届世界互联网大会上，
发布了18项世界最顶尖互联网科技成果，
其中中国团队研发的占六成以上。

创新的引擎，正在被改革点燃。
169项全面创新改革举措在全国八个试验区域全部启动。
中国正以创新寻求高质量发展，
这将为中国经济开启崭新的未来。

2016年，
我国企业科研经费投入占全社会投入的近五分之四。
其中，
大中型工业企业中有研发机构的企业达到2万多家。

德国《明镜周刊》报道称，
中国领导人用长远眼光和大局意识，
部署中国经济未来走向，
并明确表示将进一步开放市场，
增加进口和平衡贸易。

中国正在重新定位自己在全球的经济地位，
部署新的经济全球战略，
这是中国领导人智慧的体现。

现在许多中国制造在全世界范围流通，
大大推动了全球贸易的发展，
这个特征在未来会更加突出。

目前，
在人工智能、智联网、金融科技等方面，
中国已经走在了世界前列。

北京中关村成为继美国硅谷之后，
全球第二大吸引全球风险投资的区域，
人工智能产业与硅谷齐头并进。

从“高速增长”到“高质量发展”——中国迈向质量强国。

中车长春轨道客车股份有限公司研制的首批美国波士顿橙线地铁车在长春下线，
成为首次登陆美国市场的国产轨道交通装备。

看完了有啥感想。

我知道，
肯定会有充满负能量十吨的人，
立马回抨击说，
这种文章都是写好不写坏，
然后说实际生活有多惨云云。

一般来说，
看啥问题都喜欢从坏方面来看的人，
也就是常说的，充满负能量的人，
基本是干啥也不会干好的。

推荐一本书好了，
这里跑偏题了，
讲讲这个会议，
目前了，
无数人正在解读这份讲话，
因为这基本都是定调明年行情的。

除了大家都在看的8个方面意外，
这篇文章解读的高质量发展解读的非常好。

以前是高速发展，
现在转化成高质量发展，
这是重大信号啊。

换算到股市中来看，
641上任以来，
坚定的执行了告诉发展对应的，
高送扩展直接融资渠道，
也就是说IPO，高速的IPO发行。

但是最近风格突变了。
641再也不提这个了，
所有的风向都进行了转变。

其他不说，
我们先看证监会的官网原文，
不转载了，
我直接画几个重点。

想想最近的一系列政策：
上市公司与保荐机构挂钩，
排队企业要求年期前全部自审，
不合格的都撤回，
最严发审委等等新闻。

所以我认为：
明年，
新股发行发概率从高速发行转变为高质量发行，
这就是解读圣意解读出的机会：

1：明年的行情，
大概率将会是高质量公司股票的行情。

2：次新发行速度降低，
高质量的次新公司，
凭借其流通盘小的优势，
将会是赚钱机会最多的地方。

想想上周4,5的行情，
为啥阿石创能够逆势上板，
带动大盘逆转。

阿石创就是之前炒优质次新行情中的龙头股啊，
4板一波50%，
引领之前优质次新高潮的龙头。

先知先觉的资金的总是在吃肉，
谁优先领会了圣意，
谁总能吃上第一口肉。

周三会开完，
周四就拉板，
周五就吃肉。

这就是，
我所认为的：
炒股的几件最重的事情之解读上意。
