# 2025-1-3复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/fr8IbQ9G0xfjPeSF9I1GYw>

## 原文内容

本复盘仅为鄙人的股市思考历程，非荐股，请勿跟票。

一天崩一路资金前两天是机构主卖，量化还在正常工作，
周二冰点，元旦回来，周四市场是普反的，
这会市场还是遵循正常的博弈规律在运行的，
然后看这个节点之后的白线，
这完全不是一个符合市场正常规律的走势，
属于典型的有大资金一路摁着卖造成的，
那按理说连续两天冰点+昨晚呵护之后，今天要普反了吧，
但是谁能想到，今天开盘量化直接崩代表指数微盘股开盘直接直线。

量化崩的还有一个显著特点就是，跌停板会稳步加多，
这个参考上个月量化第一天崩的时候我写过的内容，
今天是这样的，前两天传统机构砸，今天量化砸，
一天崩一路资金，还有全程一脸懵逼的你和我跟着左手抄底又右手割肉，天天崩。

---------------

你也别反思为什么亏钱了，给你们看张图
这是顽主杯实盘比赛的，全体选手的走势图从12-10至今，
也就是那个冲3420放量滞涨高开低走墓碑线的那天开始的全体曲线，
人均只有3天是赚钱的，其他时间都在亏，
人均亏20个点，而且这个亏损跟水平也没啥关系，
钱多的，钱少的，人都在亏，
千万组百万组十万组和钱最少的菜鸟组看上去千万组亏的少一点，
那是因为人家钱多，很多人都不满仓的，常年空仓，
菜鸟组和十万组亏得多，是因为钱少天天梭哈，
这个亏损比率不重要，看曲线就行了，
不同组别的账户曲线都是高度趋同的，和指数也是高度趋同的，
我做了这么多年的投票，已经无数次验证过了，
收益率永远符合正态分布，且与指数正向关的，
学过中学数学都知道，在正态分布(  ) [^2] 中，
68.27%的数据位于平均值的一个标准差内，
95.45%位于两个标准差内，
99.73%位于三个标准差内，
当指数处于下跌趋势，那么对于中间这个68.27%的人来，大概率就是亏钱的，
所以选择合适的出手时机，就很重要，

就像我11月下旬大跌的那次写过的一个长时间的预案，
1：下一周会探底回升，但是这个不重要，这个说对了，
2：12月初指数会一路反弹上攻，最差3300最后3420，这个也差不多说对了，
这个也是当时最好的赚钱时期，一堆股票莫名其妙的天天顶一字涨停，这个时期出手大概率就赚钱，
3：上攻3420后指数放量滞涨然后撤退官网或小仓，这个也没问题，后面就是一路跌了，
你看上面的比赛曲线也是印证这一点，我本来想掏我自己一路亏的曲线出来的，
但是看到了这个比赛曲线更有代表性，就贴这个了。
这个时期就是出手大概率就亏钱。

4：然后静待1月的博弈，一月的博弈就是两个，
一个博弈月底收官阳线+开门红，这个博弈失败了血割，
还有一个博弈节点就是等春节了，这个昨天也写过了，今天参与了，没想到量化崩盘了，
后面还是一样，等一个修复到5日线10日线的，
如果等不到修复触发纪律止损了，那也是该跑就跑了，
然后休息等到后面节点再来了。
