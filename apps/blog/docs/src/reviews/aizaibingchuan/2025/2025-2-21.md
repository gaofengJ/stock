# 2025-2-21复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/Owvwjegwm_6fP3bxh53HVg>

## 原文内容

主线是什么，

能持续赋予赚钱效应的板块，

就叫做主线，

一段时间内的市场里面，

一定会有一个或者多个最有赚钱效应的板块，

那这个就是主线，

主线会产生赚钱效应，

赚钱效应会持续加强主线，

起头的时候，

可能是因为产业趋势，

可能是因为政策发布，

可能是因为舆论发酵，

等等，

这个节点要预判下这个东西能不能走成未来主线，

这个是大胆假设阶段，

过春节前就说了，

要埋伏就埋伏三个方向，

泛ai，

尤其是当时发酵的deepseek，

泛机器人，

泛消费，

过完春节也说，

还是这个三个方向，

ai的deppseek，

人性机器人，

消费里面的哪吒大胆假设就先蒙方向，

然后根据盘面强弱，

根据消息驱动，

来综合跟随，

上周我觉得deepseek是最强，

反正我每天的打板，

隔夜顶一字，

低吸，

都是围绕这个板块来的，

周一晚上看了新闻联播，

看了民营经济会议，

感受到了上层对机器人的重视，

我原本觉得机器人是节前赚钱效应的延续deepseek好的买不到时候的一个轮动，

看完新闻联播，

机器人在我内心提升到了和deepseek一样重视的阶段，

当晚就开始加大对机器人各种细分的研究，

列了一堆股票池第二天也是看见那个机器人板了打哪个，

第二天打完就碰到大退潮，

当日全是大亏，

然后隔天修复的时候，

侥幸都反包了。

------------------

这周的市场轮动主要就围绕deepseek和机器人之间进行的，

轮动节奏主要还是一个消息驱动，

deepseek周末的腾讯发酵，

昨晚的阿里发酵，

都是消息催化机器人也是各种的，

去年宇树机器狗飞檐走壁发酵了机器狗，

这周马斯克的机器人接网球发酵灵巧手，

昨晚的fig机器人的视频也是在展示机器人大模型+灵巧手，

周中宇树机器人跳舞然后还拿拖把捅捅不倒还边跳舞这个平衡能力讲的就是机器人小脑，

从事后来看，

盘面都是很清晰的，

在搁在事前，

就没这么简单了，

想想上周四晚上的文章里面还写过，

阿里要增加资本开支，

谁知道在一个星期之后的昨天，

这周四晚上就真的表示要增加资本开支了，

做到最后，

就是那天文章末尾写的所以炒股就是一件不停的在我命由我不由天和看命了反复横跳的故事，

可能轮到我的时候就是第一天：我命由我不由天，干，买了

第二天：卧槽，命不好，割了割了

祝周末愉快。
