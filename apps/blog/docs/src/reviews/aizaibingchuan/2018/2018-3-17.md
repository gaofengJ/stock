# 2018-3-17复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/nBJwRKPJqjU06f67jzYunQ>

## 原文内容

买股票就是买公司这句话我们听了不下百遍。

深层次的意思就是说好公司的股票才是好股票，我们要买就买这些好公司的股票。

这话完全没有毛病，符合价值投资的理论，真这样做的话一般也不会错，还很有可能有不错的回报。

但我是个做短线的，这个公司好不好，价值怎么样，其实我根本不关心。

直到遇到各种坑爹问题后，我发现，好公司真的很重要。

但重要的不是基本面，而是这个公司的人怎么样。

我来给大家举几个例子吧。

1：川金诺 300505 大家记得吧，
去年年报预高送转的龙头第一波启动后，横盘调整3天，从形态上来看，一点问题没有。

他能成为当时17年年报预高送转当时的龙头，就是因为在16年炒这个概念时反复活跃，
16年时，他启动，横盘，启动，横盘再启动。

走了三波，翻倍多的走势。

而去年（17年）的时候，当时他走第一波主升，已经完全奠定了自己的身份。

进入横盘期各项指标也都很好。

但，就在这个时候，坑爹的事情出现了。

竟然在周五的中午，出了个公告，业绩预减10%。

其实这个业绩报本身并没有问题。
化工行业嘛，市场有点波动，这个季度少赚点，下个季度有可能就赚回来了。

但是呢，这个发布的时间节点很有问题。
中午发布，而且是周五的中午发布。

所有中午发的公告，市场都是没有消化时间的，
很多人都是立刻用脚投票的，啪的，就给理解成了利空，直线摁到了跌停。

直接破坏了结构形态了，从此走弱。

这公司的质地本身并没有问题，年年赚钱，
受到产品涨价，业绩还在不停地稳步提升。

为啥股价走这么差？
因为这个公司并不是一个会炒股的好公司。

估计董秘连炒股知识都没有，更不要说理解市场了。

对于资金来说，买股票就是买公司。
你这个公司不会来事，那么自然就要抛弃你了。

2：回天新材这个公司，那可就会来事了。

从这个公司的董秘回复就可以看出，一直都很配合市场的炒作。

市场炒雄安，立马说我们有关系关系，我们去调研啦，我们设办事处啦，
还有模有样地说出了3个月内和雄安谈完合作，高管还配合增持了5手！

你没看错，5手。

市场一看，你这么会来事，二话不说，直线爆啦。

但，也是因为太会来事了，市场做多情绪浓烈，缩量秒板，接顶一字。

但刚好碰上当时市场环境差，直接后面把筹码结构打烂了。

要是放在雄安初期阶段，少说也是翻倍的走势。

后期，无论市场有什么热点，什么粤港澳还是什么的，
只要有人提问，董秘立刻往自己公司上联系，说，咱们有关系，概念正宗。

这个就是，会来事。

3：今飞凯达这个怎么说呢，
大中午的，涨停板上面出异动公告，来提示风险，导致下午盘面直接走弱。

你就不能老老实实地和大家一样，晚上出公告吗？
写公告的时候就不能注意下文笔吗？

4：太龙照明

周末太龙照明出了个回复函，这明显就有水平多了。
有这么一条，我给你们翻一下：

深交所责问：你家股票涨太多了，知道为啥涨不？

太龙照明硬钢答：
我司认为影响股价因素有XX,YY,AA,BB,,DD，
但我们的涨得多原因是因为，主要有以下几点：
1：年报增长
2：一季报增长
3：我们有分红以及顺带的高送转。

------------

实际市场上都知道，太龙照明就是个高送转标的，
但是他硬生生给自己定义成了价值投资股，符合主旋律，涨了没毛病。

还有，你们市场不是在炒科技股吗？
监管不是也在提倡高科技吗？

那好，我还有更绝的，顺便一起发了个公告。

你仔细一看，这个认定是去年就认定了的，但是一起放出来的言外之意理解下，
就是说，我司是一个价值投资有分红的高科技公司。

全部符合你监管的意思，所以我这波涨了30%，你监管还有啥意见？

这就侧面体现了一个管理层水平中的语文水平。

5：易见股份

一月份区块链的龙头，起于风萍之末。
瞅瞅董秘多会来事。

除了回答投资者的问题，一一敲定落实后，还会补充。
回答完以后，立刻敲定了龙头身份，一波5板。

即使后期在强监管之下，董秘也是深深的无奈，
每一条回复，都是水平爆棚。

回复区块链业务的时候，说：我司没有市场认为的区块链业务。

意外之意就是，我这有区块链，但是我这没有区块币业务。

当时还有一条是啥回复来着，我忘了，反正意外之意就是：
现在监管太严了，求求大家不要买我股票了，买了也是亏钱。

真良心啊。

所以大部分区块链股票依旧亏损较大。
而易见股份在上周几乎已经解套之前所有高位的套牢盘了。

------------------------------------------------------------

类似的例子，在A股中数不胜数，大家可以自己翻翻看看。

除了公司本身的质地以外，公司的管理层水平也是非常重要的。

为什么今天会写这一篇，源自于上周和一个朋友的讨论。

当时主要的讨论内容是，为什么金飞凯达先板的，最后被威唐工业夺了龙头。

朋友的回答是，上次被金飞凯达的白痴董秘坑过，
所以当时金飞凯达异动，二话不说买了威唐工业。

并且跟我预言，说，金飞凯达一定不会二板，因为董秘太坑，市场一定不敢拉二板。

导致我二板那天惨痛卖飞。。。

所以说，炒股，独立思考很重要。

当然独立思考，并不是讲一味地YY，YY自己的股能怎么飞天。

而是需要站在相对高点去思考，相对低点去操作。

这就是当年那句名句：
战略上藐视对手盘，战术上重视对手盘。

好了，这个讲得有点扯远了。

大家也可以留言讲一讲，
自己碰到过的因为董秘的神回复、公司的神公告，
而莫名吃到的涨停，或跌停。

祝周末愉快。
