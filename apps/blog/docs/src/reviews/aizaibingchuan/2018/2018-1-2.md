# 2018-1-2复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/lRKY2UoiuO9XLi2f-LY1iA>

## 原文内容

**闲 聊**

闲聊一：昨天我复盘说我要空翻多了，今天就给面子了大盘开门红，还是大阳古语有云，一根大阳顶三天，那么你认为能顶几天？

闲聊二：大阳是一回事，操作又是一回事，最近操作进入瓶颈期了，自从卖飞贵州燃气后，买啥啥亏，个人需要进行调整了。

**赚 钱 效 应**

从涨停曲线来看，结束了之前心电图的走势，市场重新进入了活跃期，赚钱效应：今天赚钱效应不佳的主要原因是，今天一个连板股都没有，当然，ST股有两个一字板。没有连板，对游资来说，封板的意愿就不会太强烈了，因为要考虑第二天的溢价情况。唯一早早上板共达电声，还是超级大烂板。对明天承接又是一个困难点了。

**大 盘**

大盘今天放量跳空大阳后，指数将会进入一个新的区间，大盘今天光头阳，所以这个区间大致尚无法确定，需要边走边看，大概率大盘走的下一个长上影线，将会是新的区间参考之前的这个下影线，一旦上影线形成，大概率将会横盘3~5日，再决定下一个方向。这次3254~3300这个的很盘区间，将会成为一个坚实的短期底，即使大盘后续下跌，也会在这里形成一个坚强的支撑。

**热 点**

昨晚我写的看好一，看好二，基本成为了今天的最强的两个板块。年报预披露今天都一字板，所以下面也不建议参与了，要参与，也是凭勇气参与农商环境了，因为这是龙头，龙头都搞不赢，你去小弟，那只会死的更惨。次新今天开始已经走弱了，金奥博的反包和贵州燃气的巨阳，并没有能明显带动板块。主要原因是今天的机构资金太疯狂了，强撸涨价股，把市场上的资金全部抽走了。

涨价这个板块我就不多点评了，涨价概念一直以来都是我复盘里反复讲的品种。我还是那句话，涨价是业绩增长的前奏，是年报前必炒的品种。只要产品涨价，逻辑就在那，但是股价什么时候涨，说实话，只能看市场，没人能预测的了。

比如水泥涨价，我讲过好多次了，但是呢，股价后来一路阴跌，但是逻辑任然在那，这个时候，你除了靠耐心和信仰，基本没其他办法了，没人能预知风什么时候会吹过来的。旗杆就插在那，谁知道风啥时候会吹过来把旗吹动？之前有个人每天都会后台留言问我，水泥股为啥不涨，我每天只能去查下水泥的价格，然后回复他说，今天又涨了，今天盘整了，今天啥啥啥的。讲真的我做不了更多的事了。。

想想我11月15号复盘就开始推过中线的水泥板块了，那时候除了万年青，略走了一点趋势，其他水泥股都是一滩死水噢~~~恩，那一天还推了中线的房地产板块，首推了阳光城对了阳光城今天到了我的目标价了，后面还会不会涨我也不知道了，我反正我已经叫我身边的朋友今天把阳光城都减仓了。

**个 股**

个股复盘今天原本期待晶方即可能带一带大基金持股的标的，没想一根大阴，吓的我把国科微也出了，又割了一把肉。割完尾盘在士兰微的带领下又起来了，吐血。。

今天看见大盘大阳，盘中分批买了财通证券，理由是大盘大涨，证券都会表现，证券+次新，向来都是排头兵，比如8月中旬的大盘大阳线，证券龙头一波涨了50%，那时后的证券+次新也是风骚的两连板。今天买了财通证券，也是因为他是最新的证券次新，同时超跌40%，K线形态已经理顺了，连续8天站上5日线了，多头排列。明天继续持股等待逻辑是否有机会兑现了。

今天盘中看见拉升的时候的追的，个股展望1：证券如果大盘明天继续中阳，证券是一个很钢的逻辑，但是我已经持仓了，就不多说了。2：雄安今天新闻联播放雄安了。按理明天会有所表现，但是今天板块已经表现很惹眼了，股票池，参考今天的三个包子吧。另外还有雄安最近股性较活的个股，都可以关注，红墙股份，启迪设计，中持股份，中设股份等。

3：涨价概念今天的涨价概念，明天不一定会继续涨，这里我给你讲几个还没怎么涨的概念。对应国内的指数对应国内的指数但是黄金白银都还没有新高，但是股价更是没啥动作，从目前看来比特币的持续走弱，会使资金从比特币重新流向黄金。这话不是我说的，是美国的一个著名XXX说的，很抱歉，我不记住人家名字。大A的黄金股，目前基本都在地板。可以潜伏，但是涨不涨，不是我能决定了。

再讲一个：这个指数是十年来新高了对应国内的指数还不明显，股票池：罗平锌电，盛达矿业，锌业股份，西藏珠峰，华玉矿业，4：次新股明天还会有个别股表现，但是参与起来非常难了。请谨慎了。但是热点题材的还是请优先选择，次新叠加。

好了就这么多了又好晚了祝各位安好。
