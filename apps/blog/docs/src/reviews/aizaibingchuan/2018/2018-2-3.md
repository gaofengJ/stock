# 2018-2-3复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/m-2QpwxqIxRF8wPoASVg_Q>

## 原文内容

先扯个淡，昨晚去参加公司年会了，
所以后台有几百条留言，都没时间看了，抱歉了个各位~~~~
昨天五等奖四等奖三等奖二等奖我基本都没看，
因为我觉得，不是一二等奖，我都不削，
当然，到了这两个抽奖，也跟我没发生任何关系。。。。

简单总结一下，昨天抢了几十块钱的红包，发了几百的红包，然后毛都没中。

法系列之三：一个用不了的战法
因为这是我最近复盘看美凯龙的时候，无意中发现的。
美凯龙是个AH股，就是在香港上市了又在大A上市的股票。
简单来说下吧案例吧。

然后我就发现，这几乎是最近所有在A股上上市的港股公司的相同写照啊。
比如。拉夏贝尔。
这个期间涨幅差不多30%，比较合理。

下面给你看个夸张的。
庄园牧场期间最大涨幅十倍。。尼玛呀。。。什么鬼。。。

下面这个也比较正常秦港股份涨幅差不多也是30%左右。
诸如此类的很多噢~~~大家可以自己去翻看。

这就是一个战法，我相信香港股民应该已经很熟知这个战法了，
在全球市场里面，无论成熟的，年轻的，都缺乏不了投机客，
炒概念永远都死不掉的。
哪怕是美国，之前出区块链新闻的时候，哪个股出，哪个股就是百分之几百的涨，
这不是炒概念，炒投机，是什么？

中国有涨停板制度，不可能出现当日百分之几百的涨幅，
那么打板这种基于制度出现的战法，依旧不会失效的。
人心不死，投机不止。

扯远了。继续讲个战法，
这个就是讲的是，在港股上市的公司，同步在大A上市，那么在港股中的套利机会。
炒港股的同学应该知道这个战法吧。。
我也是这个星期复盘的时候，看美凯龙无意中发现的。

可惜，，我不炒港股股。。。
【此处应该有一个港股开户的广告】
可惜，我真的随性而写。。。并没有准备到这个。。。
所以我说，这是一个用不了的战法。。。

不过还有个办法。。你也可以开个沪港通，
思考的路，任何时候都不能停。
反推到A股，就是那些每次要去港股，要去纳斯达克敲钟的那些公司，
然后挖掘在A股中的影子股，供应商之类的概念了。
参考之前炒阅文集团的时候。

又到了快12点了。。。这个时间真是尴尬。。
只能说，祝晚安，祝周末愉快了。。。
