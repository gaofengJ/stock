# 2018-1-4复盘

作者：**爱在冰川**

## 原文内容

**闲 聊**

闲聊一：今天一天在医院，基本没有看盘，应该进医院之前把软件给卸了，昨儿说的好好的锁仓的，今儿临阵又变卦了。为此今天又唉几句喷，无非是早上第一波有人追高了，然后看见我说我卖了韦尔股份，然后喷我说：“你不说锁仓的吗？就是骗我们来接盘的吗？”呵呵，呵呵。最近频卖飞，需要好好反思了。其实这有时候就是命，你无意中的一眼，刚好就看见那惨绝人寰的一杀，你走不走？反正在卖飞的道路上，我一定不会是最惨的，今天还看见一张别人的图，突然就找到了点平衡感。。

闲聊二：目瞪口呆的场景天天上演，昨天的雄安今天的两桶油，两桶油一动，那基本就是百花残了。下午各种小盘股杀的是家都不认识了。资金目前非常的亢奋，带到标的就是撸，越是资金容量大越亢奋，昨天还有人留言问我，为啥时候涨了，中石油不涨。这个呢，历来都是不跟涨的，中国的油价，一箱油，一半是税。涨与跌，都是fgw说了算，所以很少有跟的，今天这种情况实属少见。不过石化双雄一动，其他股全爬，这个倒是一点没有变

**赚钱效应**

赚钱效应资金很亢奋，热度很高，但是热点轮动快，后面的拉起来，前面的挂，赚钱效应并没有太好的好转。

**大 盘**

大盘自我空翻多以来，表现已经超出了我的想象力了。之前整整3个月，堆积出来的套牢筹码峰，刷刷刷3天，就站上去了。一切技术指标都很好，但是消化筹码峰需要时间，不可能天天突突突的，下面有横盘整理的技术需要。但是从资金亢奋度来说，只要不追高，亏钱的概率并不大。

**热 点**

今天没看盘，就不点评热点了。石油板块，向来延续性差，这个板块就看中曼石油怎么来给他们续命了。自贸港尾盘消息刺激投机，尾盘大涨，且板集体高潮的，明天不建议去接力，大概率接力=接盘雄安板块明天大概率会有3板出现，不知道是谁，这还得看盘面了。

芯片我昨天复盘给你们分享的芯片股票池厉害吗？第一股票池就两个股，士兰微和韦尔股份，今天芯片板块里面，除了深南电路以外的第一第二哟。

**个 股**

个股复盘，今天一天医院，可能阴气太重，被附体了，韦尔股份手一抖，出掉了。持仓如下。

个股展望:

1:雄安明天大概率会出3板，是谁我就不知道了，但是我认为会有我认为大概率会从这两个里面出中设股份，龙头，参与全靠勇气，成了吃肉，错了割肉。建科院，有换手，低吸的不错标的。

2:涨价/业绩预增之前讲过，涨价炒业绩的欠揍。现在有了更明确的业绩预增标的出来了。从今天的市场来看。最终目的都是年报的业绩增加，那么，炒涨价，和炒，业绩预增报告来说。很明显，业绩预增更有明确性。但这两个依旧是一脉相承的。今天的新凤鸣终于板了一个，，，目前市场资金很躁动，所有的涨价/业绩增长概念，都会成为他们的标的。

今天从医院回来的比较晚了，所以复盘只能简单谢谢了。就简单看看吧。最近题材轮动太快了，热点转化风格十分迅速，追高一个不小心，就是吃面。建议还是多挑挑暂时蛰伏的板块作下低吸，总有轮动到的时候的，比如我昨天的低吸的芯片，今天是不是就轮动到了？明天周五了，资金谨慎日，应该会比较寡淡，祝大家赚钱吧。
