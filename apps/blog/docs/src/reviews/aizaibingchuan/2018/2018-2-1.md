# 2018-2-1复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/_7iAOdBU9LyE1pmeKKi14w>

## 原文内容

**闲 聊**

年底了应酬多，今天晚上有事，所以复盘就简写了。先简单讲几个事。

1：要信历史。前几天的时候，我给大家做了个表。你们可以回过去看看。写的很清楚，春节前第13个交易日开始下跌，从历史数据看，概率在第9个交易一日止跌，那就是明天，然后盘整，下周3开始一路小阳。以上都是根据历史得出的经验。可能你会说，扯犊子，人不会走过同样的一条河。但是你要知道，千万年来，人心，永远没有变过。跳动的数字，只是人心的直观体现。之前我认为，有gjd高控盘，最多就跌个4%，到3443点。目前来看，情绪来的比想象中的猛烈。

2：要相信基础技术最基础的技术，是大众都知道的技术，有共识性，大家都会这么做。上周五尾盘的时候，我说：然后我周五，周六，周日，三天的文章，从技术面讲了为啥要跌周一的文章讲了要跌多久，和到什么点位。目前来看，大盘的杀跌，已经超乎了我的想象。。我知道，看我复盘的人，很少会听进去，管住手，规避风险。因为呢，我也是个管不住手的人。我们都一样。。。但，我很多时候，我有明确的止损点，该割我就割了，但是很多人，死拿不割，天天问我，怎么看，怎么办。。这个，问题，我很难回答。。。因为，我毕竟不是股神。很抱歉，没有办法一一回复的。。我也是个天天割肉的小韭菜。我们都一样。从1月10号至今，我已经割了近50w了。。。。

3：不要干一致的事情。这里有人要喷我了，因为我昨天说今天要有反弹，早盘我一看，这个反弹来的太早了这就是一致性的预期，大家都觉得要反弹，就有人抢反弹，抢反弹的结果，就是，恐慌盘没走，拉高了抄底的人又不愿意进，再次引发多空失衡，再次下跌时，会比原来的更强。这部分，我更新在了昨天复盘的置顶留言里。从这个时间开始，市场开始普遍发现这个问题。立刻开始28分化，然后共跌了。

4：讲明的观点，我不会重复讲。看到就是缘分，看不到就算了。上周有一天的评论里，我就讲了要回避财通证券，周日的文章末端更是写了两个屏幕为什么。今天还有人来问我怎么看。。。我表示，我没有看法了。

5：明天怎么办？很明确的告诉你。这波下跌太猛了，很多技术指标都超跌了。下面会有修复指数还会再杀一杀，但是可能是纯杀指数，不杀小票的杀，因为，小票已经没有空间跌了。这个我以前讲过。以前我画过一个比较好看的图，没找到，今天草绘一个了。。原理都是一样的。

**赚钱效应**

赚钱效应：继续1%
在这里，我只能说。卧槽，跌到屏幕外面来了。。。当然这样的日子以前也有。。其中最右边，和目前的情况最像
如果你拿的票，连跌4天，没割肉，在这个位置，我也不建议你再割了，等反弹吧，这是市场需要。

**大 盘**

大盘收在了我昨天画的黄线之上。。从15分钟，30.60分钟线来看，基本都超跌背离了。其实昨天就超跌背离了。
主要原因还是因为今天盘面有个利空。。就是那个降信托的消息，这个其实已经讲了很久了，这也是我一直以来说的，要回避10大股东一堆信托计划股的原因。
可以看下，今天闪崩的股，大多都是这种。
明天，从技术上，有反抽的需要了。

**热 点**

看到这里我就生气，昨天卖的永吉股份，又板了，又卖飞了。。。
今天大部分的板都是烂板，相信我，这里面会出一个妖股的。弱市出妖股。
是哪一个？我不知道，看看明天谁能连板把。。

**个 股**

个股复盘：
昨天卖的永吉股份，今天看的我就生气。
目前我手上有三个板，英科医疗（之前潜的）天永智能（今天排板的）今飞凯达（昨晚复盘写的）
明天我准备死拿，期待运气能再来一次，出个妖股算了。
英科医疗就没啥好说了，钱几天贴过图了。
天永智能，我前天复盘的时候，讲解开板新股参与的时候，有写过这，这是最适合炒题材的，所以我今天毅然打板的他，而不是万兴。
今飞凯达这就没啥好说的了，最近复盘一直讲，今天想先抄底抄一点，做底仓的，结果他就板了。。。
另外还有一个中线潜伏股，本来是轻仓买的，可能那时候买的时候敲错键盘，仓位有点重。我手上这么多板，都敌不过那个股今天一天给我的亏损。。。太惨了，惨剧就不跟你们分享了。。。

个股展望：无。
我每天的看好标的，都是我深耕半夜，夜深人静之中思考出来的。这会要出去浪了，难道我还能写出看好标的出来？
当然从盘面里面的思路还是可以看到一些。
1：高送转和开板次新。今天的盘中的留言讲的很清楚，题材真空期的时候，高送转很容易拔地而起，今天如期应验。目前很闪崩的股，都是流通性太差了，但是开板次新流通性好。所以从炒题材的概念来说，我个人认为，高送转和开板次新，依然是炒作的重点。
2：上证50，明天不参与大盘今天放量了，但是护盘兴致明显，对标历史来看，很容走出这两天的走势，明天还有一个下影线，但是，明天大概率是杀指数，小票不动。当然内含信托的票，依然有可能会继续闪崩。记住，放量一般有两种情况1：见顶2：见底这波下跌厉害不厉害。厉害！今天放量了，所以阶段要见底了。除非明天继续放量大跌，那样的画面太美，我不敢想象。如果你相信历史，下面3天是盘整，还会打出指数低点，还会有闪崩，也会有大面积反弹。如果出现大面积反弹，那么请暂时高抛离场，真正的进场点，在下周二三。请有点耐心。

关于我每天的复盘，最近写的越来越力不从心了，感觉睡觉太晚，心脏也不舒服，胃也不舒服。吃东西也没胃口，早饭吃一个包子，还感觉恶心想吐，都是强咽下去的。股票类的公众号，乔帮主写了连载小黄文后就断更了。刺客也断更了今天霸哥也over，虽然我还私聊了下霸哥，说要坚持。其实我也特么的坚持不下去了，市场好的时候还好说，心情愉悦。亏钱的日子太难受了，一遍亏钱，一遍还要写复盘，一遍还要被骂，一遍还天天有人问，SS票怎么看，BB票怎么操作。从中航黑豹开始，我已经持续亏了快40W了，想想，我当年的本金也就几万块，40w相对于几万块，真是一笔巨资啊。。。。如果坚持模式，好好干，肯定是不会亏这么多的。所以我很赞同霸哥的说法，专心炒股，才是正途。所以后续留言的回复，都随缘吧。看到的，可能会回一下，看不到的，那就是佛性不够。至于那些一遍骂我，一遍还威胁我说，不回复留言，就要取关还曝光我，还要建议周围人都要取关我的人，我只想说，谢谢你，少你一个，我也清净不少。

对于明天的一些观点和机会，我明天早上会更新在下面的置顶留言里面的，有空可以来看看，对了，最后再讲下服装板块，我昨天已经将的很详细了，今天有个新消息。

我也问过一些人，腾讯系的线上，主要是靠京东商城。你每天看到的朋友圈的京东广告，那都是不要京东钱的，他们是一家人。京东商城里面的服装板块，弱到爆，是个超级短板，很多都没有，完全不是天猫的对手。下面补强短板最好的方法就是买买买海澜之家只是第一步，这是个高档男装。下面再收购个高档女装是预期内的。什么内衣啊，也是有可能的。

拉夏贝尔是最好的炒作标的，但不是潜伏标的，女装这个板块，后续依旧可以低吸，有3个刺激预期，
