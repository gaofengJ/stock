# 2018-2-23复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/kN18PptgkIB-re1xCMkkaw>

## 原文内容

详细的复盘，周日送上，周五惯例瞎扯淡。

------------------------------------------------

大盘继续缩量，并且中阳。

已经缩量5连阳了，目前5跟阳线，也仅仅是收复了之前两根阴险的失地

白天跳水的时候，后台很多留言问我是不是要挂了。

我更新了置顶留言，统一进行了回复，昨天的大阳没有放量，今天有获利抛压是很正常的，

大阳之所以是大阳，就是他会有他支撑性，大阳之后，大概率的事情是，震荡。

当然也有小概率会连续大阳，但是我们目前的形态显然是不可能的。

极小概率会出现大阴，但是这种情况一般是出现在政策利空的时候，显然，今天的市场也没有利空，所以这种也是不可能的。

所以只剩下了震荡。

所以我说，跳水不用担心，因为既然是震荡，那么有涨，就会有跌

有跌，就会有涨

这就是震荡。

基本符合昨天的预判。

-------------------------------------------------------------------

回复下之前有人在我后台的留言说我是个死空头的。。

这里我并不想做什么解释，常期关注我的人，自然知道我是不是死空头。

我最喜欢的行情，就是那些行情不好不坏的震荡行情里面去找机会。

-----------------------------------------------------------------

再讲个眼瞎的事情，

昨天我先是看见某个扣扣群里面有人讲了句：回天新材  10送5  这句话啪的就锚定在了我脑海里，先入为主了。

昨天复盘雄安的时候，我也想到了他，我点开F10一看。

什么还10派5元？股息率超过余额宝一个股，又是雄安，又是高股息，又符合绿色雄安的装配式建筑，又是超跌，又是低价，还有业绩增长，还有那深入我脑海并不存在的送转10送5.。

他停牌完美躲过了大盘的暴跌，终止重大事项复牌，理应补跌，

当我还做着大低开，抄底，水下拉板的美梦的时候。

开盘三十万封单顶一字板，把我给看傻了。。。。。

即使这样，不存在的送转依旧在我脑海里。。。。

---------------------------------------------------------------

再惯例讲下今天的操作。

今天操作比较多

因为昨天的看好标的比较多，所以我想尽可能的都参与一下。

1：盘前的操作。

昨晚看好雄安里的智慧雄安，所以昨晚就挂单了

但是早上看了下，排的比较后面，就撤单了。

不过帮朋友账户挂单的排到了

2：开盘的操作。

开盘一看雄安太狂躁了。

二话不说，就把昨天打板的阿石创给出了。

就因为开盘多看了会阿石创和数字政通的排单。

出阿石创，和撤了后排的数字政通的单。

又多看了几眼雄安的走势。

导致我错过了昨天复盘里面最看好的南京聚隆。。

错过就错过了，我平复了下心情，又看了会雄安，发现资金狂躁但进攻点不够集中，属于东拉一下，西拉一下，

当时我就判断了雄安已经不能再上了。

所以更新了置顶留言后，我就去我昨天的复盘里面写的其他几个点找机会了。

3：计划中的操作

昨天复盘说了，智慧农业我会继续作T，所以我今天又做了一个T。

智慧农业作T如下。

买点其实不是很好。挂3.81，只是因为他是今天的开盘价，我认为他一定会有下影线，就先挂着了。

没想到才挂完没2分钟，就提示我成交了。。

卖点还不错，因为那会农发种业板了，这种卖点，我以前写过战法。属于板块联动股的卖法。卖的相对高位了。

智慧农业这个股还是相当不错的。

3.92建的仓，目前股价3.93，只涨了1分钱。

但是我通过两个交易日做T股价没动，做T做了13个点的利润。。也是奇葩了。。

但是需要提一句的是，下周一智慧农业就要踩10日线了，如果不能如期反攻，那就是走弱了，我就要出掉了。

所以后面我就不打算作T了，准备持股了。

另外一个计划是昨天和前天反复说的高送转板块。

我之前有列过一个高送转股票池。

后面我又讲过元龙雅图有点势弱，从股票池拿掉，增补威唐工业。

所以重新排名，就是美联新才，创业软件，英科医疗，今飞凯达，威唐工业。

创业软件，英科医疗，前几天有详述过。

美联新才，之前我也有详述过，他是我目前心中，今年年报高送转最好的标的，所以给他排在第一位的。

今天盘中，我看到高送转板块有集体性的异动，所以我同步立刻更新了置顶留言。

这里，请忽略我那个脑海中不存在的回填新才的送转。。

其实当时我很想立刻上美联新才的，但是我还是等了下，看到有资金点火美联了，我才上的。

当时那一刻。。请看看我那个如同月亮一样挂在天上的B （买点）

那一刻，我只觉得脑门上被很多小动物反复践踏。

4：尾盘的操作。

昨天还打板了一个股，中石科技，炸了。

但是看了昨晚的龙虎榜非常的漂亮。所以我今天选择了坚守。

守到了尾盘，等来了春天。

尾盘炸板，无力回封，我就只能出掉了。

出掉后也没有其他好的出手标的了，因为大盘下午又拉起来了，我觉得这样并不好，所以就没有作尾盘潜伏了。

5：其他操作

盘中还做了一个杂毛操作。纯粹就是，无聊想打个板，想撤个单，就是这样。。

最后持仓如下

最喜欢这种震荡行情了，可以逻辑买票，持股待涨。
