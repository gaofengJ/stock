# 2018-3-16复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/x6_hmmTiMDvwf8eTI_Z47w>

## 原文内容

周五惯瞎扯淡。

问：龙头都是哪儿来的？
答：龙头起因有很多种，而最后，无一例外，都是走出来的。

昨天写了一篇高送转，有人问我，谁会是龙头，我想说，我哪知道，龙头都是走出来的，没涨出来之前，谁也不知道龙头是谁。

比如说。
之前美联新材，大家都知道好。
到了4板以后，一样要回调，回调的时候，很多人都说，不行了不行了。
但后面还是走出来了。
可惜停牌了。

后面一波的时候，当时的市场几乎没什么人看好威唐工业，一直横盘震荡。
但是最后他走出来了，走了4板。

同理。
如果下一波高送转行情到来的话，谁能成为龙头？
没人知道，最后还是要看市场走出来的。

你能做的，只是做好计划，跟随市场。

比如今天上板的凯伦股份。
这个股我之前复盘又讲过，质地也不错，也在我自选股内，但是我没加到我的计划内。
今天看到他启动了，这个时候能做的，就是跟随市场。

刚好以此来回答昨天论坛的一条回复。
看好的票，当然要买了，既然看好，自然买定离手，成败看市场。
套了就套了呗，看看逻辑还值不值得你拿，值得就继续拿，不值得就扔。

今天市场老师既然看好凯伦股份，那么我肯定也要跟上。
尾盘市场老师又看好超频三，那么我只能跟着再加点仓了。
就是这样。
做自己的计划，跟随市场，进行调整。

我明知道我昨天持仓图一贴，立马就会有人质疑，但我还是贴了，看好又不是一天两天了，买了也不是一天两天了。
没啥藏着掖着的。

多亏万兴科技出来了，结束了我天天吃面的日子。。

关于昨晚的列的股票池板3跌1，可我自己一个没上。。。
