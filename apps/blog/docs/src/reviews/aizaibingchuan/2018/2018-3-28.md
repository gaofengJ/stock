# 2018-3-28复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/JTLOdFYIu2ASVjUNn4wj8g>

## 原文内容

### 闲聊

闲聊一：今天的市场惨爆了。

对很多人来说，比大前天几百股跌停还要惨。

但是，没有最惨，只有更惨。

这个席位大家都知道吧。

日内4碗大面：
3044*0.08 + 3046*0.1653 + 2264*0.1653 + 3141*0.1487 = 1344.8万。

盘龙的盈利和润健通信的亏损大致持平。

一天浮亏 1344 万。

你是不是觉得这简直是个难以想象的事？

其实，多大的能力做多大的事。

一天亏这么多钱，是怎么样的体验呢？

差不多你身价有50万，下午去小区打麻将输了1300块钱。

差不多的体验。

显然，也是非常非常心疼的。

闲聊二：关于目前的市场。

首先你要明白市场的风险在哪里。

其次你才能去看机会在哪里。

风险我最近的复盘写得比较多了。

我其他的账户大部分已经空仓了（前天讲过了）。

只留了超短账户还在玩。

空头趋势下，做超短，就是火中取栗。

取好了，大把栗子。

没取好，那就只能割个肉了。

闲聊三：空头趋势下，空仓毫无问题。

但是难免碰到反弹行情。

有的时候反弹行情走得太强，那就是逼空行情了。

这种行情下，根本不用担心赚不到钱。

参考我昨天写的攻略，以及实操案例。

可以说是稳稳赚钱的。

今天这两个股，盘中都板了。

这就是踏空板的溢价。

隔天10个点的大肉。

你还会觉得空仓赚不到钱吗？

你说你不会打板。

那么如果你分仓低吸其他热点：

买到雄安也是大赚。

买到国改，设好止损点，也不会亏多少。

分仓大致可以盈亏持平加小赚。

闲聊四：市场情绪变化之快，已经跟不上节奏了。

很多通用的常识，这两天都派不上用处。

换手上来的股，干不过筹码断层的。

大流通盘的股，还能缩量再缩量。

一度颠覆了我三观。

唯有尾盘的炸板潮，让我感到熟悉的场景又回来了。

现在的时候，节奏一旦踏错，都是超级大面。

看后台留言，我发现很多人从不看风险提示。

死命满仓撸，只看代码，不看分析，就是无脑撸。

愁死我了。

### 赚钱效应

高位股的杀跌，诱发盘面对题材股的打击。

尤其是近期高位的题材，几乎都是重锤。

今天日面 -10%+ 的股，一大把。

今天还有个极限操作的人。

### 大盘

大盘这个形态眼熟吗？

因为跳空比较多，你可能看得不是很清楚。

你把跳空缺口补起来再看。

现在大盘最大的问题有两个（天天讲，累死了）：

1. 外盘，各色萝卜蹲。
2. 以上证50为代表的价值投资股被抛弃。

这两个对大盘的杀伤力特别巨大。

大盘走不好，小盘股虽然有机会，但还是很影响情绪。

比如上午上证50在10点后杀跌。

带垮了指数，带开了一堆板。

尤其是创业黑马的带开回封无力。

直接为今天高度股杀跌埋下了伏笔。

关于价值投资，本来就是个超级大的谎言。

股市的最大题材股之一。

这是我昨晚写完复盘后去看的。

今天茅台如期大跌。

创业板是真的强。

下面的行情大概率是：

大盘横得住，创业板就涨。

大盘跌，创业板横。

大盘大跌，创业板跟跌。

13年的时候那年春节，价值投资也是吹上了天。

然后春节时，价值投资见顶，创业板开启大牛。

这一波会不会这样，我不知道。

我只能说，有这个预期。

### 热点

次新股及同年涨停股依然多。

但是 3、4 连板股急速下降。

说明市场风险增加，开始逃离高位，切换至低位股。

今天涨停太多，不一一点评了。

### 个股

个股复盘。

早上的时候，手上的票全部涨停。

一度差点以为悟道了。

下午的时候，手上的票全部嗝屁。

超级大肉到亏钱倒贴。

明天还有提计。

昨天写得好好的，今天又手闲了，打板全炸。

早上：

天华炸板回封无力，卖光。

蔬菜炸板回封无力，卖光。

思特奇炸板回封无力，但感觉没走完，所以卖了一半，先锁定利润。

安达韦尔，昨天复盘有写。

今日策略死拿，日内回落 10% 卖掉。

按计划执行。

我知道肯定有人会喷我菜。

交易这个东西，反正是盈亏同源的。

有时死拿能赚钱。

有时死拿赚不到钱。

盈亏同源。

万兴科技途中不知道嗝屁了多少回了。

不死拿早卖飞了。

另外盘中还割了中成股份。

这个股本身就博弈二浪，没有特别强的逻辑。

所以用了通用止盈止损模型。

日内回落 5%，无脑割肉卖出。

（本来一直是挂单卖的，但是别人砸得太狠了，一路向下，无奈我也只能往下砸了，所以图上显示了一个分时最低点。）

本来今天是不想操作的。

但是超短账户想想小仓位打打板也没啥问题。

看见 5G 龙二万马科技，换手2板，打，炸，卒。

看见雄安龙五，前期龙头京汉股份板，打，炸，卒。

看到高送板块起来，打了第一个上板的凯伦股份，打，炸，卒。

尾盘还买了英科医疗。

因为时间点到了，他快要出除权公告了。

就这几天晚上就要出公告了。

所以少买了一点，做潜伏。

就这样。

惨。

持仓如下。

另外，我今天给其他账户做了一笔我认为最安全的操作。

md，吃了个跌停。

更惨的是，我是卖了下面这个股，买的国投。

怀疑人生。

一来一去，损失20个点。

个股展望：

昨天的展望，太特么神了。

强势股低吸一栏，贴了3个全线跌停。

我特么的人生第一回遇到。

所以今晚不写看好标的了。

太吓人了。

如果我明天要参与，基本会从今天的首板和二板票里面下手。

今天这些2板票封还是比较稳的。

今天盘中的炸板潮，基本反应了资金关注点在切换。

之前的老题材，资金已经不爱了。

喜新厌旧了。

今天的二板股最后我再讲下。

目前的行情下操作，就是火中取栗。

要操作就要做好亏钱的准备。

不过说来也神。

自从我喊了空仓，第二天涨停潮之后，阅读暴跌。

让我感觉好安心。

祝好，晚安。
