# 2018-3-31复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/gWAVe33TO0xMybEY_5WXYA>

## 原文内容

最近大家有没有发现，
接到的广告电话如同这骤热的天气一样，急速升温。

之前电话大概每个礼拜接个两三个，
现在每天大概20+，
大致都是说：
你好，我这是XX投顾公司，
（YY财富公司,XX智啊云啊什么什么高大上的名字），
请问你还炒股吗？
我这有一只下周要爆拉15%的股，你要考虑下吗？

你想想觉得可能吗？
有个陌生人打电话给你，要给你送钱。
天上给你掉了馅饼，你信不。

关于这些，
我在之前的帖子里讲过。
股市骗局全揭秘（附爱在冰川亲身被X经历）

在这里在跟大家讲下，
市场里每天给你打电话，加微信，加扣扣的，
大致有以下几类。

1：有投顾拍照的投顾公司，（合法）
2：没有拍照，但是有挂靠的投顾公司，（擦边球）
3：无牌无挂靠的荐股个体户。（违法）

以上这三类，
基本上都是作收费荐股的，
你的所得：
对方定期向你推荐股票。
（股票不包亏赚，有亏有赚）

你的失去：
定期失去的所谓的会员费，
几千到几万不等。

投顾公司给你推的股，
大多是基于对方分析师分析后推荐的股，
有的会涨，有的会跌。

股票的涨与跌嘛，
就像硬币的两面，
不是涨，就是跌，
没有第三种情况。

其实投顾公司也挺难的，
推热点吧，很有可能就接盘了，
赶上躺了吧，涨了小散户又不会卖，
硬生生又变成了亏损。

有的时候赶上回调了，
小散户割起来利索，
后面涨起来的又不敢进了，
干巴巴的问投顾公司说，再给一个再给一个，
然后，一买又套，如此往复。

这个时间上没有股神的，
成功者大多都是在输输赢赢之中博弈胜率的提高，
以及其中偶发性的大肉。
长线投资也是如此，
只是验证的时间逻辑需要的更长而已。

而你，
只会买，不会卖，
一样是没用的。

下面继续。

4：卖软件的公司，卖指标的公司。
本质上和上面的三点是差不多的。

你的所得：
得到一个软件或指标的使用权。

你的失去：
定期失去的所谓的软件和指标费，
几千到几万不等。

但是所有的指标也都一样，
有胜率的问题，
你每天筛出2个股，3个股，4个股，
你会发现，
只要你一旦作了2选一，3选一，几选一的，
最后肯定会错的，
这是一个怪圈啊。

这一类的更不建议。
通达信就很好用，免费，
网上指标一大堆，随便下，随便用，不要钱。
同花顺里面也是自带指标大堆。
（同花顺还有指标平台，想下啥下啥）
用过都知道。

以上4类，
属于一个愿打，一个愿挨，
损失可控。
有些人钱多，
愿意付个几万块钱去尝试一遍试错下，
再下就不阻拦了。

但是下面这种，
请大家务必要小心。

5：骗子公司。
这一类就非常可怕了。

你的所得：
天天有老师跟你嘴炮，
反正推的票也是有涨有跌。

你的失去：
套取信任后，
能骗掉你的钱，
是上不封顶的。

今天我写这个吃力不讨好的帖子的主要原因是，
这两天后台有个留言，
一开始跟我说，
说进了个扣扣群，
老师非常牛逼，
有2亿资金，要带我们飞。

讲到这里，
大家换位思考想一下。
你有2亿资金，
你愿意去带素不相识的人，
帮他们拉升，帮他们封板，帮他们赚钱？
你愿意不？

我当时说，
这个你谨慎点，就好了。

周四的时候又来留言了，
并且编辑了超级长的来龙去脉，
这里我把他写的公司名字和老师名字给隐去了，
因为写不写名字都没啥用，
他们都是骗完一波，
就换个名字骗下一波的。

还记得我之前讲的股市吗？
之前把我带沟里的直播前老师，
上个月还叫杨博乐呢，
下个月就叫牛犇老师了，
还要求大家亲切一点，
叫他大犇老师。。

反正，
想要不被骗，
最好的办法就是，
有人舔着脸要来说，
我有好多钱，
我要带你赚钱的时候，
你就是三个字，
别上当。

再核心的话，
就是，
千万不要把钱转去莫名其妙的平台作交易，
一定会黑光！！！

顺便讲两个题外话。

第一个题外话，
我周四看了这个朋友的留言很感触。
我就顺手点了微信上密密麻麻的要求加好友的信息，
随便通过了一个，
想去看看现在的套路。

然后我就加了一个好友，
加完立刻托我进了一个群。
当时我就看到，
咦，一个人要上钩，
然后我就加了这个人的好友，
心想能挽救一个是一个。

结果加完好友，
我看了下他的资料发现是个托，
然后我耿直的说了一句：
抱歉，原来你是托，冒犯。
这个人回了我一句：有病。
然后我就被秒踢出了群。。。

不得不说，
这位群演，
你就不能假装上个头像吗？
一次失败的拯救行动。

第二个题外话，
我个人不加微信好友，
也不加扣扣群，
如果有人以我的名字来主动加你好友，
那一定是假的。

比如瞅瞅我后台，
光是关注我的叫“爱在冰川”，
就有6个！！！
我也不懂这是想干嘛的。。。。
我的微信名，
肯定不叫爱在冰川。
干啥事，都要低调。。

炒股已然不易，
不要再轻易上当了。
如果你身边有朋友，
有被骗的征兆，
请及时提醒他。
