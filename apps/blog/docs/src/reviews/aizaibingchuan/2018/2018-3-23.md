# 2018-3-23复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/1DSL1i49gFqDTUHuPZcO3g>

## 原文内容

今天大家都亏惨了吧。

都一样。

今天开盘一看，持仓股全部到了纪律止损线。

无奈，全部纪律止损。

亏成狗，又到了一起汪汪汪汪汪的时候了。

昨晚复盘写了两个看好方向。

今天直接成为了主流，我昨晚挂单的哈高科，完全没有机会。

9.20之前忘了撤单，所以又没赶上集合竞价进万兴科技。

开盘之后，看着满手的负十几个点，无奈，纪律止损了。

其实，我对手上的持仓票，都相当看好。

但是吃过太多看好不割，最后越割越多的亏。

很多时候纪律，还是要摆在第一位的。

不过，由于对712相当看好，所以只割了一半多点，还留了点在手上。。。

万兴科技下午炸板的时候，受了点外因影响，出了一半。

胸痛。

后面再找机会加回来了。

今天还开了3笔仓。

基本都是从贸易战的角度出发去看的。

可以看下我白天的置顶留言。

贸易战上。

拆开来三个字“贸”“易”“战”

早上的炒作基本在围绕“贸”字上。

昨晚说的复盘其实已经说了。

今天基本上是在以炒大豆为主了。

然后延伸到了农业板块。

而我昨晚自己定的计划的标的，今天全是1字板，，完全不给机会。

无奈只能从昨晚的复盘的另一个角度去想了。

苯二酚的反倾销政策的延续。

可惜这个板块没找到叠加次新的股，唯一盘子小点的兄弟科技，历史股性也比较差，所以买了一点点。

显然没有得到市场的认同。。

中午的时候，我就在想，贸易战必然会引发冲突。

冲突是多层面的。

看看历史，也是这样的。

所有引发战争的开端都是一样的，各帝国主义经济发展不平衡，秩序划分不对等，占有资源的不对等。

无论是萨拉热窝时间，无论是31年的918事变，还是39年的闪击波兰，背后折射的都是各种不对等引起的矛盾。

最后终究是要靠拳头说话的。

现如今的时代，信息发展太快，真打出现的可能性及其的低。

但是并不妨碍情绪的渲染，中国的股市，大多时候，你只能通过做多来赚钱。

聪明的资金总是非常的敏锐。

这种非常时期，你炒军工，做大军工的市值。

变相支持国家的战斗力。

而且炒军工还可以降低被监管的几率。

为什么呢？

你变相想一下，我炒高你军工企业的市值，可以让你军工企业融到更多资金，可以让你投入更多的研发，造更好军备，保卫国家。

我这么美好的初衷，如果你来监管我特停我，请问，你的党性在哪里？

所以当时我觉得军工有机会，除了更新了评论立刻水下低吸了两个次新军工。

（其实当时最应该干的是，低吸回最新的次新军工：712，那会是-9%！！！但人都是一样的，减仓就不想买回来了。）

最后持仓如下:

下面4个是在上午反弹的时候，陆陆续续割掉的。

今天亏的蛮惨的。。

最伤心的是出了一半的万兴科技。

原因是今天论坛发了篇文章，所以中午顺便逛了逛论坛。

看见某大V看空万兴科技，，讲的好像很有道理的样子，，，下午真炸板了，炸的时候，我就这么信了，出了一半。

唉，拿妖股，基本就是靠死拿。

任何揣测，都是没有意义的，万兴科技，成妖的路上，多少个炸板，烂板，都没动摇过我。

今天居然因为一些流言蜚语动摇了我的党性。

最后，老铁会发现，年后回来我的操作风格跟以前完全不一样了。

但是该赚的钱，一样会赚，该亏的钱，一个不落。

其实，无论是打板低吸还是追涨，操作的方式，根本不是赚钱亏钱的核心。

核心应当还是在理解市场，去寻求资金渴望赚钱的一个共振点上。

去分析哪些股上面，会出现这些共振点。

这样的股，无论你低吸，还是追涨，还是打板，都是赚钱的。

因为这些只是术或者法。

年后回来以后，分仓操作，只是为了更多的试错机会。

后面的话，模式我应该还是会切换回去的。

很多股你光看，不从上面吃口肉，或者割块肉给他。

你的体会，始终只会停留在表象的二维平面上。

你只有深入操作后，才会有三维的体悟。

最后还有个题外话，大佬们都已经在4维空间了，我暂时还不能理解那是个什么层面了。

毕竟我还停留在算今天亏了十几万是相当于我几十月的工资的层面上。。。

最后，祝，周末愉快。
