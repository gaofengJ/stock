# 2018-1-17复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/69bxI3Qfpf6sVpCkDMs9PA>

## 原文内容

**闲聊**

闲聊一：

昨晚的暴风骤雨般的区块链监管，今天区块链板块如暴风骤雨一般的飘零。

早上一开整个板块都在地板上摩擦，看见科蓝软件集合竞价时，最后的抢筹我知道，这是为了能让大部分的区块链个股能在开盘跑路的机会。

如果没有科蓝软件最后的抢筹，下面这些股将会一点流通性没有，现在他们至少还等来开盘一瞬间的解放军，解放了一部分流通性。

闲聊二：

关于预期差这是非常重要的一点内容，你一定要看。

当一个个股所有人都看好的时候，那他肯定不能上，大热即死。

当一个个股所有人都不看好他能连板的时候，他就会不断连板。这就是预期差。

昨晚贵州燃气特停，所有人都看好今天一定有次新卡位成龙头，我也不能免俗，当所有的预期都很一致的时候，看看今天的表现。

早盘纷纷卡位上板，然后这些股开板的主要原因，很大程度都是因为早盘大金融的抽血。

这就是我昨晚写策略的时候，最重要的一条就是再看看今天早盘开盘的位置，这是今天早上开盘的走势，白线勇猛之上，黄线一落千丈，而不是一起上。

这说明什么，说明这个上涨，不是游增量资金带来的普涨，而是大金融从小盘股抽血来上涨的。

大金融急拉，小盘股急速抽血。

那么上述那些缩量的卡位龙头，自然就筹码不稳了，有人想卖，有人要撤单，那么纷纷开板，也是意料之中的了。

这就是我昨晚写大盘的时候，写了这条的最主要的原因。

没人能知道明天会发生什么，但是可以对各种情况做好计划，出现什么样的情况，那就作什么样的策略。

**赚钱效应**

盘中最灿烈的时候，跌停数超过了涨停数，照当时那样的啥跌情况来看，即使来个百股跌停都不奇怪。

但是从这个时候开始，风格又开始发生了切换。

赚钱效应今天的赚钱效应极差，冰火两重天的行情。

**大盘**

大盘今天新高了。

昨儿说了，大盘还有上涨的空间，今天就新高了，但这种没有赚钱效应的新高，并不是大家想要的新高。

今天收了一根放量上影线。一般来说，上涨路上的放量上影线，接下来的基本就是要横盘震荡了。

但是呢，大盘现在是一切都有可能，毕竟昨天的新闻也说了，

1. 新韭菜正在跑步进场。7天募400亿投A股，基金如此热销多年未见。
2. 基金大卖400亿，这场面，我只在牛市的时候见过。

如此亮点也刺激了今天券商继续大涨。

关于明天，我的建议依旧不变。

**热点**

今天次新依旧火热，资金从高位次新接力，转向了远端低位的次新。其他的直接看图吧。

**个股**

个股复盘。大盘大涨，我的曲线大跌，我准备休息一段时间了。

这一个星期的操作跟我10月的时候如出一辙。10月份的时候经历了川金诺的黑天鹅之后，一度横盘了近3周的时候，才开始有所好转。

今天同事教育我说，尼玛最近天天追热点，天天跌停割肉，还是要跟以前一样做杂毛股潜伏有前途，我想想也对。

所以接下来，我准备继续回去做我的杂毛股。最近一个礼拜追热点，追的我遍体鳞伤。

我今天花了点时间，回顾了我最近一个月以来的操作我发现了我潜伏了很多好股，都没拿住。

比如：太多了。。不贴图了。

开盘重仓割了泰奇猫，没什么好说的了。

今天又买了一点集泰股份代码是一方面的原因，另一方面的原因是因为早上看新闻说有机硅又涨了。

我早上的原计划想把昨晚复盘说的集泰股份和庄园牧场，等仓上的。早上看了新闻后，就都上了集泰股份。

下面开始我准备继续切换回去作杂毛股潜伏了。因为是潜伏，所以就不贴持仓了。

**个股展望**

写在前面，如果你下面还准备作题材股，那么上海股千万别买了，这应该是游资的共识了。今晚又停牌了两家。

因为大盘涨的好，所以他可以放心大胆的特停，因为不怕跌。

今晚就写一个标的把次新和高送转。

本周次新上会13，很多人都说，这个礼拜完蛋了。但是来看看今天的画风。38%的通过率。。

以前，解决排队多的问题，去年的办法是，死命发行，今天的办法是，死命NG，证监会也是GET了新技能。

今年次新的减少，变相的利好去年的那些像死狗一样趴在地上的次新。

从这两天的行情来看，大部分的远端次新，都在走连续小阳。

那么如何从这些远端次新里面寻找合适的标的来做潜伏？那自然是叠加概念。

别如说，缩量两板，今飞凯达今天也是一字。也就是说，潜伏预告送转标的，又有肉吃了。

你看，人家潜伏到的，10%，22%的肉，大口大口的吃。

那么，就变相利好那些高送转预期的个股。

所以你潜伏中远端次新的时候，最好的就是选择，那些，前期有人气，业绩不错，公积金高的个股，去作潜伏。

今天盘中也有一个信号。川金诺盘中点了个板。我看了下他的点火阶段，基本都是小散单的相互抢筹造成的。

这说明啥，说明市场中有很多人都已经有这个想法了，都在相互关注了，另外还有一个信号。

杭州园林也同样点了板。杰恩设计的两联板，对整个高送转板块都有带动作用。

所以接下来可以适当关注下高送转题材了。

A：做预高送转潜伏，尽可能选择那些之前有人气的，有过炒预告送转连续大涨的，业绩是赚钱的正增长的，不要选上证的股，超跌的。（像今天川金诺那种高位下来差不多腰斩的股）

股票池：川金诺，星帅尔，金银河，园林股，设计公司股。

B：已公布高送转中，今年的已公布告送转板块，整体标的都很差，只有三个小盘次新，杰恩设计，今飞凯达，杭州高新，值得一看，但他们都有老庄股的痕迹。今天凯普生物盘中也是一度冲板，也是一个信号。

C : 中报填权股今天的杭州园林就是一个扩散信号。

股票池：杭州园林，皮阿诺，达安股份，（老三样了）

以上，如果你觉得最近不顺的话，最好就休息休息吧。

我反正准备开始休息了，调整下状态，最近亏的太灿烈了，静下心。

上文说了，我今天看了下，我这波的亏损的，和10月的川金诺后期亏损如出一辙，都是心态炸了，模式开始走样，然后又碰上了强力监管这个黑天鹅。

唉。一个字，痛。

我知道今天一定会有很多人问我怎么操作手上的个股，我想说的是，设置好止盈止损吧。
