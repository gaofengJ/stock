# 2023-2-3复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/HaPdD32js6-Kv4n4tq5PcQ>

## 原文内容

本复盘仅为鄙人的股市思考历程，非荐股，请勿跟票。

周二的时候聊过，指数要开始做盘整了，这个是量决定的，今天创业板的收盘价和周二的收盘价几乎一致。
指数要走长时间的单边是很难的，中间都一定要是有个盘整过程的，盘整完了再重新选择方向。
之前汇率也是，这一天的时候，我说然后他就开始做平台震荡了，另外上证指数的3300点本身也是一个比较强的压力位，这个点位在11~12业绩真空那波上涨行情的时候就聊过。
指数这里盘一下，做一下区间震荡，是一个合理的事。

每个人对市场的理解是不一样的，一千个人眼里有一千个哈姆雷特，所以市场是多样化博弈的，所以市场对题材的审美也是千变万化的，有时候还是奇奇怪怪的。
就像我家飞来了一直鸽子，有人觉得要割了，有人竟然觉得是鸽尔。。。
所以在这个市场里，千万别说什么“我们要提高审美” 云云之类的话，要用调整，不断的调整审美，向钱看的审美。。。

祝周末愉快。
