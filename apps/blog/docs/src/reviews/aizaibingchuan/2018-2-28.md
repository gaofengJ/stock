# 2018-2-28复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/VkWuVN_mjwDQo6IFqhieJQ>

## 原文内容

**闲 聊**

本周周末会发行IPO对市场会是个情绪利空，
所以周五肯定有人会砸会跑，
当大家都知道这个事的时候周四肯定有人要抢跑
本来大家还遮遮掩掩相互博弈的时候，
昨晚的特停轰炸，直接就诱发了一致性，
就是多空观望的大家集体提前跑路了。。

这里就要讲一个有意思的地方了。
昨天特停的几个股，一个是代表的雄安板块，
一个代表的是泛股权板块，一个代表的是LNG板块、
最后今天死的是次新板块。
这，就是情绪的杀跌。
而次新板块，就是市场情绪的反馈最直观的地方。

最近上市的20个股，去掉2个没开板的，6个跌停，
基本包揽了所有的自然跌停股。
今年上市的第一个股，新疆火炬，2个月不到，成功完成腰斩。
最近一个开板的三个核桃还缺3块钱，就破发了。
在以前，新股破发，很多时候，都是大底的开始。
而我们现在，离新股出门就开板，好像也不远了。

那么我们接下来是，缓慢筑底？
还是加速赶底？
还是震荡向下？

**赚钱效应**

赚钱效应又开始转差咯~~~~
今天炸板率40%
下面的行情，有种有复制之前这一段的走势的感觉了。
总之，又到了要开始谨慎的时候了，，

**大 盘**

大盘进入微妙期，
今天受到各种坑爹影响，大低开，然后缩量震荡。
大盘原文macd要金叉了，目前走平了，
这就很微妙了，如果后续金叉，则还可以继续玩一波，
如果水下死叉，则威力请参考水上金叉或者空中加油等技术指标的走势
（当然你要转一下坐标轴）

目前的主要问题是，上证50上证50的这波缩量反弹太厉害，
所以目前有获利盘回吐，和之前禁卖的卖了，也是预想当中的。
这个位置，没有必要跌了就仍未要加速探底了。
大盘这波急跌的厉害，所以急反弹的也厉害
这种走势，一般要缓慢几天，走平，
然后再决定方向，是不是要继续探底。
你可以参考收敛三角形的走势及同年华森只要的分式，
堪称经典的收敛三角形走势

原本预期是周四开跌，周五震荡，
这两天提前跌掉了，明天会有一个指数上的弱反弹，
当然前提是不要有什么幺蛾子出来。

**热 点**

昨天题材遭到打击后，今天资金转向进攻了业绩增长股，
而且大多是符合新潮流的科技类业绩增长股
这个脑洞，今天白天看了走势才想到。。
只能叹自己太愚钝了。。

**个 股**

个股复盘：
昨天复盘说了，美联新才我会锁仓的。
所以，我也没动，实际上合力的操作应该作个T，
但是非常时期，还是谨慎为主。
真有个什么闪失，还能果断跑。

美联新才这个这个三板，是有重大的意义的，
他的意义是，讲他从大家嘴炮定义的年报高送转龙头，
演变成了实锤的年报高送转龙头。
就像讲到17年报高送转龙头，你肯定想到的是新洪泽，
16年中报，你肯定想到的是煌上煌，
17年中报，你肯定想到的是杭州园林，
就是酱紫，高送转的龙头，除了本身质地之外，
这个换手三板，就是盖章一样，盖了上去。
（本来我把，美联新才的下一波博弈机会和时间节点都写好了，但是我最后还是删掉了）
各位还是走一步看一步吧。

昨天买的海川智能，有说，今天要看三丰智能的走势再订，
开盘一看，三丰智能上冲后翻绿了，
基本可以判定，今天没啥机会了，
海川智能就亏2%出掉了，刚好还是小高点。。

另外今天盘中迸发出一条逻辑。
认为今天这狗屎一样的次新，下午一定会有一个情绪逆转板出现，
并当时更新了置顶留言，所以我就买了点奥飞数据，
这里，我只能说，逻辑没有问题，但是押宝显然太难了。。

下午资金确实选择了一个次新，发动了一下，
从水下差不多-4，直线拉板。
然并卵，拉的太冷门了，对次新板块毫无带动作用。
所以他自然也炸了。
他今天这个图形，明天也许会有资金抄当初南京聚隆的图形作业，也可能没有。

持仓如下。
噢，对了，贴图的时候，我才发现，，
我今天还打板了10手天华院，原因无他，
都是连板股，支持他一下，就是等于支持我手里的美联新才一下。
不过我心里还是晓得他大概率还是封不起来的，所有只支持了10手。。

个股展望：
这个地方呢，我不是很建议，去开新仓了，
我觉得下面的肉，很可能会来自于目前这些强势股的低吸。
12/27号，贵州燃气特停，28号29号，尤其是29号，很多开板次新都是大跌。
但是后面呢，这个三板股同样，盘整回调了近10%后，开启了第二波。
当然，当时的市场情况，和现在的市场情况又有所不同。
所以也不可同日而与。。

1：次新
今天次新大跌，很多人都说次新结束了，要坚决回避，
但是给我的感觉，应该还有机会，
其他例子不说，就说一个上面讲新疆火炬，2个月腰斩，
次新板块，真是跌的，超级透了。
还继续举华森的例子，这个股，当初，同样是2个月不到腰斩。
然后起飞。

比如今天下午被从水下拉起上板的这个同样是一个月多点就腰斩了。
我个人认为，今天杀跌，是提前释放了情绪，后面还会有机会。
当然我的观点也可能是错的。
我曾经在这一天也是这样的认为，最后的走势也是啪啪啪啪的打脸。。
市场才是老师。。
这也是我今天盘中先买股的原因，
如果明天走势不及预期，我就可以直接 割！肉！离！场！了。。

明天这些，盘子又小，今天又抗跌的，（跌幅小于4%），的深交所个股。
都会有机会。
另外感谢今天的上证报。。我一搜，云计算+次新，好巧不巧。
刚好这两个昨天都是板，今天都是相对抗跌。。。。。
奥飞数据的尾盘有抢筹，科创信息的尾盘有砸盘。。
我默默的感觉到，我明天奥飞数据可能可以喝上点汤了。

2：贵州燃气
明天复盘了，这个地方有两种走势。
1：参考今天的天华院
2：参考之前这一天。
但是这里有个本质区别，之前贵州燃气是没有套牢盘的，
现在他是有套牢盘的，明天的位置是一个高密度的筹码峰。
所以，我个人建议是，不要上。

3：连板股
连板股，是目前市场的目光所在，
今天的连板股，所以都是深交所的股了。。
这里面依旧会有机会存在的。

好晚了，这个星期发复盘发的好累。。审核死我了。
祝好。
