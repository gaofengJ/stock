# 2019-1-12复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/EHZ4KDdLxLFaiWh7BJYZGQ>

## 原文内容

本复盘仅为鄙人的股市思考历程，非荐股，请勿跟票。

上周因为偷懒（睡觉）导致万字雄文没写。。很可惜，，，
本周把核心大概抽取了点主要内容，当成彩蛋分成两段，分两次发在公众号某些不起眼的角落里面了。
大概有1000多人进行了阅读~~

为什么会说可惜呢？
因为写复盘很多时候都是为了梳理一下市场，梳理一下自己的思路。
为了第二天操作做好准备~~
彩蛋里面也写了，如果上周六写一下，把思路再捋一下，能可能这周一就无脑买了~~
错过个大肉~~
这周留没啥新东西想写~~

今天比较热议的是方*的讲话，主要讲了8点，但几乎都是口号式利好~
但是几乎所有的媒体文章都把：《建议取消新股首日涨停板限制》做为了标题。
因为这个如果改了是一个规则改动~~
今天也有密密麻麻的人进行了解读~~

我主要讲一下我的看法，在新股出来首日44%之前，次新股是没有一路一字板这种说法的，
一路一字板是首日44%限制之后的产物，
一旦规则改动，那么市场很有可能会回复成之前那种模式，
就是开盘放量，然后筹码沉淀，然后才有可能重新走一波（注意是可能），
在以前的市场里面开盘破发的事情，也经常有~~

而目前的市场是，，一路一字版顶到投后，然后放量，开始一路阴跌，
差一点的标的，还是会破发~~~
如果这个建议，变成新政，并且很快实行的话，，
对与现有次新绝对是一个利空，，因为有更便宜的新股出来了，
你想想首板就能买了噢~~
那么现在这些10个一字板之后还在横盘的次新，，这里面的水分要用什么样的方式挤？

当然这个目前也只是建议，对于市场来说，涨跌还是看资金推送，
资金如果还愿意忘上面推了做，那么我连上涨的理由都给他们准备好了。
想想，这是新政前的最后一批次新啊，这么高估的次新，以后就没有了啊，稀缺了啊，
还不得炒一把，这是利好啊？

-----------

跌是因为有潜在利空
涨是因为变成稀缺了
历史是由胜利者书写的，谁赢了，谁来书写理由，
涨了，这就是利好，跌了，这就是利空~~~

---------------

昨天晚上出门，有生以来第一次在路上看见两个人在血战。
起因仅仅是因为两辆车在路口的时候你不让我不让你，两辆车也没有碰到。
一辆车说，你让让
另外一辆车说我是实习新手，你让让
说话的时候，话语中夹带了一点nmd,cnm,之类的语气助词。

然后突然车门一开，两个人下来推搡起来
其中一人掏出了美工刀，对你没有看错，是美工刀。
一把扎了过去，啥事没有。只有美工刀应声而断。
然后一个人抓着剩下的半把美工刀，另外一个人抓着断下来的美工刀片，在那边相互划来划去。
肉片横飞！！！

当场我就震惊了，这才是真正的割肉啊，相互割肉啊。
我们白天那点割肉算个啥呀？
在这里想跟各位说一句，人在江湖飘，哪有不挨刀。
错了，换一句，是人在社会能低调就低调，能让别人一下就让别人一下，退一步海阔天空。
眼瞅着两个人的伤，最起码得一个月不能炒股了，何必呢？
老夫认为能在股市里文割，决不武割！（照片场面极度血腥就不放了。）

最后讲点真切的，最近看见不少人在喊，被裁员了。。等等，，
社会上戾气者很多，你根本不知道他们的戾气哪里来的，
所以走在路上能苟就苟一下~~~安全第一~~~
