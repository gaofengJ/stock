# 2019-2-25复盘

## 原文内容

本复盘仅为鄙人的股市思考历程，非荐股，请勿跟票。

闲聊

闲聊一：

目前市场普遍高呼，牛市来了，对于牛市的定义，其实很多人都不明白是什么意思，这里有一个名词叫做，技术性牛市（即从技术的角度讲进入了牛市）其门坎就是，指数从低点反弹20%以上，即可称之为，进入技术性牛市。

从本轮低点到今天为止上证涨幅20%深圳涨幅28%创指涨幅25%均已进入技术性牛市。

另外美股。道琼斯指数也是+20%，也进入到了技术性牛市。。纳斯达克也是20%+标普还差0.5%个点另外欧洲普遍反弹了10%，香港反弹了17%

*-----*

依稀记得第一次学“技术性牛市”这个名词的时候，还是在2015年。

大概是这个时候吧。。也是反弹了20%，第一次反弹到了20%，然后媒体喊出了技术性牛市，无动于衷，后面横盘了两周，就是一个千古跌停，

当时候小庆幸了下，还好没上当，后面又涨回来了，又涨到20%了，然后又喊出了技术性牛市，这把我冲了进去了，

心想，第一次是假的，第二次可能是真的，精挑细选了几个好股，准备捂股了，大概捂了一星期，就碰上了举世震惊的---熔断。。

开盘二十几分钟就收盘了。。也不知道全世界有没有。。。。

闲聊二：

牛市的特征有很多，比如韭菜和大妈跑步进场，今天网传什么地方营业部为了开户又打架了，打架纷纷调侃，牛市特征之一。

晚上已经被辟谣了。这个白天一看就是假的啦~~现在网上开户这么方便，佣金又低，还送lv2，谁还去营业部开啊

赚钱效应

大盘

上周五收盘后写到，大盘这里连续构筑N字。

N字的还有一个学名叫做N字多方炮，以前的那些技术流派的老师，看见多方炮，都喜欢手，多方炮后面就开炮，

今天这一炮冲天的，应该是把大家都干蒙了。成交量放到了万亿~~~熔断后的第一次。。

之前讲过，技术指标对大盘已经没有啥用了，都是超买，超顶的，现在就是情绪市，纯资金推动市，没有什么好分析的了。

个股

本栏目仅为鄙人的股市思考历程，非荐股，请勿跟票。

个股复盘（记录我自己的操作，非操作建议）：

昨晚复盘的时候，知道大部分的一线好股，都是一字，所以昨天也没写股票池，今天就准备上几个二线好股，，

结果早上开盘一看，二线牛股都是一字。。。直接就蒙圈。。只能盘中看盘临时找了，

符合昨晚复盘写的，低价的，涨停能阶段新高的，辨识度高的，盘后看看论坛，还有人说他有互联网内容审核概念，能对标人民网。。

希望明天能吃点肉早上打了好几个板，，都没到。。老师们速度太快了。。

今天上午早早的就涨停100+了，后面也没上再打板了，愣是没有想到涨停了近300家。

尾盘买了银星能源，准备明天再低吸点核按钮，做做T了，不然光看盘看的也是蛋疼。

个股展望（记录我自己明天观察的标的，非操作建议）：

一：新热点

今晚没有什么新热点新闻，可以忽略了。

二：现有热点接力博弈

今天近300个涨停，复盘说难非常难，说简单也可以说非常简单。

手上有票，持股就好了，牛市里热点是各种轮动的

手上没票的，要么买最强的票，比如，大智慧，人民网，这一类的（当然，反正也是买不到）

要么就考虑今天的一字板~~有身位的一字板~~

其实最好的是什么股？是那些走出强趋势然后刚开始板的股，在牛市里，这种所有为有资金引导开始首板的股，后面往往都是主升浪的开始~~~

你说，今天怎么没有列代码？我是真的列不出来。。想买的，都是一字。。差一点的去了，又难受。

基本就是这样。依稀记得当年牛市的时候，每天复什么盘啊，持股就完事了，，唉，希望现在真的是牛市。。
