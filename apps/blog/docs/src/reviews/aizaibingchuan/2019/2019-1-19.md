# 2019-1-19复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/mdV95S7pxOQFoU7I09R5vQ>

## 原文内容

本复盘仅为鄙人的股市思考历程，非荐股，请勿跟票。

先梳理下目前周五仅存的连板票
方大集团：3连板，业绩增长
顶固集创：3连板，业绩增长+送转
硅宝科技：2连板，业绩增长
宏达股份：2连板，资源股，磷矿石涨价
坚瑞沃能：2连板，市场热炒的燃料电池的衍生

这里可以看到，5个票里面有4个与年报相关，基本可以看成是年报周期的开始
那么年报周期主要炒什么呢？
最主要的就是两个方向

一，业绩方向
炒业绩，这是年报周期的主要方向，可以细分为

1：公告驱动
就是当公司发布业绩预告的时候，业绩超出了预期，比如说方大集团~
可以看到今年中报，3季报的净利润和去年几乎持平。
但是到了年报，就几乎翻倍，，这个就是超预期，
二且这个票去年年报的时候也是大增63%，炒了一波。
想想也是蛮厉害，
15年净利润1亿
16年净利润7亿
17年净利润11亿
今年净利润22亿

炒业绩呢，，主要就是看公告炒股了，，
每天晚上的公告看一看，，有没有高的公告，开盘就买，，
然后别人看见这个股票今天为啥涨了，，一看噢原来业绩这么好，，然后也买，，
最后板了自然有打板客来打板，，
做公告板基本上是努力的人赚，没努力到这一块的人的钱
当然不管业绩怎么涨，，炒作之后都是一地鸡毛~~
同样以方大为例。18年过年那波最高炒了63%，后面跌了个腰斩出来~~~

2：潜伏系列
有做公告拉升的人，自然就会有做潜伏的人。
有很多公司的业绩预增其实是可以算出来的。。
举个例子~~我今年做了好几次的中石科技~~
还记得炒中报业绩预增那会的，我7-5号列了个股票池~
排第一的就是中石科技，为啥我当时把他列第一呢，以为他是我当时我自己算了，中报业绩增长可能非常高的票
业绩增长有两个告诉，一：去年要低，二：今年要高。
很多时候，要找今年增长很多的票，很难找，但是找去年的很号召，当时找到了中石科技，
17年才42万的净利润，18年中报当时我自己估算了下，大概能有4000万左右，这就是100倍的业绩增长啊。
最后出年报的时候，果然是100倍，。
之前我草估的时候，只是觉得这个票会很高，没想到最后成了两市增长率最高~~~

基本上，潜伏业绩的票，就是要找这种，
1：业绩持续增长（这是首要的）
2：去年基础一定要低。
这样一除出来，增长倍数妥妥的。

再然后业绩增长一般会来源3个途径
A：产品涨价，涨价了利润增加，像最近炒的磷矿石，就是炒的这个预期~~
再拓展到钢铁煤炭等周期股，收益供给侧的涨价，同样在此列，，这个基本上也是市场这两天的主流
B：扩产，很多高成长公司，销售增长快，但是产能不足，上市之后募集的钱，主要就是拿来扩产，，
扩产之后会有非常迅速的利润爆发，比如中石科技就是这一类。
之前还有英科也是的，做手套的，拿来募集的钱，就扩产线，然后利润增长~~~
C：资产出售，或者收购资产后并表。
例如，下面这家公司今年中报后并表龙门教育，机构预计今年1.1亿利润，去年年报只有500万利润。
所以他今年的年报大概率就会是业绩增长20倍左右
潜伏系列还有一些节点，之前写过
以上是业绩方向的主要博弈思路和方法

二：送转方向
在以前的市场，还会分高送转和低送转，
现在新规之后，只有低送转，没有高送转了。
还好炒股软件还算与时俱进~~~
以前这里面。只显示10送5以上的，现在是只要有送就显示~~
因为大家都知道，明面上等的高送转可能没有了，高送转的细分同样有很多。

1：已出的高送转
出了高送转以后，还分预披露炒作，抢权炒作，填权炒作，很复杂，就不一一赘述了。
之前列过了，后面我再有好的，我会再更新的。
（还记得我去年列过的股票池不~~美联新材，智能自控，超频三，太龙照明，，只有一个太龙解禁解废了，其他表现都很优秀）
对于已公布的炒作，每年都是个伊葫芦画瓢的过程。
中间等的差别就是你的葫芦对标的对不对·~

2：有预期的高送转
周五的板的好几个，都是有高送转预期的近端次新。。
最奇葩的就是有人吹雅运股份要送转，，拉上了板，，中午人家立刻就出了澄清公告，，，惊呆了所有人，，，再然后也还是板的~~
寻找优质的送转潜力标的，即使没有送转，也能吃一波潜伏盘的肉~~
比如去年智能自控。再出送转之前，潜伏盘都拉了26个点。。。
