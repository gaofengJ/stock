# 2018-2-26复盘

作者：**爱在冰川**

## 原文内容

**闲聊**

有一样东西很重要，那就是名字。
这是个非常重要的。
因为名字是接触一样新的物/人的第一印象来源之一，会有锚定效应。
比如，今天给你安排一个陌生人旅行，给你2个陌生人，让你二选一一个陪你出行，
那么，你会选择 王根妹 还是 凌志玲 ？
妳会选择 张二狗 还是 刘杰伦 ？
同样，你买股票的时候，两个股看上去差不多，你会买哪个？
A股，自古三个字多妖股，
明天很多涨停股或者大涨的股，名字真是好。
昨晚你看到的那些好名字股，今天没有走好，
那是因为大热即死，高开太多，关注度太高，都很难走好，
但是于无声处起惊雷，
昨晚你没关注到的，今天就会走的很好，
比如等等等等，每个名字都能拓展出来，写一篇颂词

**赚钱效应**

今天盘面除了大涨和涨停多以外，
最好的一个地方是，跌停板家数降到10家一下了
今天涨停家数很多，所以明天承接会很困难，
大部分今天涨停股，明天都会冲高回落。这是一定的。
上一次这多，是节前的2-12号。
在第二天，那些涨停股在2-13号的综合走势是这样的。
所以，明天心里要有点数。
手里的票，要怎么卖，什么时候卖。
你应该有点感觉了吧。
不过这只是综合走势，个别独立题材的股，自然有他自己的独立的走势。

**大盘**

如昨天复盘所说。
大盘大概率跳空高开在这个位置。
今天如期兑现，从盘面看，机构限卖的要求可能解除了？
开盘上证50高开后，就直线出逃。
地产领跌，但是这个时候，出现了非常好的一幕。
大盘跳，小盘不跟跳。这就是风格的切换。
从而判断今日无忧。
再讲讲明天，明天大概率是震荡行情了。
阳线后的震荡行情。
今天主板被创业板救了一命。
可以看到，深证，和创业板，都已经完成了补量了，量能已经开始回复了。
主板依旧是阉割量。
参考我今天的置顶留言。
之前我讲倡导价值投资的时候，曾经提过一次2013，
这是一个比较大的话题，篇幅会很长，我打算周末再来写一写。
但是也不要乐观，而是要谨慎一点2013还有那惨绝人寰的最后一跌。
明天的行情，总的来说，震荡行情，后天也应该是。

**热点**

总的来说，科技股，今天爆炸了。
盘中爆的超出了很多人的预期。
所有你看不懂的东西，盘后都会有解释。
信息不对等，这种事情是没有办法的。
在A股因为有涨停这个制度，所以打板客永远不会消亡，
先上车，后补票。先打板，再研究为什么板。
所以打板，永远是最快致富的路，也是最快亏损的路，
也是最简单的战法，同样也是最难的。
回归正题，今天科技股高潮了，高潮之后，自然是分化，
今天这种程度的高潮，明天科技股里面前排品种大概率是要出连板股的
后排品种基本上都是冲高回落、高开低走的命，

一个好标的，不是写1000字的论文才是好标的，
而是把他能获得市场认同度的点领出来，别人一看就懂，
