# 2018-1-22复盘

作者：**爱在冰川**

## 原文内容

**闲聊**

闲聊一：解股

正常来说，我是从来不分析个股，今天刚好看到一个，关注我超过半年的老铁，提了个问题，讲到这个荣泰健康这个股，我心里也是默默的哔了个狗。

我上上个周日的复盘里，简单讲了个从生活中去选股的方法。

如下谈谈如何选股（2018-1-14 爱在冰川复盘）本意只是举例，没有任何荐股的意思。

但这个股确实不错。阶段新高横盘4天，本身业绩好，增长稳定，在行业内也是龙头。

我在上文中也说了，这是妥妥的价值投资股的典范，但是我个作短线的是不会去参与的。

当晚就有人留言了，说我这是在给庄打广告叫我们去接盘吗？

我心里也是哔了个狗，拜托，看一眼10大流通，这是一个明显的基金票好吗。庄个鬼啊。

第二天，这个收了个放量小阳，其实呢，这个股最近一段时间，机构调研不断，有些是持股机构，有些是新机构，有些是写研报的。

但是呢，这么多机构调研，却不涨，为何呢？

里面的资金基本都处于死拿的状态，实际市面上的流通股，已经很少。

新的机构，看好了这个股，想要拿筹码，怎么拿，只有往上面吃挂单，来吃筹码。但是呢，这一波强吃，依然吃不到多少筹码。

按照正常的操盘手法，高位横盘缩量，就要向下打压了股价，清洗浮筹，顺便拿点便宜筹码了。

但是呢，刚才我这个晚上顺口提了下，然后就有若干吃瓜群众买了进去了。

第二天他向下打压的时候，用来打压的筹码，被人吃瓜群众全部吃了。

这个事情就超级尴尬了，我用来打压的筹码，白送给别人了。

怎么办？这个时候，只有抄必杀了。你不肯卖？你还想买？那我出个利空吧，看你卖不卖，买不买。

吃瓜群众本身拿的不是很坚定的，一看，我艹，大减持，这还不得卖了。

而这，是我提这个股的第二天。

我当时心中就是晴空霹雳，心想，我艹，我这是被狗庄看上了吗？一说票就出利空减持公告。

当晚一看后台留言，大概十几条，委婉点的问我，减持怎么办，凶恶点直接喷我，你个狗托，果然给庄做托，xxx，yyy，zzz等等。

其实，宝宝心里也很苦，最近确实手风不顺，脑残手残，一直亏钱，但是庄托这个锅，我是肯定不会来背的。

我当晚随即安慰了几个委婉的，意思是说，按照止损模型来做，如果不触发止损，可以继续拿。

第二天，如期低开，浮筹跑路，资金开心捡筹码，重点不是捡便宜筹码，重点是成功拿到了筹码。

拿完筹码当晚。股东一看，队友筹码捡完，立刻发公告，我不减持了。

等于啥时候没发生，思想不坚定的小散户的筹码就被骗走了。这个剧情，满满的套路。

我松了一口气，至少，当晚没人来喷我是庄托了。

最后回过头来讲下这个股。

这波拉升不确定是已持仓资金拉的，还是新来的基金拉，从K先图来看，应该是新机构入驻，但是不确定是做短庄的，还是长线入驻的。

不管背后的资金是什么样的，这个股票的基本面依旧还是那样，我知道很多人肯定已经拿不住了，说好的价值投资股，涨的比短线票还要暴利。看的我眼都红了。

这个股，短期内涨幅过大，偏离5日线太远，有盘整的需要了。

部分获利资金也会考虑止盈的，所以建议，没买的暂时不要再买了，持仓的你已经有了20%的安全垫了，你可以选择继续持股不动，也可以抛掉小部分仓位，等回踩均线的时候再买回。

就这样子，其实没有什么好的操作建议，持股靠的是信仰，是对你基本面分析结果的信仰。

**赚钱效应**

赚钱效应：今天市场是冰火两重天的世界。

大部分银行股如期回调，但是出来的资金依然就很亢奋，转身就投向了其他蓝筹股。

昨晚我的复盘也写了。昨晚没刷出来上会家数，今早一看，我的妈呀，本周上会18家，疯掉了，so，开板次新直接全线跌停，连个扑腾都没有。

从而造成了今天两极分化的市场画面。

**大盘**

今天受到银行利空低开影响，大盘也低开，但是走势非常的强势。

但是开盘28分化的走势，基本也是所有人都吓尿了。

10点之后，创业板在上海钢联的带领下，绝地大反攻。

上证，今天缩量反包，上3500点，深证，今天平量长阳，小阶段新高。

创业板，放量长阳，反弹回了前期箱体内部，总的来说，从指标上来会所，大盘积累的风险已经越来越大了。

但是呢，资金依旧买买买，指数依旧涨涨涨，大盘的技术分析，最近已经失效了。

这个地方，没有什么建议可以给了，我只能说，作你看的懂的行情吧，如果实在看不懂，那就空仓，等看懂了再来。

**个股**

个股复盘：今天早上搜了下，雾霾+次新，然后开盘就买了点，上周集泰股份-7%到了止损位了，今天开盘直接止损了。

然后持仓股继续持仓，也没有动。

个股展望：

1：创业板

今天创业板大涨，从形态上面来说，和之前的这一波的起点很像，但是也有一点不一样。

上次的起爆点是，金爸爸买了创业板，有风向引导作用，这一波的起爆，是市场合力自发形成的。

和上一次有所不同，所以后面怎么走，太远的没人能预测的出来，但是从短的讲，机会还是有的。

标的：

上海钢联，今天的创业板龙头板，一定程度上带飞了创业板，所以他明天会享受最强的溢价，深谙龙头战法的各位游资各个哥哥，很多目前都在排板了。

易事特，今天最早的创业板自然板，主因是业绩增长，叠加老填权属性，一定程度上带起了填权板块。

以上两个创业板票，明天会享受最强的溢价。但是，涨停板后接力有风险，投资需谨慎。

3：上证50内的轮动

昨晚说道，银行今天出来的资金可能会有轮动到其他板块，今天最后看，轮动到了他头上。

但是上证50内的轮动预期依旧不变，之前讲的中国建筑依旧可以关注。

中国建筑今天尾盘抢筹依旧明显，本来我对他的预期是横盘若干天，等到回踩了5日线再上，目前来看，资金有点饥渴难耐的意思，建议依旧保持关注。

同步可关注标的：重庆建工，前天中国建筑异动，他率先上板的先锋。明天也有N字预期。

中铁二局，这个我之前讲过了。

3：高送转叠加填权

目前依旧是场内的热点，预告送转板块今天受到次新连累，没有表现，但是炒作预期依然存在。

股票池：杰恩设计，今飞凯达，杭州高新（今年年报最小盘的三个。）

杰恩设计，今天盘中历史新高了下，今年次新中10送15是最高的。

今飞凯达，是最便宜的，送完以后会更便宜，低价次新本身就是这波次新行情的引爆点，那么低价次新的送转，优势会更明显。

这个板块，只建议低吸，尤其是横盘以后的箱体低吸/转势后的隔日低吸。都是分批建仓的买点，预告送转板块，观点依旧保持不变。
