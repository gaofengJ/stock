# 爱在冰川复盘精华

本文档用于记录爱在冰川复盘中的精华内容，重点在于认知提升、核心观点及市场规律总结。

## 2020-1-1 复盘摘要

### 1. 认知提升

**Q：怎么提高认知力？**

其实这个东西没有太好的办法，本质上就是一个**闭环**：

> **学习 → 实践 → 总结 → 再实践 → 再总结优化 → 再实践**

如此往复，不断螺旋上升。

**核心方法论：**

* **Lesson Learn**：每一年的交易，每一月的交易，甚至每一天、每一笔交易，都可以作为一次复盘学习的机会。
* **重点关注**：重点复盘自己**爆赚**的票和**爆亏**的票，寻找其中的规律。
  > **Q：有人说，我没有爆赚过，怎么复盘？**
  >
  > 很简单，你肯定有过**爆炸卖飞**的经历，这个也可以总结下，为什么爆炸卖飞？这也是宝贵的经验。
* **复利公式**：
    > **稳定复利 = 赚钱的事情重复做 + 亏钱的事情不再犯**
