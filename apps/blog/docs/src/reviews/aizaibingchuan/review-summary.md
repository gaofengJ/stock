# 复盘关键信息提取

本文档用于规范复盘文章的关键信息提取格式，以便于快速回顾和总结市场规律。

## 格式实例

以下是一个标准的复盘信息提取格式：

### [日期] 复盘摘要

**原文链接**：[链接地址]

#### 1. 市场环境 (Market Context)

* **大盘走势**：(如：缩量小阳、放量大跌、探底回升等)
* **赚钱效应**：(如：极差、爆棚、一般、次新股强等)
* **政策/消息面**：(如：降准、贸易战、新规发布等)

#### 2. 核心观点 (Key Insights)

* **观点 1**：(如：当弱不弱是为强，空头转多头)
* **观点 2**：(如：稳定复利 = 赚钱的事情重复做 + 亏钱的事情不再犯)

#### 3. 机会与风险 (Opportunities & Risks)

* **机会**：(如：年报预披露、次新股博弈、5G板块低吸)
* **风险**：(如：高位股补跌、解禁压力、外围市场波动)

#### 4. 交易策略 (Trading Strategy)

* **仓位建议**：(如：不建议重仓、下午进场、空仓观望)
* **操作方向**：(如：低吸核心资产、博弈超跌反弹)

## 示例：基于 2020-1-1 复盘的提取

### 2020-1-1 复盘摘要

**原文链接**：<https://qcnevnajeo7h.feishu.cn/docx/E9o2dQ0uSoXIY9xkCUtcaqDCnPf>

#### 1. 市场环境

* **大盘走势**：受降准利好预期影响，市场情绪偏乐观。
* **消息面**：
  * 央行全面降准 0.5 个百分点，释放 8000 亿资金。
  * 中美预计 1-15 签署协议。

#### 2. 核心观点

* **降准博弈**：周五砸盘和周一抄底的可能是同一拨资金，利用信息差博弈。
* **认知提升**：学习 -> 实践 -> 总结 -> 再实践 -> 优化，如此往复。
* **复利公式**：稳定复利 = 赚钱的事情重复做 + 亏钱的事情不再犯。

#### 3. 机会与风险

* **机会**：金融（证券、保险）、地产、周期（水泥、钢铁）、网红、转基因。
* **风险**：高开低走（如果量能不足）。

#### 4. 交易策略

* **明日计划**：
    1. 开盘先买一点。
    2. 观察量能，若量能足够且走势强劲（√形），可二次加仓。
    3. 若高开量能不足，不加仓。
