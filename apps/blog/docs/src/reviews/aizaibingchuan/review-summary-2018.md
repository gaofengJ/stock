# 爱在冰川复盘精华-2018

本文档用于记录爱在冰川复盘中的精华内容，重点在于认知提升、核心观点及市场规律总结。

## 2018-1-1 复盘

### 节后第一天的买点

节后首个交易日，多空双方的博弈往往异常激烈。因此，不建议在早盘重仓介入，更稳妥的进场时机通常在**下午**。

具体应对策略如下：

* **如果大盘收大阳线**：下午通常会有踏空资金进场，发动新的热点题材（这一点在之前的文章中已详细阐述）。
* **如果大盘收大阴线**：下午可视情况尝试参与抄底博弈。
* **如果大盘不阴不阳**：则建议继续观望，多看少动。

### 节后第一天关注个股

节后第一天可以关注节前市场的核心标的，除了是指标，也是市场炒作属性的发散源。

* 一个稳定的炒股模式，必然是，小亏，小赚，小亏，小赚，小亏，小赚，循环，然后偶尔的一笔大肉，来实现财富增长。

* 打板的核心，就是打的硬板，小赚来自于，第二天的高开溢价。小亏来自于，第二天闷杀的亏损。这两者长期来说，是可以持平的。大赚来自于，第二天直接一字板了，或者高开秒板了。大亏来自于，日内炸板+第二天的低开。所以对打板客们来说，非常注重出手时，市场的气氛。因为对于一个合格的打板客来说，出手，板不炸，才是第一位的。

* 对于低吸客来说，大部分时候，是不会大亏的，当然，大部分的时候，也是不会大赚的。因为逻辑潜伏是个非常被动的战法，你永远处于被动等待临幸的状态下。

* 缩量包覆阳，往往就是风险的信号

  * 包覆阳（阳包阴）： 指由两根 K 线组成的形态。第一根是阴线，第二根是阳线。第二根阳线的实体完全覆盖（包住）了第一根阴线的实体（有时甚至包住上下影线）。这通常信号：空头力竭，多头反扑。

  * 缩量： 指第二根阳线出现时，成交量反而比第一根阴线更小
