# 2018-1-3复盘

作者：**爱在冰川**

原文链接: <https://mp.weixin.qq.com/s/mfzc89CEfSDvo8S4K1jTKg>

## 原文内容

**闲 聊**

今天雄安板了29家。我的天呐，这个场景我敢说，超出了所有人的预料，昨晚我也只是一般看好雄安，当然我最看好的标的是中设股份今早开盘秒板，完全不给我上车的机会。从而导致我一下字，就啥兴趣都没有了。你越是没兴趣，他就走的越是好，我就静静的看着雄安板了一家又一家。明天中设股份目测要一字板了，一个拥有28个小弟的龙头股，明天要是连板都没有的话，那也不正常了。

**赚钱效应**

赚钱效应一个板块打出29个涨停，可想而知市场的热度了。但是昨天的热点，今天基本全挂了，赚钱的板块缺乏持续性，所以赚钱效应还是绿的。

**大 盘**

昨天说大盘下面会出一个上影线，然后决定目前的这个区间，才说完，今天就出上影线了。大盘今天继续放量大涨，但是上影响已成，下面指数连续大涨显然不可能了，今天已经打压指数了看看盘中平安双煞的表现，很多有钱人脸色都不好看了。这两个，一个调上证指数，一个调深圳成指。所以我一看强调指数了，所以也把手上的证券股给出掉了。gjd已经不希望指数再涨了，证券就更没有涨的希望了。

**热 点**

雄安，主要驱动我昨晚的复盘已经写过了，领导人站台+新闻联播驱动，今天的表现，已经超出了市场所有人的想象力了。今天29个涨停股，龙头大概率继续一字板，剩下的分化，跟风的大概率直接低开低走。明天分化走势了，当然这也是市场的普遍预期了，机会肯定还会有，下面再说。

ST板块年底的ST摘帽预期行情，今天有热炒了，但是ST股我很少参与，术业有专攻就不多点评了其他热点：市场内的热钱都被雄安抽走了，这种情况下，还能板的其他热点股，都是值得重视的股。贵州燃气已经八板了，15年见顶之后，我没记错的话，应该是没有过能连续9连板的个股。在中华文化中，9是极数。所以我也不认为贵州燃气能突破9板。

**个 股**

个股复盘昨天潜伏证券，今天坐等拉升逻辑，勉强成功，但是随着gjd开始打压指数，我就只能把财通出掉了，整体盈利2%。。少的可怜。盘中看着天上飘这的二十几个雄安，明天接力肯定是难度巨大的，大概率是少部分个股表现，大部分就如同今天水泥昨跟风股的走势。

想来想去，看看芯片，连续两天下杀后拉起，芯片中军士兰微今日盘中再创新高，5G也很强势，所以我选择低吸芯片。低吸了一个韦尔股份。我元旦在家做了一个股票池，这两个是，只要大阳，就能历史新高的芯片股。这四个是，只要大阳，就能阶段新高的芯片股。这十个是，5个月内都不会有解禁股的芯片+次新股。所以我今天出掉财通后，小仓位水下低吸了点韦尔股份。

没想到吸了一嘴毛。。尾盘还有小跳水。韦尔股份已经走到了三角区域的末端了，需要选择方向了。所以我小仓搏一把，看能不能向上。如果向下，那回头就只能止损了。临外盘中脑抽，买了点罗顿发展。

个股展望。1：雄安：没啥好说的，今天29个板，明天一定会有连板股的，个人最看好的4个：今天没有板的雄安股，也同样看好的如下：2：新闻联播概念昨晚新闻联播的雄安炸天裂地，今天一众游资纷纷坐在电视面前看新闻联播。我给大家圈了一下分别是A：军队训练，寻找军队训练股，这个我还没开始搜，我好像记得洪都航空是造教练机的。但是这个比较冷门，你可以看直接看成利好军工和军民融合概念。B：浙江一带一路，标的就一个宁波港，C：光伏，这个一代而过了，估计没什么刺激了以上是今晚看到的一些短线机会。

另外关于昨晚提的两个黄金涨，和锌涨，开盘的时候，这两个都有上冲，但是被雄安干下来了，黄金至少还板了一个山东地矿。这两个逻辑，请继续关注现货的涨跌吧，另外中线板块，还是芯片，5G，中车，中线考验的是耐心和信仰，没啥好说的，我之前讲过中线水泥，房地产。

昨日大涨了，我昨晚复盘讲了下，说这两个中线板块终于又表现了，结果后台一堆喷我的，大致意思是说，跌的时候不说，涨了又拿出来嘚瑟，然后巴拉巴拉喷了很长一段，本来我还想回复下，还没给机会，就取关了。这里我再次统一回复下，我的公众号主要是写短线的，中线板块，我看到了机会，我也会说，但是我不会反复提，每天都在公众号的里反复讲一下，我看好的中线板块，这不是我风格。

我以前也讲过，复盘的最初目的，是便于我第二天换个地方看盘的时候，便于回忆，每个中线板块我讲完后，我都会讲他的驱动逻辑，你是否持仓，你就看下，逻辑还在不在。水泥我是最早11.15号：我看到有个牛叉股，叫万年青，平台突破了，然后我去查阅了很多研报，和产业报告，才发现，是水泥价格涨价了，然后我就写了各位也可以回过去找了那会的复盘，我并没有嘚瑟的意思，我自己都没买，我只是想跟各位讲下，中线股如何持仓？

水泥股的驱动就是水泥涨价，你每天登陆下水泥网，看看价格涨还是跌，库存怎么样，限产怎么样，心里有个术，如果水泥价格高位运行，你就放心拿好了。比如我之前讲过的猪肉，中线逻辑，你也去查查价格，不要天天问我，今天唐人神盘中又新高了。这种逻辑我讲过的，我不会天天碎碎念的再来反复讲，我也不是搞服务行业的，你们看我复盘也别想太多，谁也不欠谁的。我讲我的，你看你的，看的不爽，就取关就可以。我也不接广告，也不靠阅读数量吃饭。多一个少一个，真的无所谓的。明天我要去医院一天，看不了盘，芯片我也暂时不准备动了，各位祝好
