* 揉搓线与反揉搓线

| 维度 | 揉搓线 (Rubbing Line) | 反揉搓线 (Anti-Rubbing Line) |
| :--- | :--- | :--- |
| **K线组合** | 第一天：长上影线 (倒锤头)<br>第二天：长下影线 (锤头线) | 第一天：长下影线 (锤头线)<br>第二天：长上影线 (倒锤头) |
| **形态顺序** | 先上后下 (向上试盘 → 向下挖坑) | 先下后上 (向下支撑 → 向上受阻) |
| **主力动作** | 洗盘：故意制造宽幅震荡，把浮筹“搓”出去。 | 确认：在底部反复磨炼多空预期，测试抛压。 |
| **理想位置** | 上涨途中、均线多头排列时。 | 持续下跌后的底部、或横盘整理区间。 |
| **成交量** | 缩量为佳，证明主力没有离场。 | 底部放量表示反转增强；缩量则表示观望。 |
| **预判信号** | 洗盘结束，通常预示即将开启新一轮拉升。 | 底部确立，通常预示行情可能由跌转涨。 |

## 2017-9-8 复盘

* 炒股炒预期。对于高送转填权的炒作来说，大家都知道肯定填不满。在这个击鼓传花的游戏里面，你的预期其实很不清晰。
* 杭州园林的3个预期：实施公告公布的预期、除权日之前抢权的预期、除权后的填权预期。

| 概念名称 | 定义/解释 | 发生阶段 | 市场含义与心理 |
| :--- | :--- | :--- | :--- |
| 送红股 (Bonus Issue) | 公司将利润以股票形式分配给股东。 | 预案/实施 | 属于红利发放，需缴纳红利税（按持股时间）。 |
| 转增股本 (Capitalization) | 将资本公积金转为股本，不涉及利润。 | 预案/实施 | 不计入红利，通常不涉及税收，视觉效果同送股。 |
| 预案/预披露 | 董事会初步提出的高送转计划。 | 炒作起点 | 炒预期。资金往往在此时点开始潜伏。 |
| 股权登记日 (R-Day) | 确定谁有权参与送转的最后交易日。 | 实施前一天 | 分水岭。当天收盘持股的人才能享受送转。 |
| 除权除息日 (Ex-Date) | 股价进行折算、剔除权力的日期。 | 实施当天 | 股价视觉上瞬间变“便宜”，K线出现跳空。 |
| 填权 | 填权就是股价在除权除息后，重新涨回到除权前价格的过程。 | |
| 贴权 (Discount) | 除权后，股价跌破了除权基准价。 | 除权后 | 弱势表现。说明市场不认可，或处于派发末期。 |
| 抢权 (Running before Ex) | 在股权登记日前，大量资金涌入买入。 | 登记日前夕 | 强势表现。投资者为了拿股权或博弈填权而买入。 |
| 高送转潜力 | 根据每股公积金、每股收益等指标筛选。 | 年度财报前 | 盲盒阶段。炒作尚未公告但具备送转条件的个股。 |
