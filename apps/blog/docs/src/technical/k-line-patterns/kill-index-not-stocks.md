# 杀指数不杀个股

在市场运行中，经常会出现一种特殊的背离现象：**三大指数虽然下跌，但个股却跌幅有限甚至普遍上涨**。这种“纯杀指数”的行情，往往蕴含着特定的市场逻辑和机会。

## 一、 现象特征：白线向下，黄线坚挺

与“指数黄白线分叉”中提到的“黄线向下开叉”截然相反，这种行情的特征是：

- **白线（指数）**：大幅下挫，受权重股（如银行、保险、白酒等）拖累明显。
- **黄线（个股）**：走势平稳，甚至逆势上扬，处于白线上方。
- **涨跌家数**：虽然指数绿盘，但上涨家数往往多于下跌家数，或者跌停家数极少。

## 二、 市场内在逻辑

### 1. 权重股的补跌或调整

指数的下跌主要源于大市值权重股的调整。这可能是因为权重股前期涨幅过大需要消化获利盘，或者是受特定宏观消息影响。

### 2. 情绪与资金的“避风港”

当权重股杀跌时，场内资金并没有选择离场，而是流向了中小盘题材股。这说明**市场情绪依然活跃，资金做多意愿强烈**。个股的赚钱效应有效地对冲了指数下跌带来的恐慌。

## 三、 为什么“V起来的概率相对较大”？

正如文中提到的：“**纯杀指数的行情，V起来的概率相对较大**”。原因如下：

- **杀跌动能不足**：既然是个股不跌，说明市场并没有全面恐慌，仅仅是局部权重在调整。
- **资金承接力强**：题材股的活跃意味着场外资金仍在积极寻找机会。
- **合力反弹**：一旦权重股抛压减轻或止跌企稳，叠加题材股原本的强势，多方合力极易引发指数的日内探底回升（V型反转）。

## 四、 操作策略

遇到杀指数不杀个股的行情，核心策略是**轻指数，重个股**。

- **心态**：不要被指数的绿色吓倒，重点关注黄线的走势和涨停家数。
- **方向**：聚焦当前的热点题材和前排个股，这类行情往往是题材股表现的最佳窗口期。
- **警惕**：如果后期黄线也开始跟随白线补跌，则需警惕行情性质转化为全面退潮。
